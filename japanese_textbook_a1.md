> A1是「基礎階段的語言使用者」。能理解並使用日常熟悉的表達方式和非常基礎的片語，以滿足具體的需求。能自我介紹、介紹他人，並回答有關個人資訊（住在哪裡、認識誰、擁有什麼）的簡單問題。如果對方說得慢而清晰並提供協助，就能進行簡單的交流。

# 第1章 名詞文（判斷句）

## A1-001 ～は～です
**意味**：A是B（名詞謂語句）  
**结构**：N1 + は + N2 + です  
**説明**：將主題N1說明為N2。

1. わたしは**学生です**。
   → 我是學生。
2. これは**本です**。
   → 這是書。
3. あの人は**山田さんです**。
   → 那個人是山田先生。

## A1-002 ～は～じゃありません／～ではありませ
**意味**：A不是B（名詞否定句）  
**结构**：N1 + は + N2 + じゃありません / ではありませ  
**説明**：否定形式。口語用「じゃありません」，正式用「ではありませ」。

1. わたしは**先生じゃありません**。
   → 我不是老師。
2. これは**辞書ではありません**。
   → 這不是字典。
3. あの人は**田中さんじゃありません**。
   → 那個人不是田中先生。

## A1-003 ～は～ですか
**意味**：A是B嗎？（疑問句）  
**结构**：N1 + は + N2 + ですか  
**説明**：句尾加「か」形成疑問。回答時用「はい／いいえ」。

1. あなたは**学生ですか**。——はい、学生です。
   → 你是學生嗎？——是的，我是學生。
2. これは**かばんですか**。——いいえ、かばんじゃありません。
   → 這是包包嗎？——不，不是包包。
3. あの人は**鈴木さんですか**。
   → 那個人是鈴木先生嗎？

## A1-004 ～は～ですか、～ですか
**意味**：是A還是B？（選擇疑問）  
**结构**：N1 + は + N2 + ですか + N3 + ですか  
**説明**：列出兩個選項讓對方選擇。

1. これは**日本語の辞書ですか、英語の辞書ですか**。
   → 這是日語字典還是英語字典？
2. あの人は**先生ですか、学生ですか**。
   → 那個人是老師還是學生？
3. あなたは**日本人ですか、中国人ですか**。
   → 你是日本人還是中國人？

## A1-005 ～も～です
**意味**：A也是B  
**结构**：N1 + も + N2 + です  
**説明**：用「も」代替「は」，表示「也」。

1. わたし**も**学生です。
   → 我也是學生。
2. これ**も**本です。
   → 這也是書。
3. あの人**も**先生です。
   → 那個人也是老師。

## A1-006 NのN（屬性・所有）
**意味**：A的B（所有／所屬／內容）  
**结构**：N1 + の + N2  
**説明**：「の」連接兩個名詞，表示所屬關係、屬性、內容等。

1. **わたしの**本（所有）
   → 我的書
2. **日本語の**先生（內容）
   → 日語老師
3. **大学の**学生（所屬）
   → 大學生

## A1-007 ～は［人］のです（所有代名詞用法）
**意味**：～是某人的（東西）  
**结构**：N1 + は + 人 + の + です  
**説明**：避免重複名詞，用「の」代替。

1. この本は**わたしのです**。
   → 這本書是我的。
2. そのかばんは**山田さんのです**。
   → 那個包包是山田先生的。
3. あの車は**社長のです**。
   → 那輛車是社長的。

## A1-008 この／その／あのNは～です
**意味**：這個／那個／那個 N 是～  
**结构**：この／その／あの + N + は + ～ + です  
**説明**：「こそあど」連體詞修飾名詞。

1. **この**本はわたしのです。
   → 這本書是我的。
2. **その**かばんは山田さんのです。
   → 那個包包是山田先生的。
3. **あの**車は新しいです。
   → 那輛車是新的。

# 第2章 指示詞（こそあど）

## A1-009 これ／それ／あれは～です
**意味**：這個／那個／那個是～  
**结构**：これ／それ／あれ + は + ～ + です  
**説明**：これ（近稱）、それ（中稱）、あれ（遠稱）的事物代名詞。

1. **これは**スマートフォンです。
   → 這是智慧型手機。
2. **それは**何ですか。
   → 那是什麼？
3. **あれは**東京タワーです。
   → 那是東京鐵塔。

## A1-010 ここ／そこ／あそこは～です
**意味**：這裡／那裡／那裡是～  
**结构**：ここ／そこ／あそこ + は + ～ + です  
**説明**：場所的指示詞。

1. **ここは**駅です。
   → 這裡是車站。
2. **そこは**コンビニです。
   → 那裡是便利商店。
3. **あそこは**公園です。
   → 那裡是公園。

## A1-011 こちら／そちら／あちらは～です
**意味**：這邊／那邊／那邊是～（禮貌形）  
**结构**：こちら／そちら／あちら + は + ～ + です  
**説明**：場所指示詞的禮貌形。

1. **こちらは**受付です。
   → 這邊是接待處。
2. **そちらは**会議室です。
   → 那邊是會議室。
3. **あちらは**トイレです。
   → 那邊是洗手間。

## A1-012 ～はどこですか
**意味**：～在哪裡？  
**结构**：N + は + どこ + ですか  
**説明**：詢問地點的基本句型。

1. トイレ**はどこですか**。
   → 洗手間在哪裡？
2. 駅**はどこですか**。
   → 車站在哪裡？
3. 山田さん**はどこですか**。
   → 山田先生在哪裡？

## A1-013 どれ／どのNが～ですか
**意味**：哪個／哪個N是～？  
**结构**：どれ／どの + N + が + ～ + ですか  
**説明**：從多個項目中選擇的疑問。

1. あなたの傘は**どれですか**。
   → 你的雨傘是哪一把？
2. **どの**本があなたのですか。
   → 哪一本書是你的？
3. **どの**かばんが安いですか。
   → 哪一個包包便宜？

# 第3章 數字與時間

## A1-014 ～時～分です
**意味**：～點～分  
**结构**：數字 + 時 + 數字 + 分 + です  
**説明**：表達時間。注意「4時」「7時」「9時」的特殊讀音。

1. 今**8時です**。
   → 現在是8點。
2. 授業は**9時15分からです**。
   → 課程從9點15分開始。
3. 今**4時半です**（4時30分）。
   → 現在是4點半。

## A1-015 ～月～日です
**意味**：～月～日  
**结构**：數字 + 月 + 數字 + 日 + です  
**説明**：表達日期。注意「1日～10日」「14日」「20日」等特殊讀音。

1. 今日は**4月1日です**。
   → 今天是4月1日。
2. 私の誕生日は**12月25日です**。
   → 我的生日是12月25日。
3. 試験は**7月第1週です**。
   → 考試在7月第一週。

## A1-016 ～曜日です
**意味**：星期～  
**结构**：〜曜日 + です  
**説明**：表達星期。

1. 今日は**月曜日です**。
   → 今天是星期一。
2. 明日は**水曜日です**。
   → 明天是星期三。
3. 休みは**土曜日と日曜日です**。
   → 休息日是星期六和星期日。

## A1-017 ～から～までです
**意味**：從～到～  
**结构**：N + から + N + まで + です  
**説明**：表達時間或空間的範圍。

1. 授業は**9時から12時までです**。
   → 課程從9點到12點。
2. 銀行は**午前9時から午後3時までです**。
   → 銀行從上午9點到下午3點。
3. 東京**から大阪まで**新幹線で2時間半です。
   → 從東京到大阪坐新幹線要2個半小時。

## A1-018 ～に～ます（動作時間）
**意味**：在～時間做～  
**结构**：時間 + に + 動詞  
**説明**：「に」標記動作發生的時間點。注意：星期後可加可不加「に」，相對時間詞（今日、明日、毎日等）不加「に」。

1. 毎朝**6時に**起きます。
   → 每天早上6點起床。
2. 授業は**9時に**始まります。
   → 課程9點開始。
3. **日曜日に**買い物に行きます。
   → 星期天去買東西。

## A1-019 ～歳です
**意味**：～歲  
**结构**：數字 + 歳 + です  
**説明**：表達年齡。注意「1歳」「8歳」「20歳」的特殊讀音。

1. わたしは**25歳です**。
   → 我25歲。
2. 息子は**3歳です**。
   → 我兒子3歲。
3. もう**20歳です**。
   → 已經20歲了。

## A1-020 電話番号は～です
**意味**：電話號碼是～  
**结构**：電話番号 + は + 數字 + です  
**説明**：用「の」連接數字。

1. 電話番号は**090の1234の5678です**。
   → 電話號碼是090-1234-5678。
2. 部屋番号は**305号室です**。
   → 房間號碼是305室。
3. 郵便番号は**100-0001です**。
   → 郵遞區號是100-0001。

## A1-021 ～いくらですか
**意味**：多少錢？  
**结构**：N + は + いくら + ですか  
**説明**：詢問價格。

1. これは**いくらですか**。——500円です。
   → 這個多少錢？——500日圓。
2. そのかばん**はいくらですか**。
   → 那個包包多少錢？
3. りんごは**いくらですか**。
   → 蘋果多少錢？

# 第4章 動詞基本

## A1-022 ～ます（肯定現在・未來）
**意味**：做～（現在／未來肯定）  
**结构**：動詞ます形  
**説明**：動詞的禮貌形肯定。集團1動詞（五段）、集團2動詞（一段）、不規則動詞（する、くる）。

1. 毎日**勉強します**。
   → 每天學習。
2. 電車に**乗ります**。
   → 搭電車。
3. コーヒーを**飲みます**。
   → 喝咖啡。

## A1-023 ～ません（否定現在・未來）
**意味**：不做～（否定）  
**结构**：動詞ません形  
**説明**：動詞禮貌形的否定。

1. タバコを**吸いません**。
   → 不抽菸。
2. 今日は**働きません**。
   → 今天不工作。
3. 朝ごはんを**食べません**。
   → 不吃早餐。

## A1-024 ～ました（過去肯定）
**意味**：做了～（過去肯定）  
**结构**：動詞ました形  
**説明**：動詞禮貌形的過去肯定。

1. 昨日**勉強しました**。
   → 昨天學習了。
2. 先週**京都へ行きました**。
   → 上週去了京都。
3. 朝ごはんを**食べました**。
   → 吃了早餐。

## A1-025 ～ませんでした（過去否定）
**意味**：沒做～（過去否定）  
**结构**：動詞ませんでした形  
**説明**：動詞禮貌形的過去否定。

1. 昨日**勉強しませんでした**。
   → 昨天沒學習。
2. 朝ごはんを**食べませんでした**。
   → 沒吃早餐。
3. 友達に**会いませんでした**。
   → 沒見到朋友。

## A1-026 ～を～ます（目的語）
**意味**：做～（賓語）  
**结构**：N + を + Vます  
**説明**：「を」標記動作的對象（賓語）。

1. 新聞**を**読みます。
   → 看報紙。
2. 映画**を**見ます。
   → 看電影。
3. 音楽**を**聞きます。
   → 聽音樂。

## A1-027 ～で～ます（動作場所）
**意味**：在～做～  
**结构**：場所 + で + Vます  
**説明**：「で」標記動作發生的場所。注意區分存在句的「に」。

1. 図書館**で**勉強します。
   → 在圖書館學習。
2. レストラン**で**昼ごはんを食べます。
   → 在餐廳吃午餐。
3. 公園**で**散歩します。
   → 在公園散步。

## A1-028 ～へ行きます／来ます／帰ります
**意味**：去／來／回～  
**结构**：場所 + へ + 行きます／来ます／帰ります  
**説明**：「へ」標記移動方向。

1. 日本**へ**行きます。
   → 去日本。
2. 図書館**へ**来ます。
   → 來圖書館。
3. 家**へ**帰ります。
   → 回家。

## A1-029 ～と～ます（共同動作）
**意味**：和～一起做～  
**结构**：人 + と + Vます  
**説明**：「と」表示動作的共同對象。

1. 友達**と**映画を見ます。
   → 和朋友看電影。
2. 家族**と**食事します。
   → 和家人吃飯。
3. 山田さん**と**話します。
   → 和山田先生說話。

## A1-030 ～で～ます（手段・工具・語言）
**意味**：用～做～  
**结构**：手段／工具／語言 + で + Vます  
**説明**：「で」表示方法、手段、工具、使用的語言。

1. 日本語**で**話します。
   → 用日語說話。
2. 鉛筆**で**書きます。
   → 用鉛筆寫。
3. バス**で**行きます。
   → 搭公車去。

## A1-031 ～に～ます（着點・對象）
**意味**：對～做～  
**结构**：N + に + Vます  
**説明**：「に」表示動作的著落點、對象。

1. 電車**に**乗ります。
   → 搭電車。
2. 友達**に**メールを送ります。
   → 寄郵件給朋友。
3. 椅子**に**座ります。
   → 坐在椅子上。

## A1-032 ～ませんか（勸誘）
**意味**：要不要～？  
**结构**：Vませんか  
**説明**：禮貌地邀請對方一起做某事。

1. 一緒に映画を**見ませんか**。
   → 要不要一起看電影？
2. コーヒーを**飲みませんか**。
   → 要不要喝杯咖啡？
3. 土曜日に**買い物に行きませんか**。
   → 星期六要不要去買東西？

## A1-033 ～ましょう（意志・勸誘）
**意味**：～吧（意志／勸誘）  
**结构**：Vましょう  
**説明**：表示說話人的意志或勸誘對方一起做。

1. そろそろ**行きましょう**。
   → 差不多走吧。
2. 一緒に**勉強しましょう**。
   → 一起學習吧。
3. 今日は早く**帰りましょう**。
   → 今天早點回家吧。

## A1-034 ～ましょうか（提議幫忙）
**意味**：要不要我～？／我來～吧  
**结构**：Vましょうか  
**説明**：提議為對方做某事或主動提供幫助。

1. 窓を**開けましょうか**。
   → 我來開窗戶吧？
2. 荷物を**持ちましょうか**。
   → 我來拿行李吧？
3. コーヒーを**入れましょうか**。
   → 我來泡咖啡吧？

# 第5章 形容詞

## A1-035 い形容詞：～いです（肯定）
**意味**：～（形容詞肯定）  
**结构**：い形容詞（去掉い）＋いです  
**説明**：い形容詞以「い」結尾，直接加「です」表示禮貌。

1. この本は**おもしろいです**。
   → 這本書很有趣。
2. 今日は**暑いです**。
   → 今天很熱。
3. そのかばんは**高いです**。
   → 那個包包很貴。

## A1-036 い形容詞：～くないです／～くありませ
**意味**：不～（否定）  
**结构**：い形容詞（い→く）＋ないです / くありませ  
**説明**：い形容詞的否定形。口語常用「～くないです」。

1. この映画は**おもしろくないです**。
   → 這部電影不有趣。
2. 今日は**暑くないです**。
   → 今天不熱。
3. この辞書は**高くありません**。
   → 這本字典不貴。

## A1-037 い形容詞：～かったです（過去肯定）
**意味**：過去～（形容詞過去肯定）  
**结构**：い形容詞（い→かった）＋です  
**説明**：い形容詞的過去形。

1. 昨日は**暑かったです**。
   → 昨天很熱。
2. 試験は**難しかったです**。
   → 考試很難。
3. 映画は**おもしろかったです**。
   → 電影很有趣。

## A1-038 い形容詞：～くなかったです（過去否定）
**意味**：過去不～  
**结构**：い形容詞（い→くなかった）＋です  
**説明**：い形容詞的過去否定形。

1. 昨日は**暑くなかったです**。
   → 昨天不熱。
2. 試験は**難しくなかったです**。
   → 考試不難。
3. 値段は**高くなかったです**。
   → 價格不貴。

## A1-039 な形容詞：～です（肯定）
**意味**：是～（な形容詞肯定）  
**结构**：な形容詞（不加な）＋です  
**説明**：な形容詞在謂語位置時不加「な」。

1. 京都は**静かです**。
   → 京都很安靜。
2. この町は**有名です**。
   → 這個城鎮很有名。
3. わたしは**元気です**。
   → 我很有精神。

## A1-040 な形容詞：～じゃありません／～ではありませ（否定）
**意味**：不是～（否定）  
**结构**：な形容詞 + じゃありません / ではありませ  
**説明**：な形容詞的否定形。

1. この町は**静かじゃありません**。
   → 這個城鎮不安靜。
2. ここは**有名ではありません**。
   → 這裡不有名。
3. 彼は**元気じゃありません**。
   → 他沒有精神。

## A1-041 な形容詞：～でした（過去肯定）
**意味**：過去是～  
**结构**：な形容詞 + でした  
**説明**：な形容詞的過去肯定形。

1. 昔、この町は**静かでした**。
   → 以前這個城鎮很安靜。
2. あの映画は**有名でした**。
   → 那部電影很有名。
3. 若いときは**元気でした**。
   → 年輕的時候很有精神。

## A1-042 な形容詞：～じゃありませんでした（過去否定）
**意味**：過去不是～  
**结构**：な形容詞 + じゃありませんでした  
**説明**：な形容詞的過去否定形。

   → 那個時候沒有精神。
3. 試験は**簡単じゃありませんでした**。
   → 考試不簡單。

## A1-043 形容詞修飾名詞
**意味**：～的N（形容詞修飾名詞）  
**结构**：い形容詞 + N / な形容詞 + な + N  
**説明**：形容詞放在名詞前修飾名詞。い形容詞直接加名詞，な形容詞加「な」再加名詞。

1. **おいしい**料理（い形容詞）
   → 好吃的料理
2. **静かな**町（な形容詞）

# 第10章 比較

## A1-072 ～は～より～です（比較）
**意味**：A比B～  
**结构**：N1 + は + N2 + より + 形容詞 + です  
**説明**：比較兩者，N1比N2更～。

1. 日本語**は**英語**より**難しいです。
   → 日語比英語難。
2. 東京**は**大阪**より**大きいです。
   → 東京比大阪大。
3. 飛行機**は**電車**より**速いです。
   → 飛機比電車快。

## A1-073 ～より～のほうが～です（比較②）
**意味**：比起A，B更～  
**结构**：N2 + より + N1 + のほうが + 形容詞 + です  
**説明**：強調「B方更～」。

1. 英語**より日本語のほうが**難しいです。
   → 比起英語，日語更難。
2. 電車**よりバスのほうが**便利です。
   → 比起電車，公車更方便。
3. 肉**より魚のほうが**好きです。
   → 比起肉，更喜歡魚。

## A1-074 ～と～とどちらが～ですか（選擇比較）
**意味**：A和B哪個更～？  
**结构**：N1 + と + N2 + と + どちらが + 形容詞 + ですか  
**説明**：詢問兩者中哪一方更～。

1. **日本語と英語とどちらが**難しいですか。
   → 日語和英語哪個難？
2. **コーヒーと紅茶とどちらが**好きですか。
   → 咖啡和紅茶哪個喜歡？
3. **電車とバスとどちらが**速いですか。
   → 電車和公車哪個快？

## A1-075 ～の中で～が一番～です（最高級）
**意味**：在～之中，～最～  
**结构**：N（集団）+ の中で + N + が + 一番 + 形容詞 + です  
**説明**：表示在某範圍內最～。

1. 季節**の中で**夏**が一番**好きです。
   → 在季節之中最喜歡夏天。
2. 日本の食べ物**の中で**寿司**が一番**おいしいです。
   → 在日本食物之中壽司最好吃。
3. 家族**の中で**母**が一番**優しいです。
   → 在家人之中媽媽最溫柔。

**意味**：～完了／不小心～了  
**结构**：Vて形 + しまいます  
**説明**：表示動作完全結束（完了）或發生不好的事（遺憾）。

1. 宿題を**やってしまいました**。
   → 做完作業了。
2. 財布を忘**れてしまいました**。
   → 不小心忘了錢包。
3. ケーキを全部**食べてしまいました**。
   → 把蛋糕全吃完了。

# 第12章 理由・原因

## A1-083 ～から（理由）
**意味**：因為～  
**结构**：普通形／です形 + から  
**説明**：表示理由、原因。放在句末或句中。

1. 今日は**暑いから**、水をたくさん飲みます。
   → 因為今天熱，所以喝很多水。
2. 時間が**ありませんから**、急ぎましょう。
   → 因為沒時間了，快點吧。
3. 日本語が**好きですから**、勉強しています。
   → 因為喜歡日語，所以在學。

## A1-084 ～ので（理由・客觀）
**意味**：因為～（客觀理由）  
**结构**：普通形 + ので  
**説明**：比「から」更客觀、禮貌，常用於正式場合。

1. 風邪を**ひいたので**、学校を休みました。
   → 因為感冒了，所以請假沒去學校。
2. 値段が**安いので**、買いました。
   → 因為價格便宜，所以買了。
3. 時間が**ないので**、タクシーで行きます。
   → 因為沒時間，所以搭計程車去。

## A1-085 ～は～が～（主語・對象）
**意味**：～的～（主謂結構）  
**结构**：大主題 + は + 小主語 + が + ～  
**説明**：雙重主語結構，大主題用「は」，小主語用「が」。

1. わたし**は**頭**が**痛いです。
   → 我頭痛。
2. 日本**は**物価**が**高いです。
   → 日本物價高。
3. この町**は**交通**が**便利です。
   → 這個城鎮交通方便。

## A1-086 ～のが～（名詞化）
**意味**：做～是～  
**结构**：V普通形 + の + が + ～  
**説明**：用「の」將動詞句名詞化。

1. 料理**をするのが**好きです。
   → 喜歡做菜。
2. 音楽**を聞くのが**上手です。
   → 擅長聽音樂（很會欣賞音樂）。
3. 毎日運動**するのが**大切です。
   → 每天運動很重要。

**説明**：表示義務、必須。

1. 明日**早く起きなければなりません**。
   → 明天必須早起。
2. 薬を**飲まなければなりません**。
   → 必須吃藥。
3. レポートを**出さなければなりません**。

## A1-091 ～なくてもいいです（不需要）
**意味**：不需要做～  
**结构**：Vない形（ない→なくて）+ も + いいです  
**説明**：表示不需要做某事。

1. 明日は**来なくてもいいです**。
   → 明天不來也可以。
2. 全部**食べなくてもいいです**。
   → 不用全部吃完。
3. お金を**払わなくてもいいですか**。——はい、無料です。

## A1-092 ～たほうがいいです（建議）
**意味**：最好做～  
**结构**：Vた形 + ほうがいいです  
**説明**：提出建議。

1. 病院へ**行ったほうがいいです**。

2. 早く**寝たほうがいいです**。
   → 最好早點睡。
3. 薬を**飲んだほうがいいですよ**。
   → 最好吃藥喔。

## A1-093 ～ないほうがいいです（建議否定）
**意味**：最好不要做～  
**结构**：Vない形 + ほうがいいです  
**説明**：建議不要做某事。

1. あまり**無理しないほうがいいです**。
   → 最好不要太勉強。
2. その言葉は**使わないほうがいいです**。
   → 最好不要用那個詞。
3. 夜遅く**出かけないほうがいいです**。
   → 最好不要深夜出門。

## A1-094 ～ことができます（可能）
**意味**：能夠做～  
**结构**：V辞書形 + こと + が + できます  
**説明**：表示能力和可能性。

1. 日本語で手紙を**書くことができます**。
   → 能用日語寫信。
2. ここで**泳ぐことができます**。
   → 能在這裡游泳。
3. 一人で**行くことができません**。
   → 不能一個人去。

# 第15章 推量・傳聞・説明

## A1-095 ～でしょう（推量）
**意味**：～吧（推測）  
**结构**：普通形 + でしょう  
**説明**：表示推測，相當於「大概～吧」。

1. 明日は**雨でしょう**。
   → 明天大概下雨吧。
2. 彼は**来ないでしょう**。
   → 他大概不來吧。
3. この料理は**辛いでしょう**。
   → 這道菜大概辣吧。

## A1-096 ～かもしれません（可能）
**意味**：也許～／可能～  
**结构**：普通形 + かもしれません  
**説明**：表示可能性較低的推測。

1. 明日は**雨かもしれません**。
   → 明天也許會下雨。
2. 彼は**遅れるかもしれません**。
   → 他可能會遲到。
3. 値段は**高いかもしれません**。
   → 價格可能貴。

## A1-097 ～そうです（様態・傳聞）
**意味**：看起來～／聽說～  
**结构**：様態：Vます形／い形容詞語幹 + そうです
      傳聞：普通形 + そうです  
**説明**：様態（看起來～）與傳聞（聽說～）用法不同。

1. このケーキは**おいしそうです**。（様態）
   → 這個蛋糕看起來很好吃。
2. 雨が**降りそうです**。（様態）
   → 好像要下雨了。
3. 山田さんは**来ないそうです**。（傳聞）
   → 聽說山田先生不來。

## A1-098 ～のです／～んです（説明・強調）
**意味**：～的／就是～（説明・強調）  
**结构**：普通形 + のです／んです  
**説明**：用於説明狀況、理由、強調。口語常用「んです」。

1. 昨日は**休みました。熱があったんです**。
   → 昨天休息了。因為發燒了。
2. **どうしたんですか**。——頭が痛いんです。
   → 怎麼了？——頭痛。
3. 日本語を勉強しています。将来、日本で**働きたいんです**。
   → 我在學日語。將來想在日本工作。

## A1-101 いつ
**意味**：什麼時候  
**说明**：詢問時間。

1. 誕生日は**いつですか**。

3. 試験は**いつ**ですか。

## A1-102 なぜ／どうして

# 第17章 頻度・程度

## A1-106 いつも～ます、よく～ます
**意味**：總是～／常常～  
**结构**：頻度副詞 + Vます  
**説明**：表達頻率。

1. **いつも**6時に起きます。
   → 總是6點起床。
2. **よく**映画を見ます。
   → 常常看電影。
3. **いつも**ここで朝ごはんを食べます。
   → 總是在這裡吃早餐。

## A1-107 ときどき～ます、あまり～ません、ぜんぜん～ません
**意味**：有時候～／不太～／完全不～  
**结构**：頻度副詞 + Vます／Vません  
**説明**：表達頻率。「あまり」「ぜんぜん」需搭配否定。

1. **ときどき**料理をします。
   → 有時候做菜。
2. **あまり**テレビを**見ません**。
   → 不太看電視。
3. **ぜんぜん**お酒を**飲みません**。
   → 完全不喝酒。

## A1-108 とても～、少し～、あまり～ない
**意味**：很～／稍微～／不太～  
**结构**：程度副詞 + 形容詞  
**説明**：表達程度。

1. このラーメンは**とても**おいしいです。
   → 這個拉麵非常好吃。
2. 部屋は**少し**狭いです。
   → 房間有點窄。
3. この本は**あまり**おもしろく**ないです**。
   → 這本書不太有趣。

## A1-109 ～より～のほうが（再掲・程度比較）
**意味**：比起A，B更～  
**结构**：N2 + より + N1 + のほうが + A / V  
**説明**：程度比較的表達。

1. 電車**よりバスのほうが**安いです。
   → 比起電車，公車更便宜。
2. 肉**より魚のほうが**健康にいいです。
   → 比起肉，魚對健康更好。
3. 日本語**より英語のほうが**簡単です。
   → 比起日語，英語更簡單。

# 第18章 接續表現

## A1-110 そして（並列）
**意味**：然後／而且  
**结构**：文 + そして + 文  
**説明**：連接句子，表示添加或順接。

1. 昨日は買い物をしました。**そして**映画を見ました。
   → 昨天去買東西了，然後看了電影。
2. 彼は頭がいいです。**そして**親切です。
   → 他頭腦很好，而且很親切。
3. 朝ごはんを食べました。**そして**学校へ行きました。
   → 吃了早餐，然後去學校了。

## A1-111 それから（然後・而且）
**意味**：然後／而且  
**结构**：文 + それから + 文  
**説明**：添加新資訊，常用於列舉。

1. スーパーで野菜を買いました。**それから**肉も買いました。
   → 在超市買了蔬菜，然後也買了肉。
2. 京都に行きました。**それから**奈良にも行きました。
   → 去了京都，然後也去了奈良。
3. 勉強しました。**それから**少し寝ました。
   → 學習了，然後睡了一會兒。

## A1-112 でも（逆接）
**意味**：但是  
**结构**：文 + でも + 文  
**説明**：表示轉折。

1. この町は便利です。**でも**家賃が高いです。
   → 這個城鎮很方便，但是房租很貴。
2. 日本語は難しいです。**でも**おもしろいです。
   → 日語很難，但是很有趣。
3. 昨日は雨でした。**でも**出かけました。
   → 昨天下雨了，但是還是出門了。

## A1-113 それで（原因・結果）
**意味**：所以／因此  
**结构**：文 + それで + 文  
**説明**：表示前句是後句的原因或理由。

1. お金がありません。**それで**買いませんでした。
   → 沒錢，所以沒買。
2. 頭が痛いです。**それで**早く帰ります。
   → 頭痛，所以要早點回去。
3. 駅が遠いです。**それで**バスで行きます。
   → 車站很遠，所以搭公車去。

## A1-114 そして／それから（再掲・時間順序）
**意味**：然後  
**结构**：文章＋そして／それから＋文章  
**説明**：表示時間順序。

1. うちへ帰りました。**そして**ごはんを食べました。
   → 回家了，然後吃飯了。
2. 顔を洗います。**それから**歯を磨きます。
   → 洗臉，然後刷牙。
3. 仕事が終わりました。**そして**友達に会いました。
   → 工作結束了，然後見了朋友。

# 第19章 授受動詞（基礎）

## A1-115 ～をあげます（給予・向外）
**意味**：給～（我給別人）  
**结构**：人 + に + 物 + を + あげます  
**説明**：主語是給予的一方，接受者用「に」。

1. 友達**に**プレゼント**をあげます**。
   → 給朋友禮物。
2. 母**に**花**をあげました**。
   → 給了媽媽花。
3. 先生**に**お土産**をあげました**。
   → 給了老師伴手禮。

## A1-116 ～をくれます（給予・向内）
**意味**：給我（別人給我）  
**结构**：人 + が + 物 + を + くれます  
**説明**：以我方為視角，別人給我或我方的人。

1. 友達**が**本**をくれました**。
   → 朋友給了我書。
2. 母**が**セーター**をくれました**。
   → 媽媽給了我毛衣。
3. 山田さん**が**チケット**をくれました**。
   → 山田先生給了我票。

## A1-117 ～をもらいます（接受）
**意味**：從～收到～  
**结构**：人 + に／から + 物 + を + もらいます  
**説明**：以接受者為視角，從別人那裡得到。

1. 友達**に**プレゼント**をもらいました**。
   → 從朋友那裡收到了禮物。
2. 先生**から**辞書**をもらいました**。
   → 從老師那裡得到了字典。
3. 誕生日に両親**に**時計**をもらいました**。
   → 生日時從父母那裡得到了手錶。

# 第20章 常見助詞總結

## A1-118 は（主題）
**意味**：～是（主題標記）  
**説明**：提出句子的主題。對比時也使用。

1. わたし**は**学生です。
2. 今日**は**暇です。
3. 日本語**は**難しいです。

## A1-119 が（主語）
**意味**：～（主語標記）  
**説明**：標記小句主語、新資訊主語、能力對象、喜好對象等。

1. 誰**が**来ましたか。——山田さん**が**来ました。
2. 猫**が**好きです。
3. 日本語**が**わかります。

## A1-120 を（賓語）
**意味**：～（賓語標記）  
**説明**：標記他動詞的動作對象。

1. 水**を**飲みます。
2. 新聞**を**読んでいます。
3. 部屋**を**掃除します。

## A1-121 に（時間・場所・對象・着點）
**意味**：在～／對～  
**説明**：標記時間點、存在場所、動作對象、移動着點。

1. 8時**に**起きます。（時間）
2. 公園**に**猫がいます。（存在場所）
3. 友達**に**電話します。（對象）

## A1-122 で（場所・手段）
**意味**：在～／用～  
**説明**：標記動作場所、手段方法。

1. 図書館**で**勉強します。（場所）
2. バス**で**行きます。（手段）
3. 日本語**で**話します。（語言）

## A1-123 へ（方向）
**意味**：向～／往～  
**説明**：標記移動方向。

1. 日本**へ**行きます。
2. 駅**へ**向かいます。
3. 学校**へ**来ました。

## A1-124 と（共同・並列）
**意味**：和～  
**説明**：標記共同動作對象、列舉全部。

1. 友達**と**話します。
2. りんご**と**ばななを買いました。
3. 家族**と**住んでいます。

## A1-125 から（起點・理由）
**意味**：從～／因為～  
**説明**：標記時間或空間起點、原因理由。

1. 東京**から**大阪まで新幹線で行きます。（起點）
2. 9時**から**5時まで仕事です。（時間起點）
3. 時間がない**から**、急ぎましょう。（理由）

## A1-126 まで（終點）
**意味**：到～  
**説明**：標記時間或空間的終點。

1. 家**まで**歩いて10分です。
2. 5時**まで**働きます。
3. 東京**まで**行きます。

## A1-127 の（所有・名詞化）
**意味**：的（所有／名詞化）  
**説明**：連接名詞表示所有關係；將動詞句名詞化。

1. わたし**の**本（所有）
2. 日本語**の**先生（內容）
3. 音楽を聞く**の**が好きです（名詞化）

## A1-128 か（疑問・選擇）
**意味**：嗎／或者  
**説明**：句尾加「か」形成疑問；句中用「か」表示選擇。

1. 学生です**か**。（疑問）
2. コーヒー**か**紅茶をください。（選擇）
3. どこへ行きます**か**。（疑問）

# 第21章 其他基本表現

## A1-129 ～ましょうか（提議）
**意味**：～吧／要不要我～？  
**説明**：提議或提供幫助。

1. 窓を**開けましょうか**。
2. いっしょに**行きましょうか**。
3. お茶を**入れましょうか**。

## A1-130 ～ながら（同時動作）
**意味**：一邊～一邊～  
**结构**：Vます（去ます）+ ながら + V  
**説明**：表示兩個動作同時進行。

1. 音楽を**聞きながら**勉強します。
   → 一邊聽音樂一邊學習。
2. テレビを**見ながら**ごはんを食べます。
   → 一邊看電視一邊吃飯。
3. 歩き**ながら**話しましょう。
   → 一邊走一邊說吧。

## A1-131 ～でしょう（確認）
**意味**：～吧（確認）  
**说明**：句尾上昇調用於確認。

1. あなたは山田さん**でしょう**。
   → 你是山田先生吧？
2. これ、おいしい**でしょう**。
   → 這個好吃吧？
3. 明日も来ます**でしょう**。——はい、来ます。
   → 明天也會來吧？——是的，會來。

## A1-132 ～ね（確認・同意）
**意味**：～呢／～吧（確認・感嘆）  
**说明**：句尾的確認或感嘆。

1. 今日はいい天気です**ね**。
   → 今天天氣真好呢。
2. 日本語は難しいです**ね**。
   → 日語很難呢。
3. おいしいです**ね**。——そうです**ね**。
   → 很好吃呢。——對啊。

## A1-133 ～よ（告知・主張）
**意味**：～喔（告知・提醒）  
**说明**：句尾的告知或提醒。

1. その電車は行きません**よ**。
   → 那班電車不開喔。
2. 試験は来週です**よ**。
   → 考試是下週喔。
3. お金を忘れました**よ**。
   → 你忘了錢喔。

## A1-134 ～よね（確認尋求）
**意味**：～對吧  
**説明**：尋求對方同意的表達。

1. ここは駅です**よね**。
   → 這裡是車站對吧。
2. 彼は来ます**よね**。
   → 他會來對吧。
3. これ、高かったです**よね**。
   → 這個很貴對吧。

## A1-135 お～します（謙讓）
**意味**：我為您做～（謙讓語・初步）  
**结构**：お + Vます（去ます）+ します  
**説明**：謙讓語的基本形，用於自己的行為。

1. 荷物を**お持ちします**。
   → 我為您拿行李。
2. 後で**お電話します**。
   → 我之後打電話給您。
3. 案内**します**→**お案内します**。
   → 我為您帶路。
