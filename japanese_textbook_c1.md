# 第1章 Time - Immediate Sequence（時間・即時）

## C1-001 ～が早いか
**意味**：剛一～就～（書面語，表示緊接發生）  
**结构**：V辞書形／Vた形 + が早いか  
**説明**：表示前項動作結束後緊接著發生後項，多用於書面語或文學描述。

1. 彼はベルが鳴る**が早いか**、教室を飛び出した。
   → 他鈴聲一響就衝出了教室。
2. 目を覚ます**が早いか**、彼女は携帯をチェックし始めた。
   → 她一睜開眼就開始檢查手機。
3. 新商品の発売の知らせを聞く**が早いか**、客が殺到した。
   → 一聽到新商品發售的消息，顧客就蜂擁而至。

## C1-002 ～や／や否や
**意味**：剛一～就～（書面語）  
**结构**：V辞書形 + や／や否や  
**説明**：表示前項發生後緊接著後項發生，屬於正式書面語。不可用於意志、命令、推量。

1. 電車が駅に着く**や否や**、乗客が一斉にドアに向かった。
   → 電車一到站，乘客就一齊湧向車門。
2. 彼は家に帰る**や**、ソファに倒れ込んだ。
   → 他一回到家就癱倒在沙發上。
3. 社長が会場に現れる**や否や**、拍手が起きた。
   → 社長一出現會場，掌聲就響了起來。

## C1-003 ～そばから
**意味**：剛～就～（馬上又重複）  
**结构**：V辞書形／Vた形 + そばから  
**説明**：表示前項剛做完，後項馬上又發生同樣的事，帶有「做了也白做」的負面含義。

1. 掃除する**そばから**、子供がおもちゃを散らかす。
   → 剛打掃完，小孩就把玩具弄亂了。
2. 覚える**そばから**忘れてしまう。
   → 剛記住就忘了。
3. 直す**そばから**壊れるので、もう諦めた。
   → 剛修好就又壞了，我已經放棄了。

## C1-004 ～か～ないかのうちに
**意味**：剛～還沒～完就～  
**结构**：V辞書形 + か + Vない形 + ないかのうちに  
**説明**：表示前項動作幾乎同時發生後項，強調時間上幾乎沒有間隔。

1. ベッドに横になる**かならないかのうちに**、眠りに落ちた。
   → 剛躺到床上就睡著了。
2. 話し終わる**か終わらないかのうちに**、彼は席を立った。
   → 話還沒說完他就站起來離開了。
3. 家に着く**か着かないかのうちに**、雨が降り出した。
   → 剛到家（還沒完全到家）就下起雨來了。

## C1-005 ～たとたん（に）
**意味**：在～的那一瞬間（突然發生）  
**结构**：Vた形 + とたん（に）  
**説明**：表示前項動作完成的一瞬間，發生了意想不到的後項變化。

1. 立ち上がった**とたんに**、めまいがした。
   → 一站起來就頭暈了。
2. ドアを開けた**とたん**、猫が外に飛び出した。
   → 一打開門，貓就衝出去了。
3. 彼の顔を見た**とたんに**、涙が溢れ出た。
   → 一看到他的臉，眼淚就湧了出來。

## C1-006 ～次第
**意味**：一～就～（正式場合）  
**结构**：Vます形（去ます）+ 次第  
**説明**：表示前項動作完成後立即進行後項，用於正式場合。

1. 資料が準備でき**次第**、メールで送信いたします。
   → 資料一準備好，我就用郵件發送。
2. 結果が分かり**次第**、ご連絡します。
   → 一知道結果我就聯繫您。
3. 向こうに着き**次第**、電話します。
   → 一到那邊我就打電話。

## C1-007 ～なり
**意味**：一～就～（文學體）  
**结构**：V辞書形／Vた形 + なり  
**説明**：表示前項動作完成後緊接後項，同樣用於書面語。後項為過去時。

1. 彼女は部屋に入る**なり**、泣き出した。
   → 她一進房間就哭了起來。
2. 息子は帰ってくる**なり**、「お腹空いた」と言った。
   → 兒子一回到家就說「肚子餓了」。
3. 電話を切る**なり**、彼は怒り出した。
   → 他剛掛斷電話就發起火來。

# 第2章 Time - Limits and Boundaries（時間・限界）

## C1-008 ～が最後
**意味**：一旦～就完了（一旦～就再也～）  
**结构**：Vた形 + が最後  
**説明**：表示一旦做了前項，就必須承擔後果或無法挽回。

1. あのゲームを始めた**が最後**、やめられなくなる。
   → 一旦開始玩那個遊戲，就停不下來了。
2. 秘密を話した**が最後**、終わりだ。
   → 一旦說出秘密，就完了。
3. 彼に頼んだ**が最後**、しつこく付きまとわれる。
   → 一旦拜託了他，就會被他纏著不放。

## C1-009 ～たが最後
**意味**：一旦做了～就（導致不好結果）  
**结构**：Vた形 + が最後  
**説明**：與C1-008相同，強調一旦造成某種狀態就導致負面後果。

1. あのサイトを見た**が最後**、ウイルスに感染する。
   → 一旦看了那個網站，就會感染病毒。
2. 一目見た**が最後**、彼女の魅力に囚われてしまった。
   → 只看了一眼，就被她的魅力俘虜了。
3. 喧嘩した**が最後**、もう二度と仲良くできない。
   → 一旦吵了架，就再也無法和好了。

## C1-010 ～を限りに
**意味**：到～為止（以～為限）  
**结构**：N + を限りに  
**説明**：表示以某個時間點為截止期限，到該時間為止。

1. 今日**を限りに**、この店は閉店します。
   → 以今天為限，這家店關門歇業。
2. 3月**を限りに**、今の仕事を辞めるつもりだ。
   → 我打算到3月底為止辭掉現在的工作。
3. 今月**を限りに**、タバコをやめよう。
   → 以這個月為限，戒菸吧。

## C1-011 ～をもって
**意味**：以～／藉由～（正式表示期限或手段）  
**结构**：N + をもって  
**説明**：用於正式場合，表示以某種方式、手段，或表示時間的截止點。

1. 本日の営業は**以上をもって**終了させていただきます。
   → 今天的營業到此結束。
2. 結果は**書面をもって**お知らせいたします。
   → 結果將以書面方式通知。
3. 申込期限は**3月15日をもって**締め切ります。
   → 報名截止日期為3月15日。

## C1-012 ～限りだ
**意味**：非常～／極其～（表示強烈感情）  
**结构**：い形容詞連体形／な形容詞な + 限りだ  
**説明**：強調說話人的某種感情到了極限。

1. 皆様にご支援いただき、**嬉しい限りです**。
   → 得到大家的支持，我極為高興。
2. 彼に裏切られるとは、**悔しい限りだ**。
   → 被他背叛，真是悔恨至極。
3. 久しぶりに会えて、**楽しい限りです**。
   → 久別重逢，真是無比愉快。

## C1-013 ～極まる／極まりない
**意味**：極其～／無以復加（正式）  
**结构**：な形容詞語幹 + 極まる／極まりない  
**説明**：表示程度達到極點，用於正式場合。常用於負面或強烈評價。

1. 彼の態度は**失礼極まる**ものだった。
   → 他的態度極其無禮。
2. そんな行動は**危険極まりない**。
   → 那種行為極其危險。
3. 会議での彼の発言は**不適切極まりなかった**。
   → 他在會議上的發言極不合適。

# 第3章 Contrast and Concession Advanced（讓歩・對比）

## C1-014 ～とはいえ
**意味**：雖說～但是～（書面語）  
**结构**：普通形 + とはいえ  
**説明**：表示承認前項事實，但後項提出相反或保留意見。相當於「雖說」。

1. 便利になった**とはいえ**、人間関係が薄れた気がする。
   → 雖說變方便了，但我覺得人際關係變淡薄了。
2. 彼は優秀**とはいえ**、まだ経験が足りない。
   → 雖說他很優秀，但經驗還不足。
3. 値段が高い**とはいえ**、品質は確かだ。
   → 雖說價格貴，但品質確實有保障。

## C1-015 ～といえども
**意味**：即使～也～／雖説～但～（正式）  
**结构**：N／普通形 + といえども  
**説明**：表示讓歩，強調即使是在前項條件下後項也不成立。比「とはいえ」語氣更強。

1. **先生といえども**、間違えることはある。
   → 即使是老師，也有犯錯的時候。
2. **天才といえども**、努力なしに成功はできない。
   → 即使是天才，不努力也無法成功。
3. **いかなる理由があろうといえども**、暴力は許されない。
   → 無論有什麼理由，暴力都是不被允許的。

## C1-016 ～ものを（悔恨）
**意味**：明明～卻～（遺憾、不滿）  
**结构**：普通形 + ものを  
**説明**：表示對未能實現的事感到遺憾，或對對方做了不該做的事表示不滿。

1. 教えてくれればよかった**ものを**、なぜ黙っていたんだ。
   → 明明告訴我就好了，為什麼保持沉默呢。
2. ちゃんと勉強すれば合格できた**ものを**、遊んでばかりいた。
   → 明明好好學習就能及格，卻一直玩。
3. 早く病院に行けば治った**ものを**、我慢して悪化させてしまった。
   → 明明早點去醫院就能治好的，硬撐卻惡化了。

## C1-017 ～ものを（逆接・意外）
**意味**：本該～卻～（與預期相反）  
**结构**：普通形 + ものを  
**説明**：表示事與願違，常用於責怪或惋惜的語氣。

1. 簡単にできる**ものを**、なぜあんなに難しく考えるんだ。
   → 明明很簡單就能做到，為什麼要想得那麼複雜。
2. 謝れば許してくれた**ものを**、意地を張って関係が壊れた。
   → 明明道歉就會被原諒的，卻固執而破壞了關係。
3. 一言断ってくれれば良かった**ものを**、無断で使ってしまった。
   → 明明跟我說一聲就好了，卻擅自使用了。

## C1-018 ～ながらも
**意味**：雖然～但是～（書面）  
**结构**：Vます形去ます／い形容詞い／な形容詞語幹 + ながらも  
**説明**：表示讓歩，前後項對比。

1. 狭い**ながらも**、この部屋には温かみがある。
   → 雖然狹窄，但這個房間有溫馨感。
2. 若い**ながらも**、彼の判断は的確だ。
   → 雖然年輕，但他的判斷很準確。
3. 不自由**ながらも**、充実した日々を送っている。
   → 雖然不自由，但過著充實的日子。

## C1-019 ～つつも
**意味**：一邊～一邊～／雖然～卻～  
**结构**：Vます形去ます + つつも  
**説明**：表示兩個動作同時進行，或者讓歩關係（明明做了前項卻還是做後項）。

1. 悪いと知り**つつも**、彼は嘘をついた。
   → 明知不對，他還是說了謊。
2. ダイエットしなければと思い**つつも**、つい食べてしまう。
   → 明知必須減肥，卻還是不小心吃了。
3. 彼女の意見に同意し**つつも**、心の中では納得していなかった。
   → 雖然同意她的意見，但心裡並不服氣。

## C1-020 ～にも関わらず
**意味**：儘管～卻～（正式）  
**结构**：N + にも関わらず／V普通形 + にも関わらず  
**説明**：表示讓歩關係，後項結果與前項預期相反。

1. 激しい雨**にも関わらず**、多くの人が集まった。
   → 儘管下著大雨，還是有很多人聚集了。
2. 何度も注意した**にも関わらず**、彼は同じミスを繰り返した。
   → 儘管提醒了很多次，他還是重複同樣的錯誤。
3. 準備不足**にも関わらず**、見事な発表だった。
   → 儘管準備不足，卻是一場出色的發表。

## C1-021 ～にもかかわらず
**意味**：儘管～卻～（標準正式）  
**结构**：N + にもかかわらず／V普通形 + にもかかわらず  
**説明**：與C1-020相同，為較常見的寫法。

1. 彼は重い病気**にもかかわらず**、仕事を続けている。
   → 儘管身患重病，他仍在繼續工作。
2. 周囲の反対**にもかかわらず**、彼女は夢を追い続けた。
   → 儘管周圍反對，她仍繼續追求夢想。
3. 何度も失敗した**にもかかわらず**、彼は挑戦をやめなかった。
   → 儘管失敗多次，他仍未放棄挑戰。

# 第4章 Cause and Reason Formal（原因・理由・正式）

## C1-022 ～こととて
**意味**：因為～所以～（正式書面語）  
**结构**：N + の + こととて／V普通形 + こととて  
**説明**：表示理由，用於正式書面或鄭重的場合。常用於辯解或說明背景。

1. **初心者のこととて**、至らぬ点が多々あるかと存じます。
   → 因為是新手，我想有很多不周到的地方。
2. **慣れぬこととて**、ご迷惑をおかけしました。
   → 因為不熟悉，給您添麻煩了。
3. **何分田舎のこととて**、夜は早くに店が閉まってしまいます。
   → 畢竟是鄉下，晚上商店很早就關門了。

## C1-023 ～とあって
**意味**：因為是～所以～  
**结构**：N／普通形 + とあって  
**説明**：表示由於前項的特殊情況而自然導致後項的結果。

1. 連休**とあって**、高速道路は大渋滞だった。
   → 因為是連休，高速公路大塞車。
2. 初めての海外旅行**とあって**、彼は前日から興奮していた。
   → 因為是第一次出國旅行，他從前一天就開始興奮。
3. 有名店の新作**とあって**、開店前から行列ができていた。
   → 因為是名店的新品，開店前就排起了長隊。

## C1-024 ～とあれば
**意味**：如果是～的話（就會～）  
**结构**：N／普通形 + とあれば  
**説明**：表示假定條件，如果前項成立則當然會做後項。

1. **必要とあれば**、いつでも駆け付けます。
   → 如果有需要，我隨時趕到。
2. 彼の**ためとあれば**、どんな努力も惜しまない。
   → 如果是為了他的話，任何努力都在所不惜。
3. **お客様のご要望とあれば**、すぐに手配いたします。
   → 如果是客戶的要求，我立刻安排。

## C1-025 ～とくれば
**意味**：提到～／說起～  
**结构**：N + とくれば  
**説明**：表示提出某個話題，然後進行相關的描述或評價。

1. 夏**とくれば**、やっぱり海だ。
   → 說到夏天，果然還是海邊。
2. この町の名物**とくれば**、ラーメンに決まっている。
   → 說到這座城市的特產，毫無疑問是拉麵。
3. 彼女**とくれば**、いつも遅刻してくる。
   → 提到她，總是遲到。

## C1-026 ～ことだから
**意味**：因為是～所以（根據已知性格推測）  
**结构**：N + の + ことだから  
**説明**：基於對某人某物的了解進行推測或評論。

1. 真面目な彼**のことだから**、約束は必ず守るだろう。
   → 因為他是個認真的人，一定會遵守約定吧。
2. 遅刻常習犯のあいつ**のことだから**、今日も遅れてくるに違いない。
   → 因為他是慣性遲到的家夥，今天也肯定會遲到。
3. おしゃれな山田さん**のことだから**、パーティーには最高の服を着てくるだろう。
   → 因為山田小姐很時尚，她肯定會穿最好看的衣服來派對吧。

## C1-027 ～ことなしに
**意味**：不～就～（在沒有～的情況下）  
**结构**：V辞書形 + ことなしに  
**説明**：表示在沒有做前項的情況下進行後項，常用於正式場合。

1. 努力する**ことなしに**、成功はあり得ない。
   → 不努力就無法成功。
2. リスクを冒す**ことなしに**、大きな利益は得られない。
   → 不冒風險就無法獲得巨大利益。
3. 許可を得る**ことなしに**、この書類を公開してはならない。
   → 未經許可不得公開此文件。

## C1-028 ～ゆえ（に）
**意味**：因為～所以～（文學體）  
**结构**：N + の／である + ゆえ（に）／V普通形 + ゆえ（に）  
**説明**：表示原因理由，用於書面語、文學作品或正式演講。

1. 準備不足**ゆえに**、結果を出せなかった。
   → 因為準備不足，沒能拿出成果。
2. 彼は温厚**ゆえに**、人に騙されることもある。
   → 因為他為人溫厚，有時也會被人騙。
3. 女性である**がゆえに**、不利益を被ることがある。
   → 因為是女性，有時會遭遇不利待遇。

## C1-029 ～がゆえに
**意味**：因為～所以～（古典式原因）  
**结构**：N／な形容詞語幹 + である + がゆえに／V普通形 + がゆえに  
**説明**：與ゆえに同義，語氣更文雅、古典。

1. 貧しい**がゆえに**、十分な教育を受けられなかった。
   → 因為貧窮，沒能接受足夠的教育。
2. リーダーである**がゆえに**、孤独を感じることもある。
   → 因為是領導者，有時也會感到孤獨。
3. 彼を愛する**がゆえに**、厳しく接しているのだ。
   → 正是因為愛他，才對他嚴厲。

# 第5章 Purpose and Objective（目的・意圖）

## C1-030 ～べく
**意味**：為了～（正式書面語）  
**结构**：V辞書形 + べく  
**説明**：表示為了某個目的而行動，相當於「為了～」。

1. 合格**すべく**、毎日必死に勉強した。
   → 為了合格，每天拼命學習。
2. 問題を解決**すべく**、関係者が集まった。
   → 為了解決問題，相關人士聚集了。
3. 彼に会い**べく**、空港へ向かった。
   → 為了見他，前往了機場。

## C1-031 ～べからず
**意味**：不得～／禁止～（書面語）  
**结构**：V辞書形 + べからず  
**説明**：表示禁止，常見於告示牌、規則等。比「してはいけない」更正式。

1. 芝生に入る**べからず**。
   → 禁止進入草坪。
2. ここに自転車を止める**べからず**。
   → 此處禁止停放自行車。
3. 無断で持ち出す**べからず**。
   → 禁止擅自帶出。

## C1-032 ～べからざる
**意味**：不可～的／不該～的（書面語）  
**结构**：V辞書形 + べからざる  
**説明**：表示某事物或人不應該有的狀態或行為，用於修飾名詞。

1. 彼は教育者として**許すべからざる**行為をした。
   → 他做出了作為教育者不可原諒的行為。
2. 現代社会において、インターネットは**欠くべからざる**存在だ。
   → 在現代社會，網路是不可或缺的存在。
3. これは絶対に**譲るべからざる**原則だ。
   → 這是絕對不能讓步的原則。

## C1-033 ～まじき
**意味**：不該有的～（作為～不應有的行為）  
**结构**：V辞書形 + まじき + N  
**説明**：表示某人的行為不符合其身份或地位，用於批評。

1. 医者として**あるまじき**態度だ。
   → 作為醫生不該有的態度。
2. 教師の**言うまじき**発言だった。
   → 那是老師不該說的話。
3. それはプロとして**取るまじき**行動だ。
   → 那是作為專業人士不該採取的行動。

## C1-034 ～なくして（は）
**意味**：如果沒有～就（不可能）  
**结构**：N + なくして（は）  
**説明**：表示如果沒有前項，後項就不可能成立。與「なしには」類似。

1. 努力**なくして**成功はない。
   → 沒有努力就沒有成功。
2. 皆様のご協力**なくしては**、このプロジェクトは成功しなかった。
   → 沒有大家的合作，這個專案就不會成功。
3. 平和**なくして**、豊かな生活はあり得ない。
   → 沒有和平，就不可能有富裕的生活。

# 第6章 Condition and Hypothetical Advanced（條件・假定）

## C1-035 ～とあれば
**意味**：如果是～的話（條件）  
**结构**：N／普通形 + とあれば  
**説明**：表示假定條件，強調在前項條件下，後項是理所當然的。

1. 君の**ためとあれば**、何でもする。
   → 如果是為了你，我什麼都做。
2. 緊急の用事**とあれば**、すぐに伺います。
   → 如果有急事，我立刻過去。
3. 必要**とあれば**、いつでも連絡してください。
   → 如果有需要，請隨時聯繫。

## C1-036 ～ともなれば
**意味**：一旦到了～地步／說到～（強調程度）  
**结构**：N／V辞書形 + ともなれば  
**説明**：強調到了某種程度或階段時，自然會發生後項的情況。

1. 一流のシェフ**ともなれば**、食材の選び方から違う。
   → 一旦成為一流廚師，連食材的挑選方式都不同。
2. 彼女**ともなれば**、そんなミスはしない。
   → 如果是她的話，不會犯那種錯誤。
3. 大臣**ともなれば**、発言には細心の注意が必要だ。
   → 一旦成為大臣，發言就需要極其謹慎。

## C1-037 ～ともなると
**意味**：要是到了～時候／如果是～的話  
**结构**：N／V辞書形 + ともなると  
**説明**：與C1-036相似，表示到了某種狀況就必然會～。

1. プロの選手**ともなると**、練習の質が違う。
   → 如果是職業選手，練習的品質就不一樣。
2. 年末**ともなると**、どの店も忙しくなる。
   → 一到年底，每家店都忙起來。
3. 新社会人**ともなると**、責任感が出てくるものだ。
   → 一旦成為社會新鮮人，就會產生責任感。

## C1-038 ～となると
**意味**：如果是～的話／說到～  
**结构**：N／V普通形 + となると  
**説明**：提出某個話題或條件，表示如果是這樣的話，情況就不同了。

1. 自分でやる**となると**、話は別だ。
   → 如果要自己做的話，那就另當別論了。
2. 海外転勤**となると**、家族のことも考えなければならない。
   → 如果是調到國外的話，還得考慮家人的事。
3. 彼が反対する**となると**、計画を変更せざるを得ない。
   → 如果他反對的話，就不得不改變計劃了。

## C1-039 ～となれば
**意味**：如果變成那樣的話／既然這樣  
**结构**：N／V普通形 + となれば  
**説明**：表示如果前項成立或已知某種狀況，則會採取後項行動。

1. 彼がやると言った**となれば**、もう安心だ。
   → 既然他說要做了，那就放心了。
2. 緊急事態**となれば**、このマニュアルに従って行動せよ。
   → 如果是緊急狀況，請依照此手冊行動。
3. 最後のチャンス**となれば**、全力を尽くすしかない。
   → 如果這是最後的機會，就只能全力以赴了。

## C1-040 ～ものなら
**意味**：如果能～的話（假設難以實現的願望）  
**结构**：V可能形 + ものなら  
**説明**：表示假設某件難以實現的事，如果可能的話想做後項。或表示如果做了前項會導致嚴重後果。

1. やり直せる**ものなら**、人生をもう一度やり直したい。
   → 如果能夠重來的話，我想再重來一次人生。
2. 彼に会える**ものなら**、今すぐにでも会いたい。
   → 如果能見到他，我現在就想見。
3. そんなことをしよう**ものなら**、大変なことになる。
   → 如果要做那種事，事情可就嚴重了。

## C1-041 ～ことなら
**意味**：如果是～的話（表達希望）  
**结构**：N + のことなら  
**説明**：表示談到某個話題時，自己對其很有把握或了解。

1. 日本語の**ことなら**、彼に聞いてください。
   → 如果是日語的問題，請問他。
2. この地域の**ことなら**、私が一番詳しい。
   → 說到這個地區，我最熟悉。
3. 料理の**ことなら**、彼女に任せておけば間違いない。
   → 如果是做菜的事，交給她就沒問題。

## C1-042 ～としたって
**意味**：即使～也～（口語）  
**结构**：V普通形／い形容詞／な形容詞語幹 + としたって  
**説明**：表示讓歩，口語體。相當於「～にしても」。

1. 行きたい**としたって**、お金がないから行けない。
   → 即使想去，也沒錢所以去不了。
2. 買う**としたって**、今は予算が足りない。
   → 即使要買，現在預算也不夠。
3. 便利**としたって**、ここに住むのはちょっと高いよ。
   → 即使方便，住在這裡也有點貴。

## C1-043 ～にしたところで
**意味**：即使～也～（考慮到～也～）  
**结构**：N／V普通形 + にしたところで  
**説明**：表示讓歩，承認前項但指出也不過如此。

1. 景気が回復する**にしたところで**、雇用がすぐに改善するわけではない。
   → 即使景氣復甦，就業也不會立刻改善。
2. 彼に謝る**にしたところで**、許してもらえるとは限らない。
   → 即使向他道歉，也不一定會被原諒。
3. 新しいシステムを導入する**にしたところで**、効果がすぐに出るわけではない。
   → 即使導入新系統，效果也不會立刻顯現。

## C1-044 ～であれ
**意味**：無論～／即使是～（書面語）  
**结构**：N + であれ  
**説明**：表示無論前項如何，後項都不受影響。也可表示讓歩。

1. 結果**であれ**、全力を尽くしたことに後悔はない。
   → 無論結果如何，我對全力以赴毫不後悔。
2. どんな理由**であれ**、暴力は許されない。
   → 無論是什麼理由，暴力都不被允許。
3. 雨**であれ**、風**であれ**、試合は決行される。
   → 無論下雨還是刮風，比賽都會如期舉行。

**结构**：N + の極み  
**説明**：表示某種狀態達到了極點，常用於形容非常幸福、疲勞等。

## C1-046 ～の至り
**意味**：～之至／極其～（正式）  
**结构**：N + の至り  
**説明**：用於正式場合的習慣表達，如「光栄の至り」「若気の至り」等。

1. このような賞をいただき、**光栄の至り**です。

3. ご丁寧なもてなしに、**感激の至り**です。

## C1-047 ～限りだ
**意味**：非常～（感情達到極限）  
**结构**：い形容詞連体形／な形容詞な + 限りだ  
**説明**：與C1-012相同，再次收錄以完整結構段。

1. 同期の昇進を聞いて、**嬉しい限りだ**。

2. 長年連れ添った妻に先立たれて、**悲しい限りだ**。
   → 被多年相伴的妻子先行離世，悲痛至極。
3. 彼の無責任な態度には、**腹立たしい限りだ**。
   → 對他不負責任的態度，真是憤怒至極。

## C1-048 ～ったらない

**结构**：い形容詞／な形容詞語幹 + ったらない  
**説明**：強調程度極高，帶有感嘆語氣。口語。

1. 彼の驚いた顔**ったらなかった**。
   → 他那驚訝的表情，別提有多誇張了。
2. うちの猫のかわいさ**ったらない**。
   → 我家的貓可愛得不得了。
3. 試験の結果を見た時の彼の落ち込みよう**ったらなかった**。
   → 他看到考試成績時的低落勁，別提了。

## C1-049 ～といったらない
**意味**：～得無法形容  
**结构**：N + といったらない  
**説明**：強調程度極高，無法用語言形容。比「ったらない」稍正式。

1. 彼の料理のまずさ**といったらなかった**。
   → 他的料理難吃得無法形容。
2. 久しぶりに故郷に帰った時の懐かしさ**といったらなかった**。
   → 時隔多年回到家鄉時的懷念感，無法形容。
3. 彼女の演技の素晴らしさ**といったらなかった**。
   → 她的演技精彩得無法形容。

## C1-050 ～てはならない
**意味**：不可以～／不能～  
**结构**：Vて形 + は + ならない  
**説明**：表示禁止或強烈的否定，比「てはいけない」更正式、語氣更強。

1. このことは絶対に忘れ**てはならない**。
   → 這件事絕對不能忘記。
2. 二度と同じ過ちを繰り返し**てはならない**。
   → 不能再重複同樣的錯誤。
3. 人は外見だけで判断し**てはならない**。
   → 不能僅憑外表來判斷一個人。

## C1-051 ～てはたまらない
**意味**：～得受不了  
**结构**：Vて形 + は + たまらない  
**説明**：表示程度過高，讓人無法忍受。

1. 毎日暑く**てはたまらない**。
   → 每天這麼熱，真是受不了。
2. 彼に無視され**てはたまらない**。

3. 仕事が忙しすぎ**てはたまらない**から、転職を考えている。
   → 工作太忙受不了了，正在考慮換工作。

# 第9章 Evaluation and Judgment（評價・判斷）

**意味**：值得～／足以～  
**结构**：V辞書形 + に足る  
**説明**：表示有足夠的價值或值得做某事。

3. 推薦**するに足る**学生を紹介してください。

2. そんな小さな失敗は気**にするに足りない**。

3. 彼の業績は特に評価**するに足りない**。

## C1-059 ～にたえる
**意味**：值得～／經得起～  
**结构**：V辞書形 + にたえる  
**説明**：表示某物有足夠的價值去承受或值得被做。

1. この作品は鑑賞**するにたえる**。
   → 這部作品值得欣賞。
2. 彼の論文は学術的に読む**にたえる**内容だ。
   → 他的論文在學術上值得一讀。
3. その品質は厳しい検査**にたえる**ものだ。

# 第12章 Formal Written Conjunctions（書き言葉接続）

## C1-081 ～したがって
**意味**：因此／所以  
**结构**：文 + したがって + 文  
**説明**：表示因果關係，正式書面語。

1. 以上のことから、本件は解決済みと判断する。**したがって**、これ以上の調査は不要である。
   → 綜上所述，本案判斷為已解決。因此無需進一步調查。
2. 売上が大幅に減少した。**したがって**、人員削減を余儀なくされた。
   → 銷售額大幅減少。因此不得不裁員。
3. 法律で禁止されている。**したがって**、この行為は処罰の対象となる。
   → 法律禁止該行為。因此這一行為將受到處罰。

## C1-082 ～ゆえに
**意味**：因此／故而（書面語）  
**结构**：文 + ゆえに + 文  
**説明**：表示因果關係，文學或正式書面語。

1. 準備不足なり。**ゆえに**、目標達成は困難である。
   → 準備不足。因此難以達成目標。
2. 彼は常に誠実であった。**ゆえに**、多くの人に信頼されていた。
   → 他一向誠實。因此受到許多人的信賴。
3. 予算が限られている。**ゆえに**、優先順位を明確にすべきだ。
   → 預算有限。因此應明確優先順序。

## C1-083 ～ところが
**意味**：可是／然而  
**结构**：文 + ところが + 文  
**説明**：表示轉折，後項結果與前項預期相反。

1. 彼は優秀な選手だ。**ところが**、今回は全く実力を発揮できなかった。
   → 他是優秀的選手。然而這次完全沒能發揮實力。
2. 新しい薬を飲んだ。**ところが**、症状が悪化した。
   → 吃了新藥。結果症狀卻惡化了。
3. 何事も簡単に考えていた。**ところが**、実際はそれほど甘くなかった。
   → 對什麼都想得很簡單。然而實際並沒有那麼簡單。

## C1-084 ～ところで
**意味**：話說回來／對了（轉換話題）  
**结构**：文 + ところで + 文  
**説明**：用於轉換話題，引出新的話題。

1. 今日の会議はこれで終わりです。**ところで**、来週の予定は決まりましたか。
   → 今天的會議到此結束。對了，下週的計劃決定了嗎？
2. 最近仕事が忙しいです。**ところで**、ゴールデンウィークの予定はありますか。
   → 最近工作很忙。對了，黃金週有什麼計劃嗎？
3. そう言えば、田中さんは元気ですか。**ところで**、来月のパーティーには参加しますか。
   → 說起來，田中先生還好嗎？對了，下個月的派對你會參加嗎？

## C1-085 ～ただし
**意味**：但是／不過（帶條件或補充説明）  
**结构**：文 + ただし + 文  
**説明**：用於補充説明或附加條件，正式書面語。

1. 参加費は無料です。**ただし**、事前登録が必要です。
   → 參加費免費。但是需要提前註冊。
2. この製品は返品可能です。**ただし**、未開封のものに限ります。
   → 本產品可以退貨。但僅限未開封的。
3. 入場は自由です。**ただし**、動物の同伴はお断りします。
   → 入場自由。但禁止攜帶動物。

## C1-086 ～なお
**意味**：另外／再者（補充説明）  
**结构**：文 + なお + 文  
**説明**：用於補充説明或添加資訊。

1. 明日の集合時間は午前9時です。**なお**、場所は追って連絡します。
   → 明天集合時間是上午9點。另外地點稍後通知。
2. 申込期限は3月末日です。**なお**、定員になり次第締め切ります。
   → 報名截止日期為3月底。另外一旦額滿即截止。
3. 本イベントは雨天決行です。**なお**、荒天の場合は中止になります。
   → 本活動雨天照常舉行。另外天氣惡劣時將取消。

## C1-087 ～かつ
**意味**：並且／而且（書面語）  
**结构**：文／句 + かつ + 文／句  
**説明**：連接兩個並列的成分或句子。

1. 当社は品質**かつ**安全性を重視しています。
   → 本公司重視品質和安全性。
2. 彼は優秀**かつ**誠実な人物だ。
   → 他是優秀且誠實的人。
3. 安く**かつ**美味しいレストランを知っていますか。
   → 你知道又便宜又好吃的餐廳嗎？

## C1-088 ～すなわち
**意味**：即／也就是説  
**结构**：N／文 + すなわち + N／文  
**説明**：用於定義或換句話說。

1. これは我が社の最高傑作、**すなわち**、10年の技術の結晶です。
   → 這是我公司的最高傑作，即10年技術的結晶。
2. 成人、**すなわち**20歳以上の者には選挙権がある。
   → 成年人，也就是20歲以上的人有選舉權。
3. 彼は日本でいうところのサラリーマン、**すなわち**会社員だ。
   → 他就是日本所謂的工薪族，也就是公司職員。

## C1-089 ～要するに
**意味**：總而言之／簡而言之  
**结构**：文 + 要するに + 文  
**説明**：總結前述內容。

1. 長々と説明したが、**要するに**、金がないということだ。
   → 說了一大堆，總而言之就是沒錢。
2. 色々な意見が出たが、**要するに**、結論は一つだ。
   → 雖然出現了各種意見，但總而言之結論只有一個。
3. **要するに**、彼は責任を取りたくないのだ。
   → 總而言之，他不想承擔責任。

## C1-090 ～ただし（補足）
**意味**：但～（附加條件或例外）  
**结构**：文 + ただし + 文  
**説明**：與C1-085相同，再次強調補充條件的用法。

1. どなたでも応募できます。**ただし**、年齢制限があります。
   → 任何人都可以報名。但有年齡限制。
2. コピーは自由です。**ただし**、出典を明記してください。
   → 可以任意複製。但請注明出處。
3. この機械は誰でも使えます。**ただし**、使用後は必ず電源を切ってください。
   → 這台機器誰都可以使用。但使用後請務必關閉電源。

## C1-091 ～および
**意味**：以及／和（正式書面語）  
**结构**：N + および + N  
**説明**：連接兩個或多個並列的名詞。常用於法律、公文等正式文件。

1. 本契約の内容**および**条件に同意します。
   → 同意本合約之內容及條件。
2. 申込書**および**必要書類をご提出ください。
   → 請提交申請書及必要文件。
3. 営業時間は平日午前9時から午後5時まで**および**土曜日午前9時から正午までです。
   → 營業時間為平日上午9點至下午5點以及週六上午9點至中午。

## C1-092 ～並びに
**意味**：以及／和（法律・正式文件）  
**结构**：N + 並びに + N  
**説明**：比「および」更正式，用於法律文件等。

1. 本法律は、国**並びに**地方公共団体に適用される。
   → 本法適用於國家以及地方公共團體。
2. 製造元**並びに**販売元は、本製品の品質を保証する。
   → 製造商以及銷售商保證本產品的品質。
3. 社長**並びに**役員一同、心よりお祝い申し上げます。
   → 社長以及全體管理人員，謹致以誠摯的祝賀。

## C1-093 ～更に
**意味**：更／而且／進一步  
**结构**：文 + 更に + 文  
**説明**：表示在上述基礎上進一步添加信息。

1. 台風のため、電車は運転を見合わせています。**更に**、高速道路も通行止めです。
   → 因颱風，電車暫停運行。而且高速公路也禁止通行。
2. 彼は社内で最も優秀な営業マンだ。**更に**、後輩の指導にも熱心だ。
   → 他是公司內最優秀的業務員。而且對指導後輩也很熱心。
3. この製品は軽量で持ち運びやすい。**更に**、バッテリーの持ちも良い。
   → 本產品輕巧便攜。而且電池續航也很好。

## C1-094 ～しかしながら
**意味**：然而／儘管如此（正式）  
**结构**：文 + しかしながら + 文  
**説明**：表示轉折，比「しかし」更正式。

1. 努力した。**しかしながら**、結果は出なかった。
   → 努力了。然而沒有出成果。
2. 様々な対策を講じてきた。**しかしながら**、問題は依然として解決していない。
   → 已經採取了各種對策。然而問題依然沒有解決。
3. ご提案内容は大変魅力的です。**しかしながら**、予算の都合上お受けできません。
   → 您的提案內容非常有魅力。然而因為預算關係，無法接受。

## C1-095 ～とは言うものの
**意味**：雖說～但是～  
**结构**：普通形 + とは言うものの  
**説明**：表示承認前項事實，但後項提出保留或相反意見。

1. 留学したい**とは言うものの**、資金が足りない。
   → 雖說想去留學，但資金不足。
2. 新型コロナは落ち着いてきた**とは言うものの**、まだ予断を許さない。
   → 雖說新冠疫情已趨於平穩，但仍不能掉以輕心。
3. 良い大学を卒業した**とは言うものの**、仕事が見つからない。
   → 雖說畢業於好大學，但找不到工作。

## C1-096 ～とはいえ
**意味**：話雖如此／雖說～  
**结构**：普通形 + とはいえ  
**説明**：與C1-014相同，重新收錄以銜接順序。表示讓歩。

1. ダイエット中だ。**とはいえ**、たまには甘いものも食べたい。
   → 正在減肥。話雖如此，有時也想吃甜食。
2. 時間がかかった。**とはいえ**、良い作品ができた。
   → 雖然花了時間，但做出了好作品。
3. 値段が高い。**とはいえ**、品質を考えれば妥当だ。
   → 價格雖貴，但考慮到品質是合理的。

# 第13章 Emphasis and Focus（強調・焦點）

## C1-097 ～こそあれ
**意味**：～倒是～但（肯定前者但否定後者）  
**结构**：N + こそあれ  
**説明**：表示前項是真的，但後項不可能成立。書面語。

1. 問題**こそあれ**、解決策は見つかっていない。
   → 問題倒是存在，但解決方案還沒有找到。
2. 努力**こそあれ**、成果はまだ出ていない。
   → 努力倒是努力了，但還沒出成果。
3. 製品に改良の余地**こそあれ**、方向性は間違っていない。
   → 產品雖有改進餘地，但方向沒有錯。

## C1-098 ～こそすれ
**意味**：只會～但絕對不會（否定後者）  
**结构**：Vます形去ます + こそすれ  
**説明**：強調只會做前項，絕對不會做後項。

1. 彼に感謝**こそすれ**、恨むなんてありえない。
   → 只會感謝他，絕對不可能怨恨。
2. 皆様のご支援に感謝**こそすれ**、不満を持つはずがありません。
   → 只會感謝大家的支持，不可能有不滿。
3. 彼の行為は称賛**こそすれ**、非難されるべきものではない。
   → 他的行為只應受到讚揚，不應受到譴責。

## C1-099 ～ことこの上ない
**意味**：無比～／極其～  
**结构**：い形容詞連体形／な形容詞な + ことこの上ない  
**説明**：強調程度最高。

1. ご一緒に仕事ができて、**嬉しいことこの上ない**。
   → 能和您一起工作，無比高興。
2. 彼の無責任な態度には、**腹立たしいことこの上ない**。
   → 對他負責任的態度，極其憤怒。
3. 皆様に支えていただき、**感謝のことこの上ない**。
   → 得到大家的支持，感激不盡。

## C1-100 ～のなんのって
**意味**：別提有多～了（口語強調）  
**结构**：い形容詞／な形容詞語幹 + のなんのって  
**説明**：口語中強調程度極高，帶有感嘆語氣。

1. 彼の作った料理の美味しい**のなんのって**。
   → 他做的料理好吃得不得了。
2. 昨日の花火大会のすごかった**のなんのって**。
   → 昨天的煙火大會壯觀得不得了。
3. 彼女の驚きようったら**のなんのって**。
   → 她那驚訝的樣子，別提了。

## C1-101 ～うる／えない
**意味**：能夠～／不可能～  
**结构**：Vます形去ます + うる／えない  
**説明**：表示可能性。多為書面語，也可用於正式場合。

1. それは**ありうる**話だ。
   → 那是可能發生的事。
2. 彼の行動は理解し**うる**範囲だ。
   → 他的行為在可以理解的範圍內。
3. そんなことは絶対に**ありえない**。
   → 那種事絕對不可能。

## C1-102 ～は言うまでもない
**意味**：～是不用說的／～是毋庸置疑的  
**结构**：N／V普通形 + は言うまでもない  
**説明**：表示某事過於明顯，無需多言。

1. 環境問題の重要性**は言うまでもない**。
   → 環境問題的重要性不言而喻。
2. 健康が一番大切だという**ことは言うまでもない**。
   → 健康最重要這一點毋庸置疑。
3. 言う**までもなく**、安全が最優先だ。
   → 不用說，安全最優先。

## C1-103 ～を措いて（他にない）
**意味**：除了～之外（沒有別的）  
**结构**：N + を措いて（他にない）  
**説明**：表示除此之外沒有其他選擇或更合適的。

1. この仕事を任せられるのは、彼**を措いて他にいない**。
   → 能勝任這份工作的人，除了他之外沒有別人。
2. この問題の解決策は、この方法**を措いて考えられない**。
   → 這個問題的解決方案，除了這個方法之外想不出別的。
3. 日本料理の達人と言えば、あのシェフ**を措いて他にいない**。
   → 說到日本料理大師，除了那位主廚之外沒有別人。

# 第14章 Negation Patterns Advanced（否定表現）

## C1-104 ～ないではすまない
**意味**：不～是過不去的／必須要～  
**结构**：Vない形 + ないではすまない  
**説明**：表示如果不做某事，事情無法了結。帶有義務或責任意識。

1. このミスは謝ら**ないではすまない**。
   → 這個錯誤不道歉是過不去的。
2. 彼に借りた金は返さ**ないではすまない**。
   → 借他的錢不還是不行的。
3. 今回の事件については、責任者が説明し**ないではすまない**だろう。
   → 關於這次事件，負責人不做説明是過不去的吧。

## C1-105 ～ないではおかない
**意味**：一定會～／非要～不可（強烈意志）  
**结构**：Vない形 + ないではおかない  
**説明**：表示說話人的強烈意志或某事物必然導致某結果。

1. 彼は必ず成功させ**ないではおかない**。
   → 他一定要讓它成功。
2. この問題は社会に衝撃を与え**ないではおかない**。
   → 這個問題一定會給社會帶來衝擊。
3. 彼女の演技は観客を感動させ**ないではおかなかった**。
   → 她的演技讓觀眾無不動容。

## C1-106 ～ないではいられない
**意味**：忍不住～／不能不～（情緒性）  
**结构**：Vない形 + ないではいられない  
**説明**：表示無法抑制某種感情或行為。

1. 彼の姿を見ると、応援し**ないではいられない**。
   → 看到他的樣子，就忍不住想支持他。
2. この小説の結末を誰かに話さ**ないではいられない**。
   → 忍不住想把這部小説的結局告訴某人。
3. 悲しい知らせに、涙せ**ないではいられなかった**。
   → 聽到悲傷的消息，忍不住流淚了。

## C1-107 ～ないまでも
**意味**：即使不～也至少～  
**结构**：Vない形 + ないまでも  
**説明**：表示即使達不到某種程度，至少也應該做到另一程度。

1. 全額返済でき**ないまでも**、一部は返すべきだ。
   → 即使不能全額償還，至少也應該歸還一部分。
2. 毎日で**ないまでも**、週に一回は運動しよう。
   → 即使不是每天，至少一週運動一次吧。
3. 優勝でき**ないまでも**、決勝までは進みたかった。
   → 即使不能奪冠，也想至少打進決賽。

## C1-108 ～なくしては
**意味**：如果沒有～就（不可能）  
**结构**：N + なくしては  
**説明**：與C1-034相同，表示前項不存在時後項無法成立。

1. あなた**なくしては**、私は今の自分になれませんでした。
   → 沒有你的話，我就不會成為現在的我。
2. 日々の練習**なくしては**、この記録は達成できなかった。
   → 沒有每天的練習，就無法達成這個紀錄。
3. 改革**なくしては**、会社の未来はない。
   → 沒有改革的話，公司就沒有未來。

## C1-109 ～なしでは
**意味**：沒有～就不能～  
**结构**：N + なしでは  
**説明**：與C1-108相似，同樣表示不可或缺的條件。

1. パソコン**なしでは**、仕事が成り立たない。
   → 沒有電腦，工作就無法進行。
2. 辞書**なしでは**、この本を読むのは難しい。
   → 沒有字典，讀這本書很困難。
3. インターネット**なしでは**、現代の生活は考えられない。
   → 沒有網路，無法想像現代生活。

## C1-110 ～なしに
**意味**：不～就～（在沒有～的情況下做後項）  
**结构**：N + なしに  
**説明**：表示在沒有前項的狀態下進行後項。

1. 彼は断り**なしに**、私の部屋に入った。
   → 他沒打招呼就進了我的房間。
2. 準備**なしに**、本番に臨むのは無謀だ。
   → 沒有準備就上場是魯莽的。
3. 理由**なしに**、欠席してはいけない。
   → 沒有理由就不能缺席。

## C1-111 ～までもない
**意味**：用不著～／沒必要～  
**结构**：V辞書形 + までもない  
**説明**：表示某事不值得做或不需要做。

1. そんなことは言う**までもない**。
   → 那種事用不著說。
2. わざわざ行く**までもない**。電話で済ませよう。
   → 沒必要特意去一趟。用電話解決吧。
3. 詳しく説明する**までもない**。一目瞭然だ。
   → 沒必要詳細説明。一目了然。

## C1-112 ～には及ばない
**意味**：不必～／用不著～  
**结构**：V辞書形 + には及ばない  
**説明**：表示沒有必要做某事。

1. 心配**には及びません**。すべて順調です。
   → 不必擔心。一切順利。
2. わざわざお越し**には及びません**。こちらから伺います。
   → 您不必特意過來。我過去拜訪您。
3. 謝る**には及ばない**。君のせいじゃない。
   → 不必道歉。不是你的錯。

# 第15章 Nuanced Conditionals（條件ニュアンス）

## C1-113 ～とすれば
**意味**：如果那樣的話／假設～  
**结构**：V普通形／い形容詞／な形容詞だ + とすれば  
**説明**：對假設進行推論或判斷。

1. 彼が犯人だ**とすれば**、動機は何だ。
   → 如果他是犯人的話，動機是什麼？
2. この計画が失敗する**とすれば**、原因は資金不足だ。
   → 如果這個計劃失敗，原因就是資金不足。
3. 参加者が50人未満だ**とすれば**、この会場で十分だ。
   → 如果參加者不到50人的話，這個場地就夠了。

## C1-114 ～としたら
**意味**：假如～／假設～  
**结构**：V普通形／い形容詞／な形容詞だ + としたら  
**説明**：與C1-113基本同義，更口語化。假設不太可能或虛構的情況。

1. もし100万円当たった**としたら**、何に使いますか。
   → 如果中了100萬日圓，你會用來做什麼？
2. 今の仕事を辞める**としたら**、いつがいいだろう。
   → 如果要辭掉現在的工作，什麼時候好呢？
3. 彼女が実は宇宙人だ**としたら**、面白いのに。
   → 假如她其實是外星人，那就有趣了。

## C1-115 ～にしても
**意味**：即使～也～／就算～  
**结构**：N／V普通形 + にしても  
**説明**：表示讓歩或折中的判斷。

1. 子供**にしても**、もう少し礼儀をわきまえるべきだ。
   → 即使是孩子，也應該懂點禮貌。
2. 忙しい**にしても**、連絡ぐらいはできるだろう。
   → 即使再忙，至少也能聯繫一下吧。
3. 値段が安い**にしても**、品質は無視できない。
   → 就算價格便宜，也不能無視品質。

## C1-116 ～にしろ／せよ
**意味**：無論～還是～／即使～  
**结构**：N／V普通形 + にしろ／にせよ  
**説明**：表示無論哪種情況都一樣，或讓歩。

1. 賛成する**にしろ**、反対する**にしろ**、意見を聞かせてください。
   → 無論贊成還是反對，請告訴我你的意見。
2. 行く**にせよ**、行かない**にせよ**、準備だけはしておこう。
   → 無論去還是不去，先做好準備吧。
3. 高級レストラン**にしろ**、安い店**にしろ**、美味しければ同じだ。
   → 無論是高級餐廳還是便宜小店，好吃就一樣。

## C1-117 ～たものを
**意味**：要是～就好了（對已發生的事表示遺憾）  
**结构**：Vた形 + ものを  
**説明**：與C1-016相同，對未能實現的過去事表示惋惜。

1. 素直に謝っ**たものを**、意地を張って関係が悪くなった。
   → 要是老實道歉就好了，卻固執而導致關係惡化。
2. 早く病院に行っ**たものを**、我慢して悪化させた。
   → 要是早點去醫院就好了，卻硬撐導致惡化。
3. 一声かけてくれ**たものを**、無断で使ってしまった。
   → 要是跟我說一聲就好了，卻擅自使用了。

## C1-118 ～ばよかったのに
**意味**：要是～就好了（遺憾、建議）  
**结构**：Vば形 + よかったのに  
**説明**：對過去沒有做的事表示遺憾或輕微的責備。

1. ちゃんと勉強していれ**ばよかったのに**。
   → 要是好好學習就好了。
2. もう少し早く出発すれ**ばよかったのに**。
   → 要是早一點出發就好了。
3. 彼に相談すれ**ばよかったのに**、どうしてしなかったんだ。
   → 要是跟他商量就好了，為什麼沒有呢。

## C1-119 ～たらよかったのに
**意味**：要是～就好了（口語）  
**结构**：Vた形 + ら + よかったのに  
**説明**：與C1-118同，口語形式。

1. あの時、買っ**たらよかったのに**。
   → 那個時候買了就好了。
2. 早く言っ**たらよかったのに**。
   → 早點說就好了。
3. もっと野菜を食べ**たらよかったのに**。
   → 多吃點蔬菜就好了。

# 第16章 Formal Nouns Advanced（形式名詞高度）

## C1-120 ～次第だ
**意味**：因此～／就是這樣的原因（説明理由・経緯）  
**结构**：V普通形／N + の + 次第だ  
**説明**：用於説明事情的原委、理由或經過。

1. 以上のような経緯で、本プロジェクトは中止となった**次第です**。
   → 基於上述經過，本項目就此中止。
2. 皆様のご支援の結果、無事にゴールにたどり着いた**次第です**。
   → 在大家的支持下，我們順利達成了目標。
3. このたび、一身上の都合により退職する**次第です**。
   → 此次因個人原因，我將辭職。

## C1-121 ～次第で
**意味**：取決於～／根據～而定  
**结构**：N + 次第で  
**説明**：表示後項取決於前項的條件或情況。

# 第19章 Literary / Classical Patterns（文語表現）

## C1-148 ～ごとき
**意味**：像～那樣的（輕蔑或比喻）  
**结构**：N + の + ごとき  
**説明**：表示舉例或比喻，常用於帶有輕蔑、貶低的語氣。

1. お前**ごとき**が、私に意見できると思うな。

2. 彼**のごとき**人間は、どこにでもいる。
   → 像他那樣的人，到處都有。
3. 猫**のごとき**動きで、泥棒は侵入した。
   → 小偷以像貓一樣的動作潛入了。

## C1-149 ～ごとく
**意味**：像～那樣（文學體副詞用法）  
**结构**：N + の + ごとく／V辞書形 + がごとく  
**説明**：表示比喻或舉例，修飾後項動詞。

1. 彼は風**のごとく**現れ、去っていった。
   → 他像風一般出現，又離開了。
2. 雪**のごとく**白い肌をしている。
   → 擁有如雪一般白皙的肌膚。
3. 予想し**たごとく**、結果は最悪だった。
   → 正如預料的一樣，結果是最糟的。

**名言**：就像～一樣（古典結句用法）  
**结构**：N + の／が + ごとし  
**説明**：用於句末，表示比喻或類似。古典文語。

   → 人生如夢。
2. 彼の才能は光る星**のごとし**。

3. 時は矢**のごとし**。
   → 光陰似箭。

# 第20章 Question Words + か/も/でも Advanced

## C1-158 ～何しろ
**意味**：不管怎樣／總之／畢竟  
**结构**：何しろ + 文  
**説明**：用於強調理由或原因，引出說話人的判斷。

1. **何しろ**、彼は10年も日本に住んでいたから、日本語が上手だ。
   → 不管怎麼說，他在日本住了10年，所以日語很好。
2. **何しろ**、今日は大事なプレゼンがあるので、緊張している。
   → 總之今天有重要的簡報，所以很緊張。
3. **何しろ**、彼女は私の恩人だから、恩を返したい。
   → 畢竟她是我的恩人，所以想報恩。

## C1-159 ～何せ
**意味**：因為～（強調理由）  
**结构**：何せ + 文  
**説明**：口語，表示強調某個理由。常用搭配「何せ～だから」。

1. **何せ**初心者だから、ミスが多いのは仕方ない。
   → 因為是新手，錯誤多也是沒辦法。
2. **何せ**急に出発が決まったので、準備が間に合わなかった。
   → 因為突然決定出發，所以來不及準備。
3. **何せ**彼は有名人だから、どこへ行っても注目される。
   → 因為他是名人，無論去哪裡都受到關注。

## C1-160 ～何しろ～から
**意味**：畢竟～因為～  
**结构**：何しろ + 理由 + から  
**説明**：用於強調原因理由，口語。

1. **何しろ**日本に来たばかり**だから**、慣れないことばかりだ。
   → 畢竟剛來日本，不習慣的事情很多。
2. **何しろ**子どもの頃からの夢**だから**、ぜひ叶えたい。
   → 畢竟是從小以來的夢想，一定要實現。
3. **何しろ**相手がプロ**だから**、簡単には勝てない。
   → 畢竟對方是專業人士，不可能輕易取勝。

## C1-161 ～いずれにしても
**意味**：無論如何／不管怎樣  
**结构**：いずれにしても + 文  
**説明**：表示無論前項如何，後項結論不變。

1. **いずれにしても**、今日中に決断しなければならない。
   → 無論如何，必須在今天之內做出決定。
2. 賛成か反対か、**いずれにしても**意見を聞かせてください。
   → 贊成還是反對，不管怎樣請告訴我你的意見。
3. **いずれにしても**、準備だけはしておいたほうがいい。
   → 不管怎樣，最好做好準備。

## C1-162 ～どうせ
**意味**：反正／反正是（消極斷定）  
**结构**：どうせ + 文  
**説明**：表示說話人帶有放棄或消極的語氣，認為結果已定。

1. **どうせ**私なんか、誰からも必要とされていない。
   → 反正像我這樣的人，誰都不需要。
2. **どうせ**間に合わないから、ゆっくり行こう。
   → 反正來不及了，慢慢走吧。
3. **どうせ**彼は来ないだろう。先に始めよう。

## C1-163 ～どうにか

2. 雨だったが、**どうにか**試合は行われた。

3. お金は**どうにか**なるだろう。
   → 錢總會有辦法解決吧。

## C1-164 ～なんとか
**意味**：想辦法／好歹  
**结构**：なんとか + 文

**意味**：差不多是～／大概～的程度  
**结构**：V普通形／N + というところだ  
**説明**：表示大致在某個範圍或程度。

1. 完成まであと一週間**というところだ**。
   → 距離完成大概還有一週左右。
2. 今月の売上は先月並み**というところだ**。
   → 這個月的銷售額大概和上個月差不多。
3. 彼の実力は中級**というところだろう**。
   → 他的實力大概在中級左右。

## C1-186 ～というものでもない
**意味**：也不是說～（並非絕對）  
**结构**：普通形 + というものでもない

2. 勉強すれば必ず合格できる**というものでもない**。
   → 並非學習了就一定能及格。
3. 若ければ良い**というものでもない**。経験も大切だ。
   → 並非年輕就好。經驗也很重要。

## C1-187 ～というのは～からだ
**意味**：之所以～是因為～  
**结构**：文 + というのは + 理由 + からだ  
**説明**：後項説明前項的理由。

1. 彼が失敗した**というのは**、準備不足だった**からだ**。
   → 他之所以失敗，是因為準備不足。
2. この地域が発展した**というのは**、交通の便が良かった**からだ**。
   → 這個地區之所以發展，是因為交通便利。
3. 私が日本語を勉強している**というのは**、日本で働きたい**からだ**。
   → 我之所以在學日語，是因為想在日本工作。

## C1-188 ～とはそういうものだ

3. 勝負**とはそういうものだ**。勝つか負けるかだ。
   → 勝負就是那樣的。不是贏就是輸。
