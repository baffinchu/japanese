> **該当目安：JLPT N3相當 / 學習時數約600小時**
>
> B1是「獨立階段的語言使用者」。能在旅行時應對常見狀況，能對熟悉或感興趣的主題撰寫簡單連貫的文章，能描述經驗、事件、夢想、希望和抱負，並簡短說明意見和計劃。能理解工作、學校、休閒場合中常見標準語的重點。

# 第1章 受身形（Passive Voice）

## B1-001 ～れる/られる（直接受身）
**意味**：被～（直接被動）  
**结构**：Vない形（ない→れる/られる）  
**説明**：直接被動，動作的承受者為主語。

1. わたしは先生に**褒められた**。
   → 我被老師表揚了。
2. この本は世界中で**読まれている**。
   → 這本書在世界各地被閱讀。
3. 昨日、財布を**盗まれた**。
   → 昨天錢包被偷了。

## B1-002 ～れる/られる（間接受身 - 迷惑）
**意味**：被～（間接被動・受害）  
**结构**：Vない形（ない→れる/られる）  
**説明**：間接被動，主語因他人動作間接受害。

1. 隣の人に夜遅く**騒がれて**、眠れなかった。
   → 被鄰居夜深吵鬧，睡不著。
2. 電車の中で赤ちゃんに**泣かれた**。
   → 在電車裡被嬰兒哭了（間接受害）。
3. 雨に**降られて**、びしょぬれになった。
   → 被雨淋了，全身濕透。

## B1-003 ～によって（による - 行為者・正式）
**意味**：由～（被動行為者・正式）  
**结构**：N + によって + 受身文  
**説明**：正式文體中表示被動的行為者。

1. この寺は有名な建築家**によって**建てられた。
   → 這座寺廟由著名的建築家建造。
2. アメリカ**によって**多くの法律が改正された。
   → 美國修改了許多法律。
3. この小説は夏目漱石**によって**書かれた。
   → 這部小說由夏目漱石所寫。

## B1-004 ～に（受身の動作主）
**意味**：被～（被動行為者）  
**结构**：N + に + 受身文  
**説明**：表示被動句中的行為者，較口語。

1. わたしは母**に**叱られた。
   → 我被媽媽罵了。
2. 犬**に**噛まれたことがある。
   → 被狗咬過。
3. 彼は友達**に**笑われた。
   → 他被朋友笑了。

## B1-005 ～から（受身の動作主）
**意味**：被～（被動行為者・來源）  
**结构**：N + から + 受身文  
**説明**：從某人處接受動作，強調來源。

1. 社長**から**褒められた。
   → 被社長表揚了。
2. お客様**から**苦情を言われた。
   → 被客人投訴了。
3. 先生**から**宿題を出された。
   → 被老師布置了作業。

# 第2章 使役形（Causative Form）

## B1-006 ～せる/させる（強制使役）
**意味**：讓／使～做（強制）  
**结构**：Vない形（ない→せる/させる）  
**説明**：強制或指示他人做某事。

1. 母は子供に野菜を**食べさせる**。
   → 媽媽讓孩子吃蔬菜。
2. 先生は学生にレポートを**書かせた**。
   → 老師讓學生寫報告。
3. 父はわたしに毎日練習**させた**。
   → 父親讓我每天練習。

## B1-007 ～せる/させる（許可使役）
**意味**：讓～做（許可）  
**结构**：Vない形（ない→せる/させる）  
**説明**：允許或放任他人做某事。

1. 子供に好きなことを**させて**いる。
   → 讓孩子做喜歡的事情。
2. 彼に思い切り遊ば**せて**やろう。
   → 讓他盡情玩吧。
3. 自由に意見を言わ**せる**雰囲気が大事だ。
   → 讓大家自由發表意見的氣氛很重要。

## B1-008 ～せてください
**意味**：請讓我做～  
**结构**：Vない形（ない→せて/させて）+ ください  
**説明**：請求許可做某事。

1. すみません、一言**言わせてください**。
   → 不好意思，請讓我说一句話。
2. わたしにも**考えさせてください**。
   → 請讓我也想一下。
3. 今日は早く**帰らせてください**。
   → 今天請讓我早點回去。

## B1-009 ～せられる/させられる（使役受身）
**意味**：被迫做～  
**结构**：Vない形（ない→せられる/させられる）  
**説明**：使役＋被動，表示被強迫做某事。

1. 毎日殘業**させられた**。
   → 每天被迫加班。
2. 子供のころ、嫌いなものを無理やり**食べさせられた**。
   → 小時候被逼著吃不喜歡的東西。
3. 彼にずっと待た**せられた**。
   → 被他一直等著（被迫一直等）。

## B1-010 ～させる（許可用法）
**意味**：讓～做（許可）  
**结构**：Vない形（ない→せる/させる）  
**説明**：許可某人做某事。

1. 子供を外で遊ば**せる**。
   → 讓孩子在外面玩。
2. 学生に自由にテーマを選ば**せる**。
   → 讓學生自由選擇主題。
3. 彼の希望どおりに**させて**やろう。
   → 按照他的意願讓他去做吧。

## B1-011 ～せてやる/あげる（讓～做・恩惠）
**意味**：讓～做（施惠）  
**结构**：Vない形（ない→せて/させて）+ やる/あげる  
**説明**：允許或讓某人做某事，出於好意。

1. 今日は子供に好きなものを**買わせてやった**。
   → 今天讓孩子買了喜歡的東西。
2. 彼女に思い切り**歌わせてあげよう**。
   → 讓她盡情唱歌吧。
3. たまには彼に**休ませてあげよう**。
   → 偶爾讓他休息一下吧。

# 第3章 敬語（Honorific and Humble）

## B1-012 尊敬語：特殊形
**意味**：尊他語：特殊形式  
**结构**：特殊尊敬語動詞  
**説明**：使用專用尊敬語動詞表示對他人的敬意。

1. 先生はもう**いらっしゃいました**か。
   → 老師已經來了嗎？
2. 社長は何を**おっしゃいました**か。
   → 社長說了什麼？
3. お客様は何を**召し上がります**か。
   → 客人您要吃什麼？

## B1-013 尊敬語：お～になる
**意味**：尊他語：お～になる  
**结构**：お + Vます（去ます）+ になる  
**説明**：構成尊敬語的一般形式。

1. 先生はもう**お帰りになりました**。
   → 老師已經回去了。
2. 社長はいつ**お戻りになります**か。
   → 社長什麼時候回來？
3. お客様は何を**お探しですか**。
   → 客人在找什麼？

## B1-014 謙譲語：特殊形
**意味**：謙讓語：特殊形式  
**结构**：特殊謙讓語動詞  
**説明**：使用專用謙讓語表示自謙。

1. わたしが**参ります**。
   → 我來（去）。
2. かしこまりました。すぐに**承ります**。
   → 明白了。我這就去辦。
3. こちらで**拝見します**。
   → 我在這裡看（拜讀）。

## B1-015 謙譲語：お～する/ご～する
**意味**：謙讓語：お～する/ご～する  
**结构**：お + Vます（去ます）+ する / ご + 漢語名詞 + する  
**説明**：構成謙讓語的一般形式。

1. 荷物を**お持ちします**。
   → 我來拿行李。
2. 後ほど**ご連絡します**。
   → 稍後我聯絡您。
3. 私が**ご案内します**。
   → 我來帶路。

## B1-016 丁寧語（ですます体）
**意味**：禮貌語（ですます體）  
**结构**：です・ます・ございます  
**説明**：基本的禮貌用語。

1. こちらは**新しい商品でございます**。
   → 這是新商品。
2. 本日はお忙しいところ、**ありがとうございます**。
   → 今天在您百忙之中，非常感謝。
3. 少々**お待ちくださいませ**。
   → 請稍等。

## B1-017 敬語の使い分け（Register Switching）
**意味**：敬語的區分使用  
**结构**：根據場合選擇適當的敬語程度  
**説明**：根據對象和場合切換敬語等級。

1. 社内では普通體、社外では敬語を使う。
   → 公司內部用普通體，對外用敬語。
2. 上司には「おっしゃいました」、同僚には「言った」。
   → 對上司用「おっしゃいました」，對同事用「言った」。
3. 接客時は丁寧語、友達とはタメ語。
   → 接待客人時用禮貌語，和朋友之間用平輩語。

# 第4章 條件表現（Complex Conditionals）

## B1-018 ～と（必然結果）
**意味**：一～就～（必然結果）  
**结构**：V辞書形/ない形 + と  
**説明**：表示必然的結果、習慣、自然現象。

1. 春になる**と**、桜が咲きます。
   → 一到春天，櫻花就開。
2. このボタンを押す**と**、電源が入ります。
   → 一按這個按鈕，電源就打開。
3. 毎日運動する**と**、健康になる。
   → 每天運動的話，就會變健康。

## B1-019 ～ば（一般條件）
**意味**：如果～就～（一般條件）  
**结构**：V仮定形 + ば  
**説明**：表示一般性或普遍性的條件。

1. 安けれ**ば**買います。
   → 如果便宜就買。
2. 雨が降れ**ば**、試合は中止です。
   → 如果下雨，比賽就中止。
3. 努力すれ**ば**、夢は叶う。
   → 如果努力，夢想就會實現。

## B1-020 ～たら（特定條件）
**意味**：如果～了的話  
**结构**：Vた形 + ら  
**説明**：表示特定條件，常用於假設或既定條件。

1. 東京に着い**たら**、電話してください。
   → 到了東京請打電話。
2. 時間があっ**たら**、手伝ってください。
   → 如果有時間，請幫忙。
3. 安かっ**たら**、買おうと思う。
   → 如果便宜的話，我想買。

## B1-021 ～なら（上下文條件）
**意味**：如果～的話（基於對方話題）  
**结构**：V普通形/名 + なら  
**説明**：根據對方提出的話題或狀況進行假設。

1. 日本に行く**なら**、秋がいいですよ。
   → 如果去日本的話，秋天比較好喔。
2. 映画を見る**なら**、この作品がおすすめです。
   → 如果看電影的話，我推薦這部作品。
3. 山田さん**なら**、もう帰りましたよ。
   → 如果是山田先生的話，他已經回去了喔。

## B1-022 ～ても（讓歩）
**意味**：即使～也～  
**结构**：Vて形 + も  
**説明**：表示讓歩，即使某條件成立結果也不變。

1. 雨が降っ**ても**、試合は行われます。
   → 即使下雨，比賽也會舉行。
2. 高く**ても**、この車を買いたい。
   → 即使貴，我也想買這輛車。
3. 謝っ**ても**、許してもらえないだろう。
   → 即使道歉，大概也不會被原諒吧。

## B1-023 ～たって（讓歩・口語）
**意味**：即使～也～（口語）  
**结构**：Vた形 + って  
**説明**：「ても」的口語縮約形。

1. 急い**だって**、間に合わないよ。
   → 即使趕也來不及了喔。
2. 泣い**たって**、仕方がない。
   → 即使哭也沒辦法。
3. 今さら言っ**たって**、遅いんだから。
   → 事到如今再說也晚了。

## B1-024 ～ところで（ futile 讓歩）
**意味**：即使～也（無用）  
**结构**：Vた形 + ところで  
**説明**：即使做了某事也沒有意義。

1. 今から行った**ところで**、間に合わない。
   → 即使現在去也來不及了。
2. 謝った**ところで**、彼女は許してくれないだろう。
   → 即使道歉，她大概也不會原諒我吧。
3. 急いだ**ところで**、結果は同じだ。
   → 即使趕，結果也一樣。

## B1-025 ～ないことには（除非）
**意味**：除非～否則不～  
**结构**：Vない形 + ことには  
**説明**：表示如果不滿足某條件，某事就不能成立。

1. 実際に見**ないことには**、良さはわからない。
   → 除非實際看過，否則不知道好在哪裡。
2. 彼の了承を得**ないことには**、計畫は進められない。
   → 除非得到他的同意，否則無法推進計畫。
3. 努力し**ないことには**、成功はない。
   → 除非努力，否則不會成功。

# 第5章 接続・原因（Sentence Connectives - Reason）

## B1-026 ～から（主観的理由）
**意味**：因為～（主觀理由）  
**结构**：普通形 + から  
**説明**：表示主觀理由、原因。

1. 天気がいい**から**、散歩に行きましょう。
   → 因為天氣好，去散步吧。
2. お腹がすいた**から**、何か食べよう。
   → 因為肚子餓了，吃點什麼吧。
3. 日本語が難しい**から**、毎日勉強しています。
   → 因為日語很難，所以每天在學習。

## B1-027 ～ので（客観的理由）
**意味**：因為～（客觀理由）  
**结构**：普通形 + ので  
**説明**：表示客觀理由，比「から」更禮貌、正式。

1. 風邪を引いた**ので**、今日は休みます。
   → 因為感冒了，今天請假。
2. 電車が遅れた**ので**、遅刻しました。
   → 因為電車延誤，所以遲到了。
3. 値段が高い**ので**、買うのをやめました。
   → 因為價格貴，所以不買了。

## B1-028 ～ため（に）（正式理由）
**意味**：因為～（正式理由）  
**结构**：V普通形/Nの + ため（に）  
**説明**：表示正式書面或客觀原因。

1. 台風**のため**、イベントは中止になりました。
   → 因為颱風，活動取消了。
2. 病気**治療のため**、休職します。
   → 因為疾病治療，將休職。
3. 円安**のため**、輸出台数が増加した。
   → 因為日圓貶值，出口台數增加了。

## B1-029 ～おかげで（正面理由）
**意味**：多虧了～（正面）  
**结构**：Vた形/Nの + おかげで  
**説明**：表示好的結果歸功於某事物或某人。

1. 先生**のおかげで**、日本語が上手になりました。
   → 多虧了老師，日語進步了。
2. 友達が助けてくれた**おかげで**、間に合いました。
   → 多虧朋友幫了我，才趕上了。
3. 新しい薬**のおかげで**、病気が治った。
   → 多虧了新藥，病治好了。

## B1-030 ～せいで（負面理由）
**意味**：因為～（負面原因、歸咎）  
**结构**：V普通形/Nの + せいで  
**説明**：表示不好的結果是由於某事物或某人造成的。

1. 雨**のせいで**、試合が中止になった。
   → 因為下雨，比賽取消了。
2. 彼**のせいで**、大事な約束を忘れた。
   → 因為他，忘了重要的約定。
3. 睡眠不足**のせいで**、頭がぼんやりしている。
   → 因為睡眠不足，頭腦昏昏沉沉的。

## B1-031 ～ものだから（理由・言い訳）
**意味**：因為～（辯解理由）  
**结构**：V普通形/い形/な形な + ものだから  
**説明**：表示說話者為自己的行為辯解的理由。

1. 財布を忘れた**ものだから**、貸してくれませんか。
   → 因為忘了錢包，能借我一點嗎？
2. 一人で心細かった**ものだから**、電話しました。
   → 因為一個人覺得不安，所以打了電話。
3. 約束をすっかり忘れていた**ものだから**、急いで來たんです。
   → 因為完全忘了約定，所以趕來了。

## B1-032 ～もので（理由・説明）
**意味**：因為～（說明理由）  
**结构**：普通形 + もので  
**説明**：「ものだから」的正式說法，表示理由說明。

1. 道が分からなかった**もので**、遅れてしまいました。
   → 因為不認識路，所以遲到了。
2. 急に用事ができた**もので**、行けなくなりました。
   → 因為突然有事，所以不能去了。
3. 体調が優れなかった**もので**、早退させていただきます。
   → 因為身體不適，請允許我早退。

## B1-033 ～ことから（由～可知）
**意味**：從～來看／因為～  
**结构**：V普通形/Nである + ことから  
**説明**：表示基於某事態推斷出結論或命名由來。

1. 遺跡が見つかった**ことから**、この地域に古代文明があったと考えられる。
   → 從發現遺跡來看，可以推斷這個地區曾有古代文明。
2. 態度が不自然だった**ことから**、彼が噓をついていると分かった。
   → 從態度不自然來看，知道他在說謊。
3. 形が富士山に似ている**ことから**、「富士山型」と呼ばれている。
   → 因為形狀像富士山，所以被稱為「富士山型」。

# 第6章 讓歩・對比（Concession and Contrast）

## B1-034 ～のに（逆接・不滿）
**意味**：雖然～卻～  
**结构**：普通形 + のに  
**説明**：表示逆接，帶有意外、不滿、遺憾等語氣。

1. 勉強した**のに**、試験に落ちた。
   → 雖然學習了，卻沒考過。
2. 雨が降っている**のに**、出かけるの？
   → 雖然在下雨，你還要出門嗎？
3. あれほど注意した**のに**、また同じミスをした。
   → 雖然提醒了那麼多次，又犯了同樣的錯誤。

## B1-035 ～けれども/けど（逆接・前置き）
**意味**：雖然～但是～  
**结构**：普通形 + けれども/けど

# 第27章 連語表現（Fixed Expressions）

## B1-176 ～ことがある（有時～）
**意味**：有時～  
**结构**：V辞書形 + ことがある  
**説明**：表示有時會發生某事。

1. 時々、一人で映畫を見る**ことがある**。
   → 有時候會一個人看電影。
2. 彼は遅刻する**ことがよくある**。
   → 他經常遲到。
3. 週末は何もしないで過ごす**こともある**。
   → 週末也有什麼都不做度過的時候。

## B1-177 ～こともある（也有時～）
**意味**：也有時候～  
**结构**：V辞書形 + こともある  
**説明**：表示有時也存在另一種情況。

1. 忙しいと、晩ごはんを食べない**こともある**。
   → 忙的時候也有不吃晚飯的時候。
2. 日本語が難しいと感じる**こともある**。
   → 也有感到日語很難的時候。
3. 彼女は優しい**こともある**が、厳しい**こともある**。
   → 她也有溫柔的時候，也有嚴格的時候。

## B1-178 ～たことがある（有過～經驗）
**意味**：有過～的經驗  
**结构**：Vた形 + ことがある  
**説明**：表示曾經歷過某事。

1. 富士山に登った**ことがあります**か。
   → 你爬過富士山嗎？
2. 私は一度も海外に行った**ことがない**。
   → 我一次也沒出過國。
3. 彼に會った**ことがある**ような気がする。
   → 感覺好像見過他。

## B1-179 ～たばかり（剛～）
**意味**：剛～  
**结构**：Vた形 + ばかり  
**説明**：表示動作剛剛結束。

1. 今、家に着いた**ばかりです**。
   → 剛到家。
2. この本は昨日買った**ばかりなのに**、もう読んでしまった。
   → 這本書昨天剛買，就已經讀完了。
3. 彼は卒業した**ばかりの**新入社員だ。
   → 他是剛畢業的新進員工。

## B1-180 ～たところ（剛剛～）
**意味**：剛剛～（正式）  
**结构**：Vた形 + ところ  
**説明**：表示動作或變化剛完成的瞬間。

1. 今、電話をかけた**ところです**。
   → 剛剛打了電話。
2. 報告書は提出した**ところだ**。
   → 報告剛提交。
3. 彼は今出かけた**ところです**。
   → 他剛剛出門。

## B1-181 ～つもりだ（打算做～）
**意味**：打算做～  
**结构**：V辞書形/ない形 + つもりだ  
**説明**：表示自己的意圖或打算。

1. 來年、留學する**つもりです**。
   → 打算明年留學。
2. 彼には言わ**ないつもりだ**。
   → 不打算告訴他。
3. ずっとここに住む**つもりですか**。
   → 打算一直住在這裡嗎？

## B1-182 ～予定だ（預定～）
**意味**：預定～  
**结构**：V辞書形/Nの + 予定だ  
**説明**：表示計畫或預定。

1. 來週、大阪に出張する**予定です**。
   → 預定下週去大阪出差。
2. 會議は3時に終了する**予定だ**。
   → 會議預定3點結束。
3. 明日は何の**予定もありません**。
   → 明天沒有什麼預定。

## B1-183 ～つもりはない（不打算～）
**意味**：不打算～  
**结构**：V辞書形 + つもりはない  
**説明**：表示否定意圖。

1. 今のところ、結婚する**つもりはありません**。
   → 目前不打算結婚。
2. 彼に謝る**つもりはない**。
   → 不打算向他道歉。
3. 私は彼の意見に反対する**つもりはない**。
   → 我不打算反對他的意見。

# 第28章 文末表現（Advanced Sentence Endings）

## B1-184 ～かもしれない（也許～）
**意味**：也許～／可能～  
**结构**：普通形 + かもしれない  
**説明**：表示不確定的推測。

1. 明日は雨**かもしれない**。
   → 明天也許會下雨。
2. 彼はもう帰った**かもしれない**。
   → 他也許已經回去了。
3. この道のほうが近い**かもしれません**。
   → 這條路可能比較近。

## B1-185 ～に決まっている（肯定～）
**意味**：肯定～／一定～  
**结构**：普通形 + に決まっている  
**説明**：表示非常確定的推斷。

1. そんなことをすれば、怒られる**に決まっている**。
   → 做那種事肯定會被罵。
2. 彼が來る**に決まっている**。
   → 他肯定會來。
3. 高いのに質が悪い。買い**たくないに決まっている**。
   → 又貴品質又差，肯定不想買。

## B1-186 ～に違いない（一定～）
**意味**：一定～  
**结构**：普通形 + に違いない  
**説明**：表示很有把握的推斷。

1. 彼は成功する**に違いない**。
   → 他一定會成功。
2. この味は彼女が作った**に違いない**。
   → 這個味道一定是她做的。
3. あの雲の様子からすると、雨が降る**に違いない**。
   → 從那些雲的樣子來看，一定會下雨。

## B1-187 ～はずだ（應該～）
**意味**：按理應該～  
**结构**：普通形 + はずだ  
**説明**：表示根據常理的推測。

1. 彼はもう知っている**はずだ**。
   → 他應該已經知道了。
2. 今日は休みの**はずなのに**、誰もいない。
   → 今天應該是休息日，卻沒人。
3. そこにある**はずです**。もう一度探してください。
   → 應該在那裡。請再找找看。

## B1-188 ～わけだ（難怪／就是說～）
**意味**：難怪～／就是說～  
**结构**：普通形 + わけだ  
**説明**：表示理解或確認某種道理。

1. 彼女は日本に10年住んでいた。日本語が上手な**わけだ**。
   → 她在日本住了10年。難怪日語很好。
2. つまり、間に合わなかったという**わけだ**。
   → 也就是說，沒趕上。
3. 値段が高い**わけだ**。素材が特別なものだった。
   → 難怪價格貴，原來材料是特別的。

## B1-189 ～ものだ（真是～／本來就～）
**意味**：真是～／本來就～  
**结构**：普通形 + ものだ  
**説明**：表示感慨、一般真理或常識。

1. 時間が経つのは早い**ものだ**。
   → 時間過得真快啊。
2. 人間は誰でも間違いをする**ものだ**。
   → 人本來就會犯錯。
3. 子供は元気に外で遊ぶ**ものだ**。
   → 小孩子就應該在外面精力充沛地玩。

## B1-190 ～ものか（怎能～／絕不～）
**意味**：怎能～／絕不～（反問）  
**结构**：V辞書形/い形 + ものか  
**説明**：表示強烈的否定或反問。

1. こんな店に二度と來る**ものか**。
   → 這種店我絕不會再來了。
2. 彼に負ける**ものか**。
   → 我怎能輸給他。
3. そんなうそを信じる**ものですか**。
   → 我才不會相信那種謊話。

## B1-191 ～ことか（多麼～啊）
**意味**：多麼～啊（感嘆）  
**结构**：V普通形/い形 + ことか  
**説明**：表示強烈的感嘆。

1. 何度失敗した**ことか**。
   → 失敗了多少次啊。
2. 彼のことをどれほど心配した**ことか**。
   → 多麼擔心他啊。
3. この日をどれだけ待ち望んだ**ことか**。
   → 多麼盼望這一天啊。

# 第29章 日常連語表現（Everyday Fixed Expressions）

## B1-192 ～という（叫做～）
**意味**：叫做～／～的  
**结构**：N + という + N  
**説明**：表示名稱或內容。

1. **「はな」という**名前の貓を飼っている。
   → 養了一隻叫做「小花」的貓。
2. **山田という**人が來ていた。
   → 有個叫山田的人來了。
3. これは**日本語という**言語です。
   → 這是一種叫做日語的語言。

## B1-193 ～というと（說起～）
**意味**：說起～／提到～  
**结构**：N + というと  
**説明**：提起某事，引出聯想或確認。

1. **日本料理というと**、まず壽司を思い浮かべる。
   → 說到日本料理，首先想到壽司。
2. **休日というと**、どこへ行くのが好きですか。
   → 說到假日，你喜歡去哪裡？
3. **彼というと**、いつも笑顏を絕やさない人だ。
   → 說到他，他是個總是面帶笑容的人。

## B1-194 ～というわけで（因此～）
**意味**：因此～／就這個原因～  
**结构**：文 + というわけで  
**説明**：總結原因或理由並引出結果。

1. **というわけで**、今日はこれで終わります。
   → 因此，今天到此結束。
2. **そういうわけで**、私は辭職することにしました。
   → 因此，我決定辭職。
3. **というわけで**、みんなで協力して頑張りましょう。
   → 因此，大家一起合作努力吧。

## B1-195 ～ということだ（聽說～／就是說～）
**意味**：聽說～／就是說～  
**结构**：普通形 + ということだ  
**説明**：表示傳聞或解釋說明。

1. 社長は來月退任する**ということだ**。
   → 聽說社長下個月卸任。
2. つまり、明日は休みだ**ということですか**。
   → 也就是說，明天休息是嗎？
3. 彼女は結婚する**ということだ**。
   → 聽說她要結婚了。

## B1-196 ～というのは（所謂～）
**意味**：所謂～／～是指～  
**结构**：普通形/N + というのは  
**説明**：用於解釋詞語或話題的定義。

1. **「一期一會」というのは**、一生に一度の出會いを大切にするという意味だ。
   → 「一期一會」是指珍惜一生只有一次的相遇的意思。
2. **敬語というのは**、相手への敬意を表す言葉だ。
   → 所謂敬語，是表達對對方敬意的語言。
3. **AIというのは**、人工知能のことだ。
   → AI是指人工智慧。

## B1-197 ～とのことだ（據說～・正式）
**意味**：據說～（正式傳聞）  
**结构**：普通形 + とのことだ  
**説明**：「そうだ」的正式傳聞形式。

1. 明日の會議は延期**とのことです**。
   → 據說明天的會議延期了。
2. 彼は無事に帰國した**とのことだ**。
   → 據說他已平安回國。
3. 來月、新製品が発売される**とのことです**。
   → 據說下個月新產品將發售。

# 第30章 困難・可能性（Difficulty and Possibility）

## B1-198 ～かねる（難以～）
**意味**：難以～／不能～  
**结构**：Vます形 + かねる  
**説明**：表示心理上難以做到或委婉拒絕。

1. その要求には**応じかねます**。
   → 難以滿足那個要求。
2. 私の力では**判斷しかねます**。
   → 以我的能力難以判斷。
3. 殘念ながら、その質問には**お答えしかねます**。
   → 很遺憾，難以回答那個問題。

## B1-199 ～得る/得ない（可能／不可能）
**意味**：可能～／不可能～  
**结构**：Vます形 + 得る/得ない  
**説明**：表示客觀的可能性。

1. そんなことは**あり得ない**。
   → 那種事不可能有。
2. これは十分に**起こり得る**話だ。
   → 這是很有可能發生的事。
3. 人間は誰でも間違いを**犯し得る**。
   → 任何人都可能犯錯。

## B1-200 ～がたい（難以～・書面）
**意味**：難以～（書面語）  
**结构**：Vます形 + がたい  
**説明**：表示在客觀上難以做到。

1. 彼の気持ちは**理解しがたい**。
   → 難以理解他的心情。
2. この現実は**受け入れがたい**。
   → 這個現實難以接受。
3. 得**がたい**機會を逃してしまった。
   → 錯過了難得的機會。

## B1-201 ～っこない（不可能～・口語）
**意味**：不可能～（口語）  
**结构**：Vます形 + っこない  
**説明**：「わけがない」的口語表達。

1. 彼に**勝てっこない**。
   → 不可能贏他。
2. こんな大金、一人で**払えっこない**。
   → 這麼大一筆錢，一個人不可能付得起。
3. 一晩で覚え**られっこない**よ。
   → 一個晚上不可能背下來。

## B1-202 ～わけにはいかない（不能～）
**意味**：不能～（社會道義上）  
**结构**：V辞書形/ない形 + わけにはいかない  
**説明**：表示出於責任或社會規範而不能做某事。

1. 大事な會議だから、休む**わけにはいかない**。
   → 因為是重要會議，不能不參加。
2. 彼の頼みを斷る**わけにはいかない**。
   → 不能拒絕他的請求。
3. 知っていて知らないふりをする**わけにはいかない**。
   → 不能明明知道卻裝不知道。
