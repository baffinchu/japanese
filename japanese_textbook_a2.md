> **該当目安：JLPT N4相當 / 學習時數約300小時**
>
> A2是「進階基礎階段的語言使用者」。能理解與自身密切相關的句子和常用表達（如個人資訊、家庭、購物、工作、周圍環境等）。能在簡單、例行性的場合中進行溝通，交換熟悉的訊息。能用簡單的詞彙描述自身的背景、周遭環境及立即需求。

# 第1章 普通形（Plain Forms）

## A2-001 辞書形
**意味**：字典形／原形（非過去肯定）  
**结构**：動詞辞書形（基本形）  
**説明**：動詞的原始形態，用於普通體對話、句中等。

1. 毎日**行く**。
   → 每天去。
2. 映画を**見る**。
   → 看電影。
3. コーヒーを**飲む**。
   → 喝咖啡。

## A2-002 た形
**意味**：～了（普通體過去肯定）  
**结构**：動詞た形  
**説明**：普通體的過去肯定形，用於非正式場合。

1. 昨日、図書館へ**行った**。
   → 昨天去了圖書館。
2. 朝ごはんを**食べた**。
   → 吃了早餐。
3. 宿題を**やった**。
   → 做了作業。

## A2-003 ない形
**意味**：不～（普通體否定）  
**结构**：動詞ない形  
**説明**：普通體的否定形式。

1. 今日は学校へ**行かない**。
   → 今天不去學校。
2. お酒を**飲まない**。
   → 不喝酒。
3. 肉は**食べない**。
   → 不吃肉。

## A2-004 なかった
**意味**：沒～（普通體過去否定）  
**结构**：動詞なかった形  
**説明**：普通體的過去否定形。

1. 昨日は勉強し**なかった**。
   → 昨天沒學習。
2. 友達に会わ**なかった**。
   → 沒見到朋友。
3. 何も**食べなかった**。
   → 什麼都沒吃。

## A2-005 普通形+と思う
**意味**：我覺得～／我認爲～  
**结构**：普通形 + と思う  
**説明**：表示說話人的想法或推測。

1. 明日は雨が**降ると思う**。
   → 我覺得明天會下雨。
2. 彼は来**ないと思う**。
   → 我認爲他不來。
3. この料理は**おいしいと思う**。
   → 我覺得這道菜好吃。

## A2-006 普通形+と言う
**意味**：說～（引用）  
**结构**：普通形 + と言う  
**説明**：引用別人說的話或內容。

1. 彼は「**行く**」**と言った**。
   → 他說「要去」。
2. 山田さんは明日は**暇だと言った**。
   → 山田先生說明天有空。
3. 先生は試験は**難しくないと言った**。
   → 老師說考試不難。

## A2-007 普通形+だろう/でしょう
**意味**：～吧（普通體推量）  
**结构**：普通形 + だろう（口語）/ でしょう（禮貌）  
**説明**：表示推測。だろう是普通體，でしょう是禮貌體。

1. 明日は**雨だろう**。
   → 明天會下雨吧。
2. 彼はもう**来ないだろう**。
   → 他不會來了吧。
3. これで**大丈夫だろう**。
   → 這樣應該沒問題吧。

## A2-008 ～のだ/～んだ
**意味**：～的／就是～（説明・強調）  
**结构**：普通形 + のだ／んだ  
**説明**：用於説明情況、理由，或強調。口語用「んだ」。

1. 昨日は休みました。熱が**あったんです**。
   → 昨天休息了。是因爲發燒了。
2. **どうしたんですか**。——頭が痛い**んです**。
   → 怎麼了？——頭痛。
3. 日本語を勉強しています。日本で**働きたいんです**。
   → 我在學日語。是想在日本工作。

# 第2章 TE形進階用法（TE-form Advanced Uses）

## A2-009 ～ている
**意味**：正在～／～著（進行・狀態）  
**结构**：Vて形 + いる  
**説明**：表示動作進行中或狀態持續。普通體。

1. 今、電話して**いる**。
   → 正在打電話。
2. 彼はもう寝**ている**。
   → 他已經在睡了。
3. 私は東京に住ん**でいる**。
   → 我住在東京。

## A2-010 ～てある
**意味**：～著（結果狀態）  
**结构**：Vて形 + ある  
**説明**：表示人爲動作後的結果狀態。

1. 窓が開けて**ある**。
   → 窗戶開著。
2. 壁にポスターが貼って**ある**。
   → 牆上貼著海報。
3. もう予約して**ある**。
   → 已經預約好了。

## A2-011 ～ておく
**意味**：預先～（準備）  
**结构**：Vて形 + おく  
**説明**：爲將來做準備而預先做某事。口語常縮約爲「～とく」。

1. 明日のためにお弁当を買って**おく**。
   → 爲了明天先買好便當。
2. 旅行の前に切符を予約して**おく**。
   → 旅行前先預約票。
3. 部屋を掃除して**おいてください**。
   → 請先打掃房間。

## A2-012 ～てしまう
**意味**：～完了／不小心～了  
**结构**：Vて形 + しまう  
**説明**：表示動作完全結束（完了），或發生不如意的事（遺憾）。口語常縮約爲「～ちゃう／～じゃう」。

1. 宿題をやって**しまった**。
   → 做完作業了。
2. 財布を忘れて**しまった**。
   → 不小心忘了錢包。
3. ケーキを全部食べて**しまった**。
   → 把蛋糕全吃完了。

## A2-013 ～てみる
**意味**：試試看～  
**结构**：Vて形 + みる  
**説明**：表示嘗試做某事。

1. この料理を食べて**みる**。
   → 試試看這道菜。
2. 着物を着て**みた**。
   → 試穿了和服。
3. 日本語で話して**みてください**。
   → 請試著用日語說說看。

## A2-014 ～てくる
**意味**：（做）～再來／～起來（變化開始）  
**结构**：Vて形 + くる  
**説明**：表示去做某事後回來，或變化從過去持續到現在。

1. ちょっとコーヒーを買って**くる**。
   → 去買杯咖啡就回來。
2. 最近、日本語が上手になって**きた**。
   → 最近日語變得好了起來。
3. ずっと勉強して**きた**。
   → 一直學到了現在。

## A2-015 ～ていく
**意味**：（做）～再去／繼續～下去  
**结构**：Vて形 + いく  
**説明**：表示去做某事後離開，或變化將持續下去。

1. 荷物を持って**いく**。
   → 把行李帶去。
2. これからも勉強して**いく**。
   → 今後也會繼續學下去。
3. 人口は減って**いく**だろう。
   → 人口會繼續減少吧。

## A2-016 ～てあげる
**意味**：爲（別人）做～  
**结构**：Vて形 + あげる  
**説明**：主語（我方）爲對方做某事。

1. 友達に本を貸して**あげた**。
   → 借書給了朋友。
2. 妹に宿題を教えて**あげる**。
   → 教妹妹做作業。
3. お年寄りに席を譲って**あげた**。
   → 讓座給了老人。

## A2-017 ～てくれる
**意味**：（別人）爲我做～  
**结构**：Vて形 + くれる  
**説明**：對方爲我方做某事，以我方爲視角。

1. 友達が本を貸して**くれた**。
   → 朋友借書給了我。
2. 母がセーターを編んで**くれた**。
   → 媽媽織了毛衣給我。
3. 山田さんが駅まで送って**くれた**。
   → 山田先生送我到車站。

## A2-018 ～てもらう
**意味**：請（別人）做～／得到～  
**结构**：Vて形 + もらう  
**説明**：我方請對方做某事，或從對方那裏得到某動作的好處。

1. 友達に本を貸して**もらった**。
   → 請朋友借了書給我。
2. 先生に日本語を教えて**もらう**。
   → 請老師教我日語。
3. 医者に診て**もらった**。
   → 請醫生幫我看了。

## A2-019 ～ていただけませんか
**意味**：能請您～嗎？（鄭重請求）  
**结构**：Vて形 + いただけませんか  
**説明**：「てもらう」的謙讓鄭重形式，用於非常禮貌地請求。

1. ちょっと手伝って**いただけませんか**。
   → 能請您幫我一下嗎？
2. 住所を教えて**いただけませんか**。
   → 能請您告訴我地址嗎？
3. もう一度説明して**いただけませんか**。
   → 能請您再説明一次嗎？

## A2-020 ～てほしい
**意味**：希望（別人）做～  
**结构**：Vて形 + ほしい  
**説明**：表示希望別人做某事。

1. 静かにして**ほしい**。
   → 希望安靜一點。
2. もう少し早く来て**ほしい**。
   → 希望再早一點來。
3. このことを彼に伝えて**ほしい**。
   → 希望把這件事告訴他。

## A2-021 ～てもらいたい
**意味**：想請（別人）做～  
**结构**：Vて形 + もらいたい  
**説明**：「てもらう」的「たい」形，表示想要請對方做某事。

1. 先生に作文を直して**もらいたい**。
   → 想請老師修改作文。
2. 友達に手伝って**もらいたい**。
   → 想請朋友幫忙。
3. 彼に本当のことを話して**もらいたい**。
   → 想請他說出真相。

# 第3章 授受表現（Giving and Receiving）

## A2-022 ～てあげる（再掲・動作授受）
**意味**：爲（別人）做～  
**结构**：Vて形 + あげる  
**説明**：表示說話人或我方爲別人做某事。

1. 彼女に花を買って**あげた**。
   → 買了花給她。
2. 弟の世話をして**あげる**。
   → 照顧弟弟。
3. 道を教えて**あげましょうか**。
   → 我來告訴你路吧？

## A2-023 ～てくれる（再掲・動作授受内向）
**意味**：（別人）爲我做～  
**结构**：Vて形 + くれる  
**説明**：別人對我方施予恩惠或動作。

1. 彼が車で送って**くれた**。
   → 他開車送了我。
2. 同僚が仕事を手伝って**くれた**。
   → 同事幫了我工作。
3. 誰も助けて**くれなかった**。
   → 誰都沒有幫我。

## A2-024 ～てもらう（再掲・受惠）
**意味**：請（別人）做～  
**结构**：Vて形 + もらう  
**説明**：從接受者角度表示得到對方做某事的好處。

1. 友達に駅まで送って**もらった**。
   → 請朋友送我到車站。
2. 父に自転車を直して**もらう**。
   → 請爸爸幫我修腳踏車。
3. 先生に推薦状を書いて**もらった**。
   → 請老師寫了推薦信。

## A2-025 ～させてください
**意味**：請讓我做～  
**结构**：V使役形 + てください  
**説明**：請求允許做某事，鄭重表達。

1. 自己紹介**させてください**。
   → 請讓我自我介紹。
2. 今日は早く帰ら**せてください**。
   → 今天請讓我早點回去。
3. 一言お話**させてください**。
   → 請讓我說一句話。

## A2-026 やる
**意味**：給～（給下位者・動物・植物）  
**结构**：人／動物 + に + N + を + やる  
**説明**：用於上對下的給予，或給動物、植物。

1. 犬に餌を**やる**。
   → 給狗餵食。
2. 花に水を**やる**。
   → 給花澆水。
3. 弟にお小遣いを**やった**。
   → 給了弟弟零用錢。

# 第4章 可能形（Potential Form）

## A2-027 ～ことができる
**意味**：能夠做～  
**结构**：V辞書形 + ことができる  
**説明**：表示能力或可能性。

1. 日本語で手紙を**書くことができる**。
   → 能夠用日語寫信。
2. ここで**泳ぐことができる**。
   → 能在這裏游泳。
3. 一人で**行くことができない**。
   → 不能一個人去。

## A2-028 ～れる/られる（可能動詞）
**意味**：能～／會～  
**结构**：動詞可能形（五段→え段+る、一段→られる、する→できる、くる→こられる）  
**説明**：直接用可能形動詞表達能力。

1. 日本語が**話せる**。
   → 能說日語。
2. この漢字が**読める**。
   → 會讀這個漢字。
3. 早く**起きられない**。
   → 不能早起。

## A2-029 ～しか～ない
**意味**：只～（限定）  
**结构**：N + しか + 否定形  
**説明**：與否定搭配表達「只有～」。強調數量少或限定。

1. お金が**千円しかない**。
   → 只有一千日圓。
2. でき**ないことはない**けど、時間がかかる。
   → 不是不能做，但是花時間。——（非此句型）
3. 朝ごはんはパン**しか食べない**。
   → 早餐只吃麵包。
4. 日本語が**少ししか話せない**。
   → 只會說一點點日語。

## A2-030 ～しか＋可能形＋ない
**意味**：只能～（限定能力）  
**结构**：N + しか + 動詞可能形 + ない  
**説明**：表示能力或可能性有限。

1. ひらがな**しか書けない**。
   → 只會寫平假名。
2. 自転車**しか乗れない**。
   → 只會騎腳踏車。
3. 簡単な料理**しか作れない**。
   → 只會做簡單的菜。

# 第5章 條件形（Conditionals）

## A2-031 ～と
**意味**：一～就～（自然條件）  
**结构**：V辞書形／ない形 + と  
**説明**：表示必然結果或自然現象。後句不可接意志、命令、請求。

1. 春に**なると**、花が咲く。
   → 一到春天，花就開了。
2. このボタンを押す**と**、ドアが開く。
   → 一按這個按鈕，門就開了。
3. 左に曲がる**と**、駅があります。
   → 一左轉，就有車站。

## A2-032 ～ば
**意味**：如果～的話（條件）  
**结构**：動詞仮定形（ば形）  
**説明**：表示假設條件。用於一般條件句。

1. 安けれ**ば**買う。
   → 如果便宜就買。
2. 雨が降れ**ば**、試合は中止だ。
   → 如果下雨，比賽就停止。
3. 勉強すれ**ば**、成績が上がる。
   → 如果學習，成績就會提高。

## A2-033 ～たら
**意味**：如果～了的話（條件）  
**结构**：Vた形 + ら  
**説明**：表示假設條件，強調先完成前項再發生後項。最通用的條件形。

1. 東京に**行ったら**、連絡してください。
   → 去了東京的話，請聯繫我。
2. 時間が**あったら**、手伝ってください。
   → 如果有時間，請幫忙。
3. 安か**ったら**、買います。
   → 如果便宜的話就買。

## A2-034 ～なら
**意味**：如果是～的話（條件）  
**结构**：普通形 + なら  
**説明**：用於接對方的話題或假設，後句常爲建議或判斷。

1. 旅行**なら**、京都がおすすめです。
   → 如果是旅行，我推薦京都。
2. 忙しい**なら**、行かなくてもいいよ。
   → 如果忙的話，不去也可以喔。
3. 日本料理**なら**、寿司が一番好きだ。
   → 如果是日本料理，我最喜歡壽司。

## A2-035 ～ても
**意味**：即使～也～  
**结构**：Vて形 + も  
**説明**：表示讓步條件。

1. 雨が**降っても**行く。
   → 即使下雨也去。
2. 高く**ても**買う。
   → 即使貴也買。
3. 遅く**なっても**待っています。
   → 即使變晚了也會等。

## A2-036 ～なくても
**意味**：即使不～也～  
**结构**：Vない形（ない→なくて）+ も  
**説明**：表示即使不做某事也沒關係或仍成立。

1. 明日は**来なくてもいい**。
   → 明天不來也可以。
2. 全部**食べなくても**大丈夫だ。
   → 即使不全部吃完也沒關係。
3. 謝ら**なくても**許してくれた。
   → 即使沒道歉也原諒了我。

# 第6章 意向形（Volitional Form）

## A2-037 ～う/よう
**意味**：～吧（意志・勸誘）  
**结构**：動詞意向形（五段→う段→お段+う、一段→よう、する→しよう、くる→こよう）  
**説明**：表示自己的意志或勸誘對方一起做。

1. そろそろ**行こう**。
   → 差不多走吧。
2. 一緒に**勉強しよう**。
   → 一起學習吧。
3. 今日は早く**帰ろう**。
   → 今天早點回去吧。

## A2-038 ～う/ようと思う
**意味**：我想～（意志）  
**结构**：動詞意向形 + と思う  
**説明**：表達說話人的意志或打算。

1. 日本へ**行こうと思う**。
   → 我想去日本。
2. 新しい車を**買おうと思う**。
   → 我想買新車。
3. 大学に**進もうと思っている**。
   → 我正打算上大學。

## A2-039 ～う/ようとする
**意味**：正要～／試圖～  
**结构**：動詞意向形 + とする  
**説明**：表示正要開始做某事，或努力嘗試做某事。

1. 出かけ**ようとした**とき、電話がなった。
   → 正要出門的時候，電話響了。
2. ドアを開け**ようとした**が、開かなかった。
   → 試圖開門，但打不開。
3. 彼は何かを言**おうとしている**。
   → 他正想說些什麼。

## A2-040 ～う/ようかな
**意味**：我在想要不要～（猶豫）  
**结构**：動詞意向形 + かな

# 第7章 時間表現（Time Expressions）

## A2-041 ～前に
**意味**：在～之前  
**结构**：V辞書形 + 前に  
**説明**：表示在某一動作發生之前做另一動作。

1. 食べる**前に**手を洗う。
   → 吃飯前洗手。
2. 寝る**前に**歯を磨く。
   → 睡覺前刷牙。
3. 出かける**前に**天気をチェックする。
   → 出門前確認天氣。

## A2-042 ～た後で
**意味**：在～之後  
**结构**：Vた形 + 後で  
**説明**：表示在某一動作完成之後做另一動作。

1. 仕事が**終わった後で**飲みに行く。
   → 工作結束之後去喝酒。
2. 映画を**見た後で**感想を話した。
   → 看完電影後說了感想。
3. お風呂に**入った後で**寝る。
   → 洗澡之後睡覺。

## A2-043 ～てから
**意味**：～之後（順序連接）  
**结构**：Vて形 + から  
**説明**：強調兩個動作的先後順序。

1. 手を**洗ってから**食事する。
   → 洗手之後吃飯。
2. 大学を**卒業してから**就職した。
   → 大學畢業之後就職了。
3. 話し合って**から**決めましょう。
   → 商量之後再決定吧。

## A2-044 ～間/間に
**意味**：在～期間／在～期間内  
**结构**：V辞書形 + 間（あいだ）／間に  
**説明**：「間」表示整個期間持續；「間に」表示在某個時間點之內。

1. 授業の**間**、話してはいけません。
   → 上課期間不可以講話。
2. 夏休みの**間に**旅行する。
   → 在暑假期間去旅行。
3. 日本にいる**間に**富士山に登りたい。
   → 在日本期間想爬富士山。

## A2-045 ～時
**意味**：～的時候  
**结构**：V辞書形 + 時（とき）  
**説明**：表示動作發生的時間點。注意：前項動作未發生時用辭書形。

1. 駅に**着く時**、電話してください。
   → 到車站的時候請打電話。
2. 外国へ**行く時**、パスポートが必要だ。
   → 去外國的時候需要護照。
3. 買い物**に行く時**、いつもこの店を利用する。
   → 去買東西的時候總是用這家店。

## A2-046 ～た時
**意味**：～了的時候  
**结构**：Vた形 + 時（とき）  
**説明**：表示前項動作完成或狀態實現後的那個時間點。

1. 駅に**着いた時**、電車はもう行ってしまった。
   → 到了車站的時候，電車已經開走了。
2. 大人に**なった時**、この気持ちがわかると思う。
   → 長大了的時候，會明白這種心情吧。
3. 日本に**来た時**、日本語が全然話せなかった。
   → 來日本的時候，完全不會說日語。

## A2-047 ～ところ/たところ
**意味**：正要～／剛剛～  
**结构**：V辞書形 + ところ（正要）/ Vた形 + ところ（剛剛）  
**説明**：「辞書形+ところ」表示正要開始；「た形+ところ」表示動作剛剛結束。

1. 今、出かける**ところです**。
   → 現在正要出門。
2. ごはんを食べた**ところです**。
   → 剛剛吃完飯。
3. ちょうど電話した**ところです**。
   → 剛剛打了電話。

## A2-048 ～たばかり
**意味**：剛剛～（強調時間短）  
**结构**：Vた形 + ばかり  
**説明**：表示動作完成後經過的時間很短，比「たところ」更具主觀性。

1. 日本に来た**ばかり**です。
   → 剛來日本不久。
2. この本は昨日読んだ**ばかり**だ。
   → 這本書昨天剛讀完。
3. 卒業した**ばかり**で、まだ仕事を探している。
   → 剛畢業，還在找工作。

## A2-049 ～ながら
**意味**：一邊～一邊～  
**结构**：Vます（去ます）+ ながら  
**説明**：表示同一個主語同時進行兩個動作。

1. 音楽を**聞きながら**勉強する。
   → 一邊聽音樂一邊學習。
2. テレビを**見ながら**ごはんを食べる。
   → 一邊看電視一邊吃飯。
3. 歩き**ながら**話しましょう。
   → 一邊走一邊說吧。

## A2-050 ～ついでに
**意味**：順便～  
**结构**：V辞書形／Vた形 + ついでに  
**説明**：利用做某事的機會順便做另一件事。

1. 銀行に**行くついでに**郵便局にも寄る。
   → 去銀行的時候順便去郵局。
2. 買い物**のついでに**図書館に寄った。
   → 買東西的時候順便去了圖書館。
3. 大阪に**行ったついでに**友達に会った。
   → 去大阪的時候順便見了朋友。

# 第8章 目的・理由（Purpose and Reason）

## A2-051 ～ため（に）
**意味**：爲了～／因爲～  
**结构**：V辞書形 + ために（目的）/ N + の + ために（目的）/ 普通形 + ために（原因）  
**説明**：表示目的或原因。

1. 日本語を勉強する**ために**日本へ行く。
   → 爲了學日語去日本。
2. 健康**のために**毎日運動する。
   → 爲了健康每天運動。
3. 事故**のために**電車が遅れた。
   → 因爲事故電車晚點了。

## A2-052 ～のに
**意味**：雖然～但是～（逆接）  
**结构**：普通形（な形容詞/N + な/である）+ のに  
**説明**：表示逆接，帶有「不合常理」的意外感或不滿。

1. 勉強した**のに**合格できなかった。
   → 雖然學習了卻沒能及格。
2. 安かった**のに**買わなかった。
   → 雖然便宜卻沒買。
3. 雨が降っている**のに**出かけた。
   → 雖然下著雨卻出門了。

## A2-053 ～ので
**意味**：因爲～（客觀理由）  
**结构**：普通形 + ので（な形容詞/N + なので）  
**説明**：比「から」更客觀、禮貌，常用於正式場合或解釋。

1. 風邪をひいた**ので**学校を休んだ。
   → 因爲感冒了，所以請假沒去學校。
2. 値段が安い**ので**買った。
   → 因爲價格便宜，所以買了。
3. 時間が**ないので**タクシーで行く。
   → 因爲沒時間，所以搭計程車去。

## A2-054 ～し
**意味**：又～又～（列舉理由）  
**结构**：普通形 + し  
**説明**：列舉並列的原因或理由。

1. 値段も安い**し**、味もいい。
   → 價格又便宜，味道又好。
2. 日本語もできる**し**、英語も話せる。
   → 又會日語，又會說英語。
3. 雨も降っている**し**、風も強いから、出かけない。
   → 又下著雨，風又大，所以不出門。

## A2-055 ～から（理由・主觀）
**意味**：因爲～（主觀理由）  
**结构**：普通形 + から  
**説明**：表示主觀理由，用於説明原因或辯解。

1. 今日は**寒いから**、コートを着る。
   → 因爲今天冷，所以穿大衣。
2. 時間が**ないから**、急ごう。
   → 因爲沒時間，快點吧。
3. 好き**だから**、続けられる。
   → 因爲喜歡，所以能繼續下去。

# 第19章 接續表現（Connective Expressions）

## A2-110 しかし
**意味**：但是（正式書面）  
**结构**：文 + しかし + 文  
**説明**：正式的逆接連接詞。

1. この町は便利だ。**しかし**家賃が高い。
   → 這個城鎮很方便。但是房租很貴。
2. 日本語は難しい。**しかし**おもしろい。
   → 日語很難。但是很有趣。
3. 彼は優しい。**しかし**決断力がない。
   → 他很溫柔。但是沒有決斷力。

## A2-111 そして
**意味**：然後／而且  
**结构**：文 + そして + 文  
**説明**：連接句子，表示添加或順接。

1. 昨日は買い物をした。**そして**映画を見た。
   → 昨天去買東西了。然後看了電影。
2. 彼は頭がいい。**そして**親切だ。
   → 他頭腦很好。而且很親切。
3. 朝ごはんを食べた。**そして**学校へ行った。
   → 吃了早餐。然後去了學校。

## A2-112 それから
**意味**：然後／還有  
**结构**：文 + それから + 文  
**説明**：添加新項目或表示時間順序。

1. スーパーで野菜を買った。**それから**肉も買った。
   → 在超市買了蔬菜。然後也買了肉。
2. 京都に行った。**それから**奈良にも行った。
   → 去了京都。然後也去了奈良。
3. 勉強した。**それから**少し寝た。
   → 學習了。然後睡了一會兒。

## A2-113 それで
**意味**：所以／因此  
**结构**：文 + それで + 文  
**説明**：表示前句是後句的原因或理由。

1. お金がない。**それで**買わなかった。
   → 沒錢。所以沒買。
2. 頭が痛い。**それで**早く帰る。
   → 頭痛。所以早點回去。
3. 駅が遠い。**それで**バスで行く。
   → 車站很遠。所以搭公車去。

## A2-114 それに
**意味**：而且／再加上  
**结构**：文 + それに + 文  
**説明**：表示累加，在已有的基礎上追加資訊。

1. このアパートは駅に近い。**それに**家賃も安い。
   → 這個公寓離車站近。而且房租也便宜。
2. 彼は頭がいい。**それに**スポーツもできる。
   → 他頭腦好。而且也會運動。
3. 今日は暑い。**それに**湿気も多い。
   → 今天很熱。而且濕度也高。

## A2-115 しかも
**意味**：而且／還（強調）  
**结构**：文 + しかも + 文  
**説明**：比「それに」語氣更強，添加更重要的資訊。

1. 彼は医者だ。**しかも**有名な病院の院長だ。
   → 他是醫生。而且還是知名醫院的院長。
2. この車は速い。**しかも**燃費がいい。
   → 這輛車很快。而且油耗表現好。
3. 無料だった。**しかも**とてもおいしかった。
   → 是免費的。而且還非常好吃。

## A2-116 または
**意味**：或者（選擇）  
**结构**：N + または + N  
**説明**：表示選擇，相當於「或」。

1. 電車**または**バスで行けます。
   → 可以搭電車或公車去。
2. ペン**または**鉛筆で書いてください。
   → 請用原子筆或鉛筆寫。
3. ご飯**または**パンを選べます。
   → 可以選擇飯或麵包。

## A2-117 そして（再掲・時間順序）
**意味**：然後  
**结构**：文 + そして + 文  
**説明**：表示時間先後順序。

1. うちへ帰った。**そして**ごはんを食べた。
   → 回家了。然後吃飯了。
2. 顔を洗う。**そして**歯を磨く。
   → 洗臉。然後刷牙。
3. 仕事が終わった。**そして**友達に会った。
   → 工作結束了。然後見了朋友。

## A2-118 だから
**意味**：所以（口語）  
**结构**：文 + だから + 文  
**説明**：表示結論或結果，口語常用。

1. 時間がない。**だから**急ごう。
   → 沒時間了。所以快點吧。
2. 雨が降っている。**だから**出かけない。
   → 在下雨。所以不出門。
3. 彼は親切だ。**だから**人気がある。
   → 他很親切。所以受歡迎。

## A2-119 ただし
**意味**：但／只是（條件限制）  
**结构**：文 + ただし + 条件  
**説明**：書面語，表示補充條件或例外。

1. 入場は無料です。**ただし**予約が必要です。
   → 入場免費。但需要預約。
2. いつでも利用できます。**ただし**夜10時までです。
   → 隨時可以使用。但只到晚上10點。
3. 返品可能です。**ただし**未使用品に限ります。
   → 可以退貨。但僅限未使用商品。

## A2-120 なお
**意味**：另外／再者  
**结构**：文 + なお + 文  
**説明**：書面語，補充附加資訊。

1. 会議は10時からです。**なお**資料をお持ちください。
   → 會議從10點開始。另外請攜帶資料。
2. 入場料は無料です。**なお**駐車場は有料です。
   → 入場費免費。另外停車場收費。
3. 申込期限は来週です。**なお**詳細は添付ファイルをご覧ください。
   → 申請截止日期是下週。另外詳情請參閱附件。

# 第20章 誘い・申し出（Asking and Offering）

## A2-121 ～ませんか
**意味**：要不要～？（邀請）  
**结构**：Vませんか  
**説明**：禮貌地邀請對方做某事。

1. 一緒に映画を**見ませんか**。
   → 要不要一起看電影？
2. コーヒーを**飲みませんか**。
   → 要不要喝杯咖啡？
3. 日曜日に**買い物に行きませんか**。
   → 星期天要不要去買東西？

## A2-122 ～ましょう
**意味**：～吧（意志・勸誘）  
**结构**：Vましょう  
**説明**：表示說話人的意志或勸誘對方。

1. そろそろ**行きましょう**。
   → 差不多走吧。
2. 一緒に**勉強しましょう**。
   → 一起學習吧。
3. 今日は早く**帰りましょう**。
   → 今天早點回家吧。

## A2-123 ～ましょうか
**意味**：要不要我～？／我來～吧  
**结构**：Vましょうか  
**説明**：提議爲對方做某事或提供幫助。

1. 窓を**開けましょうか**。
   → 我來開窗戶吧？
2. 荷物を**持ちましょうか**。
   → 我來拿行李吧？
3. お茶を**入れましょうか**。
   → 我來泡茶吧？

## A2-124 ～ていただけませんか
**意味**：能請您～嗎？（鄭重請求）  
**结构**：Vて形 + いただけませんか  
**説明**：「てもらう」的謙讓鄭重請求。

1. ちょっと手伝って**いただけませんか**。
   → 能請您幫個忙嗎？
2. 住所を教えて**いただけませんか**。
   → 能請您告訴我地址嗎？
3. もう一度説明して**いただけませんか**。
   → 能請您再説明一次嗎？

## A2-125 ～させてください
**意味**：請讓我做～  
**结构**：V使役形 + てください  
**説明**：請求對方允許自己做某事。

1. 自己紹介**させてください**。
   → 請讓我自我介紹。
2. 今日は早く帰ら**せてください**。
   → 今天請讓我早點回去。
3. 一言話**させてください**。
   → 請讓我說一句話。

## A2-126 ～がいいですか
**意味**：哪個～比較好？  
**结构**：疑問詞 + が + いいですか  
**説明**：詢問選擇或建議。

1. どの**辞書がいいですか**。
   → 哪本字典比較好？
2. 京都へはいつ**行くのがいいですか**。
   → 什麼時候去京都比較好？
3. どの**料理がおいしいですか**。
   → 哪道菜好吃？

# 第21章 経験（Experiences and Events）

## A2-127 ～たことがある
**意味**：做過～（經驗）  
**结构**：Vた形 + ことがある  
**説明**：表示曾經有過的經驗。

1. 日本へ**行ったことがある**。
   → 去過日本。
2. 富士山に**登ったことがある**。
   → 爬過富士山。
3. すしを**食べたことがない**。
   → 沒吃過壽司。

## A2-128 ～たことがない
**意味**：沒做過～（無經驗）  
**结构**：Vた形 + ことがない  
**説明**：表示從未有過的經驗。

1. 一度も飛行機に**乗ったことがない**。
   → 一次也沒坐過飛機。
2. 外国へ**行ったことがない**。
   → 沒去過外國。
3. そんな料理を**食べたことがない**。
   → 沒吃過那種菜。

## A2-129 ～たとき
**意味**：～了的時候  
**结构**：Vた形 + とき  
**説明**：表示在某動作完成之後的那個時間點。

1. 大人に**なったとき**、この気持ちがわかる。
   → 長大了的時候，會明白這種心情。
2. 日本に**着いたとき**、電話してください。
   → 到了日本的時候，請打電話。
3. 彼に**会ったとき**、びっくりした。
   → 見到他的時候，嚇了一跳。

## A2-130 ～る前
**意味**：在～之前  
**结构**：V辞書形 + 前（まえ）  
**説明**：表示在某一動作發生之前。

1. 寝る**前**に歯を磨く。
   → 睡覺前刷牙。
2. 出かける**前**に確認する。
   → 出門前確認。
3. 食べる**前**に「いただきます」と言う。
   → 吃飯前說「開動了」。

# 第22章 助詞進階（Various Basic Particles）

## A2-131 が vs は（對比）
**意味**：が（主語・新情報） vs は（主題・對比）  
**结构**：N + が／は  
**説明**：「が」標記新資訊或小主語；「は」標記主題或對比。

1. 昔々、あるところにおじいさん**が**いました。（新情報）
   → 很久很久以前，某個地方有一位老爺爺。
2. おじいさん**は**山へ行きました。（既知主題）
   → 老爺爺去了山上。
3. ビール**は**飲む**が**、日本酒**は**飲まない。（對比）
   → 啤酒喝，但是日本酒不喝。

## A2-132 に vs で（場所）
**意味**：に（存在・着點） vs で（動作場所）  
**结构**：場所 + に／で  
**説明**：「に」標記存在或移動着點；「で」標記動作發生的場所。

1. 公園**に**猫がいる。（存在）
   → 公園裏有貓。
2. 公園**で**遊ぶ。（動作場所）
   → 在公園玩。
3. 駅**に**着く。（着點）
   → 到達車站。

## A2-133 を（出發點）
**意味**：從～（出發點・離開）  
**结构**：場所 + を + 移動動詞  
**説明**：「を」標記移動動詞的出發點或離開的場所。

1. 毎朝8時に家**を**出る。
   → 每天早上8點出家門。
2. 大学**を**卒業する。
   → 從大學畢業。
3. 次の駅で電車**を**降りる。
   → 在下一站下電車。

## A2-134 に（所有主・立場）
**意味**：對～來説（立場・所有）  
**结构**：人 + に + は／も  
**説明**：表示從某人的角度來看或擁有。

1. 私**には**兄が二人いる。
   → 我有兩個哥哥。
2. 彼**には**できないと思う。
   → 我覺得他做不到。
3. この問題**には**答えられない。
   → 無法回答這個問題。

## A2-135 も（程度の高さ・強調）
**意味**：竟然～／～也（強調數量多）  
**结构**：數量 + も  
**説明**：強調數量之多或程度之高。

1. 3時間**も**待った。
   → 竟然等了3個小時。
2. 映画を5回**も**見た。
   → 竟然看了5次電影。
3. 1000円**も**する。
   → 竟然要1000日圓。

# 第23章 助數詞進階（Advanced Counters and Numbers）

## A2-136 ～回（かい）
**意味**：～次  
**结构**：數字 + 回（かい）  
**説明**：計算次數的助數詞。

1. 週に**2回**ジムに行く。
   → 一週去兩次健身房。
2. この映画は**3回**見た。
   → 這部電影看了三次。
3. もう**一度**やってみる。
   → 再試一次。

## A2-137 ～階（かい）
**意味**：～樓  
**结构**：數字 + 階（かい）  
**説明**：計算樓層的助數詞。注意特殊讀音。

1. 会社は**3階**にある。
   → 公司在3樓。
2. エレベーターで**1階**まで行く。
   → 搭電梯到1樓。
3. 会議室は**4階**です。
   → 會議室在4樓。

## A2-138 ～時間（じかん）
**意味**：～小時  
**结构**：數字 + 時間（じかん）  
**説明**：計算時間長度的助數詞。

1. 毎日**2時間**勉強する。
   → 每天學習2小時。
2. 東京まで**3時間**かかる。
   → 到東京需要3小時。
3. **1時間**待った。
   → 等了一個小時。

## A2-139 ～年（ねん）
**意味**：～年  
**结构**：數字 + 年（ねん）  
**説明**：計算年份或年數。

1. 日本に**3年**住んでいる。
   → 在日本住了3年。
2. **去年**日本へ行った。
   → 去年去了日本。
3. 彼とは**5年**の付き合いだ。
   → 和他交往了5年。

## A2-140 ～ヶ月（かげつ）
**意味**：～個月  
**结构**：數字 + ヶ月（かげつ）  
**説明**：計算月數的助數詞。

1. 日本に来て**3ヶ月**になる。
   → 來日本3個月了。
2. **2ヶ月**前に引っ越した。
   → 兩個月前搬家了。
3. この仕事は**6ヶ月**の契約だ。
   → 這份工作是6個月的合約。

## A2-141 ～週間（しゅうかん）
**意味**：～週  
**结构**：數字 + 週間（しゅうかん）  
**説明**：計算週數的助數詞。

1. あと**2週間**で夏休みだ。
   → 再過兩週就是暑假了。
2. **1週間**に一度掃除する。
   → 一週打掃一次。
3. 風邪で**1週間**休んだ。
   → 因感冒休息了一週。

## A2-142 ～番（ばん）
**意味**：～號  
**结构**：數字 + 番（ばん）  
**説明**：表示編號、號碼。

1. 電話番号は**090の1234の5678**です。
   → 電話號碼是090-1234-5678。
2. **1番**のバスに乗ってください。
   → 請搭1號公車。
3. 何**番**ですか。
   → 幾號？

## A2-143 ～歳/才（さい）
**意味**：～歲  
**结构**：數字 + 歳（さい）／才  
**説明**：表達年齡。

1. 私は**25歳**です。
   → 我25歲。
2. 息子は**3歳**だ。
   → 兒子3歲。
3. もう**20歳**になった。
   → 已經20歲了。

# 第24章 敬語入門（Basic Keigo）

## A2-144 いらっしゃいます
**意味**：在／去／來（いる・行く・来る的尊敬語）  
**结构**：いらっしゃいます  
**説明**：對長輩或客人的動作表示尊敬。

1. 先生は教室に**いらっしゃいます**。
   → 老師在教室。
2. 社長は明日大阪へ**いらっしゃいます**。
   → 社長明天去大阪。
3. お客様が**いらっしゃいました**。
   → 客人來了。

## A2-145 おっしゃいます
**意味**：説（言う的尊敬語）  
**结构**：おっしゃいます  
**説明**：對長輩的「說」表示尊敬。

1. 先生は何と**おっしゃいましたか**。
   → 老師說了什麼？
2. 社長はそう**おっしゃいました**。
   → 社長是那麼說的。
3. お名前は何と**おっしゃいますか**。
   → 您叫什麼名字？

## A2-146 ご覧になります
**意味**：看（見る的尊敬語）  
**结构**：ご覧になります  
**説明**：對長輩的「看」表示尊敬。

1. もう資料を**ご覧になりましたか**。
   → 您已經看過資料了嗎？
2. 先生はその映画を**ご覧になりました**。
   → 老師看了那部電影。
3. どうぞ**ご覧になってください**。
   → 請您觀看。

## A2-147 お/ご～になります（尊敬語形）
**意味**：（您）做～（尊敬語）  
**结构**：お + Vます（去ます）+ になります / ご + 漢語名詞 + になります  
**説明**：對長輩的動作表示尊敬的句式。

1. いつ**お帰りになりますか**。
   → 您什麼時候回去？
2. こちらで**お待ちになってください**。
   → 請您在這裡等。
3. パスポートを**ご確認になります**。
   → 您確認護照。

## A2-148 お/ご～します（謙讓語形）
**意味**：我爲您做～（謙讓語）  
**结构**：お + Vます（去ます）+ します / ご + 漢語名詞 + します  
**説明**：對自己的動作表示謙虛。

1. 荷物を**お持ちします**。
   → 我來幫您拿行李。
2. 後で**お電話します**。
   → 我之後打電話給您。
3. **ご案内します**。
   → 我來帶路。

# 第25章 副詞（Adverbs and Intensifiers）

## A2-149 たくさん
**意味**：很多（數量）  
**説明**：表示數量多。

1. **たくさん**食べた。
   → 吃了很多。
2. お金が**たくさん**ある。
   → 有很多錢。
3. 友達が**たくさん**できた。
   → 交了很多朋友。

## A2-150 少し/ちょっと
**意味**：一點／稍微  
**説明**：表示少量或輕微程度。

1. **少し**待ってください。
   → 請稍等。
2. 日本語が**少し**話せる。
   → 會說一點日語。
3. **ちょっと**高いですね。
   → 有點貴呢。

## A2-151 もうすぐ
**意味**：馬上／快要  
**説明**：表示即將發生。

1. **もうすぐ**春になる。
   → 馬上就要到春天了。
2. バスは**もうすぐ**来る。
   → 公車馬上就來了。
3. **もうすぐ**着きます。
   → 馬上就到了。

## A2-152 ほとんど
**意味**：幾乎（大部分／幾乎不）  
**説明**：肯定時表示「大部分」；否定時表示「幾乎不」。

1. 宿題は**ほとんど**終わった。
   → 作業幾乎做完了。
2. 彼の話は**ほとんど**聞こえなかった。
   → 他的話幾乎聽不見。
3. 私は**ほとんど**テレビを見ない。
   → 我幾乎不看電視。

## A2-153 全然（ぜんぜん）
**意味**：完全（不）（與否定搭配）  
**説明**：與否定形搭配，表示「完全不～」。口語中也可用於肯定。

1. **全然**わからない。
   → 完全不懂。
2. お酒は**全然**飲まない。
   → 完全不喝酒。
3. 最近、**全然**会っていない。
   → 最近完全沒見面。

## A2-154 かなり
**意味**：相當／頗  
**説明**：表示程度高。

1. 日本語は**かなり**上手になった。
   → 日語變得很好了。
2. この荷物は**かなり**重い。
   → 這個行李相當重。
3. 彼は**かなり**疲れているようだ。
   → 他似乎相當累了。

## A2-155 だんだん/少しずつ
**意味**：漸漸地／一點一點地  
**説明**：表示變化緩慢持續。

1. 日本語が**だんだん**わかるようになった。
   → 漸漸地懂日語了。
2. 少し**ずつ**お金を貯めている。
   → 一點一點地存錢。
3. 天気が**だんだん**暖かくなってきた。
   → 天氣漸漸變暖和了。
