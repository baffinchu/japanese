# 日本語教材 ― CEFR C2（超上級）

> **該当目安：JLPT N1以上 / 學習時數約2500小時+**
>
> C2是「超上級／精通級語言使用者」。能毫無困難地理解所見到聽到的一切資訊。能從不同的口頭或書面來源歸納重組資訊，並以連貫的方式重現理由和說明。能自然、流暢且精確地表達自己，在複雜情境中也能區分細微的語義差異。本教材涵蓋古典文語、高級修辭、敬語極致、以及近母語者水準的語感區別。

# 第1章 古典文語（文語・古典文法）

## C2-001 ～ぬ／ず（classical negative）
**意味**：不～（文語否定形）  
**结构**：動詞未然形 + ぬ（連體形）／ず（連用形）／ざる（連體形）  
**説明**：文語否定助動詞「ず」的活用體系。現代文では「～ない」に相當。

1. これを**知らぬ**者はいない。
   → 沒有人不知道這個。
2. 言わ**ず**もがなの事實だ。
   → 不必說也明白的事實。
3. 雨**降らぬ**うちに歸ろう。
   → 趁還沒下雨回去吧。

## C2-002 ～ざる（classical negative attributive）
**意味**：不～的（文語否定連體形）  
**结构**：動詞未然形 + ざる + 名詞  
**説明**：「ず」的連體形「ざる」修飾名詞，是「～ない」的文語對應。常用於定型書面表達。

1. **止むを得ざる**事情により、本日は休業いたします。
   → 因不得已的情況，本日暫停營業。
2. それは**許すべからざる**行為だ。
   → 那是不可原諒的行為。
3. **やむにやまれざる**思いで決斷した。
   → 以無法抑制的心情做出了決定。

## C2-003 ～ねば（classical conditional negative）
**意味**：如果不～的話（文語條件否定）  
**结构**：動詞未然形 + ねば  
**説明**：「～なければ」的文語形。現代でも「～ねばならない」の形でよく使われる。

1. 努力**せねば**成功はない。
   → 不努力就沒有成功。
2. これ**を逃さねば**ならないチャンスだ。
   → 這是必須抓住的機會。
3. 書か**ねば**ならぬ書類が山積みだ。
   → 非寫不可的檔案堆積如山。

## C2-004 ～き／けり（past tense - classical）
**意味**：～了（文語過去助動詞）  
**结构**：動詞連用形 + き（連體形：し）／けり（連體形：ける）  
**説明**：「き」是親身經歷的過去；「けり」是傳聞或詠嘆的過去。

1. かく**ありき**、かく**ありけり**。
   → 曾如是，曾如是。（如此這般）
2. 昔々、あるところに、おじいさんとおばあさんが**住みけり**。
   → 很久很久以前，有個地方住著一位老爺爺和老奶奶。
3. 君の言うこと、まこと**なりけり**。
   → 你所說的，果然如此啊。

## C2-005 ～けむ（past conjecture - classical）
**意味**：過去推量（大概～了吧／曾～吧）  
**结构**：動詞連用形 + けむ  
**説明**：表示對過去的推量或回想。相當於現代語「～ただろう」「～たのだろう」。

1. いかでかくは**ありけむ**。
   → 何以竟會如此呢？
2. 誰かこれを**知らざりけむ**。
   → 誰會不知道這個呢？（反語）
3. いつの間に**過ぎにけむ**。
   → 不知不覺間就過去了吧。

## C2-006 ～らむ（present conjecture - classical）
**意味**：現在推量（大概正在～吧）  
**结构**：動詞連體形或形容詞連體形 + らむ  
**説明**：表示對現在的推量或對眼前狀況的詠嘆。相當於現代語「～ているだろう」「～のだろう」。

1. 今ごろは、彼も故郷を**思ひ出でつつあるらむ**。
   → 他現在大概也在懷念故鄉吧。
2. 行く先に何か**ありけむ**。
   → 前方大概有什麼吧。
3. いかに**思ふらむ**、我が君の心。
   → 我的心上人，此刻在想著什麼呢？

## C2-007 ～まし（hypothetical - classical）
**意味**：如果～的話就好了（文語反實假想）  
**结构**：動詞未然形 + まし  
**説明**：表示與事實相反的願望或假想。現代語「～だったらなあ」「～だろうに」。

1. もし翼**ありせまし**。
   → 如果有翅膀的話就好了。
2. 君に**逢はざりせまし**。
   → 如果沒有遇見你就好了。（或：若是不曾相遇）
3. それ**知らまし**を。
   → 如果知道那個就好了。（嘆息）

## C2-008 ～まほし（desire - classical）
**意味**：想要～（文語願望）  
**结构**：動詞未然形 + まほし  
**説明**：上代・中古的願望助動詞。現代語「～たい」に近いが、主に第一人稱に限る。

1. 我れ**見まほし**、その姿。
   → 我想看到那身影。
2. 君が面影を**忘れまほし**からず。
   → 並非想要忘記你的面容。
3. 世の中を**逃れまほし**と思えども。
   → 雖然想逃離這世間。

## C2-009 ～べし（obligation / conjecture - classical）
**意味**：應該～／會～（文語推量・當然）  
**结构**：動詞終止形 + べし（連體形：べき。已然形：べく）  
**説明**：表示當然、推量、命令、可能等多種意義。現代の「べきだ」の由來。

1. すべては順序を以て**進むべし**。
   → 一切應按順序進行。
2. 道は自然に**開かるべし**。
   → 道路自然會開通。
3. 格別の注意を**払うべし**。
   → 應格外注意。

## C2-010 ～べからず（must not - written）
**意味**：不可～（書き言葉の禁止）  
**结构**：動詞終止形 + べからず  
**説明**：「べし」の禁止形。掲示や規則などの文語的な禁止表現。

1. 芝生に**踏み入るべからず**。
   → 禁止踏入草坪。
2. 無用の者**入るべからず**。
   → 無關者禁止進入。
3. この書を他人に**見せるべからず**。
   → 禁止將本檔案給他人看。

## C2-011 ～べからざる（unacceptable - written）
**意味**：不可～的（書き言葉・連體形）  
**结构**：動詞終止形 + べからざる + 名詞  
**説明**：「べからず」の連體形。高度に文語的で、固い文章に使われる。

1. **許すべからざる**過ちを犯した。
   → 犯了不可原諒的錯誤。
2. それは**欠くべからざる**存在だ。
   → 那是不可或缺的存在。
3. **看過すべからざる**事態である。
   → 是不可忽視的事態。

## C2-012 ～なり／なる（judgment - classical）
**意味**：是～（文語斷定助動詞）  
**结构**：名詞 + なり（終止形）／なる（連體形）  
**説明**：文語の斷定助動詞「なり」（現代語の「だ」「である」に相當）。

1. 人生とはかくも無常**なり**。
   → 人生竟是如此無常。
2. 學問の道は近道**なきなり**。
   → 學問之路是沒有捷徑的。
3. すべては虛空**なる**ものぞ。
   → 一切皆為虛空。

# 第2章 文語接續（Literary Connectives）

## C2-013 ～かくて（thus - literary）
**意味**：於是／如此（文語接續詞）  
**结构**：句首接続詞  
**説明**：現代語の「こうして」に相當。物語や論説の展開に使う文語的接続詞。

1. **かくて**、長き旅も終わりを告げた。
   → 於是，漫長的旅行也迎來了終結。
2. **かくて**、日本は鎖國の道を選んだのである。
   → 如此，日本選擇了鎖國的道路。
3. **かくて**、平和は再び訪れた。
   → 於是，和平再次降臨。

## C2-014 ～されど（however - literary）
**意味**：然而（文語逆接）  
**结构**：句首接続詞  
**説明**：「しかし」の文語形。格調高い文章で使われる。

1. **されど**、それは容易ならざる業である。
   → 然而，那並非易事。
2. 多くの者が反対した。**されど**、彼は信念を曲げなかった。
   → 很多人反對。然而，他沒有動搖信念。
3. **されど**、真実は常に一つだ。
   → 然而，真相永遠只有一個。

## C2-015 ～しかれども（but - literary）
**意味**：雖然～但是（文語逆接）  
**结构**：用言已然形 + ども／句首接続詞  
**説明**：「しかし」より古風で、特に舊制高校や文語的な文章で使われる。

1. 貧し**かれども**、心は富めり。
   → 雖貧窮，但心靈富足。
2. 若**かれども**、彼の見識は大人を凌いでいた。
   → 雖然年輕，他的見識卻超越了大人。
3. **しかれども**、吾らは退くこと能わず。
   → 雖然如此，我們不能退縮。

## C2-016 ～しかして（and then - literary）
**意味**：然後／於是（文語順接）  
**结构**：句首接続詞  
**説明**：「そうして」「そして」の文語形。文章の展開を示す。

1. **しかして**、議論は混沌の極みに達した。
   → 於是，討論到達了混亂的極致。
2. まず原理を述べ、**しかして**具體例を示す。
   → 首先陳述原理，然後展示具體例子。
3. **しかして**問題は未解決のまま残された。
   → 於是問題就這麼懸而未決。

## C2-017 ～すなわち（namely - formal）
**意味**：即／也就是（形式）  
**结构**：句首或句中接続詞  
**説明**：言い換え・言い直しの表現。非常に格式張った文章で使われる。

1. 彼こそ、**すなわち**我々が探していた人物だ。
   → 他就是我們在找的那個人。
2. これが**すなわち**真実である。
   → 這就是真相。
3. 目的は**すなわち**、社會貢獻にほかならない。
   → 目的正是為社會做貢獻。

## C2-018 ～そもそも（in the first place - rhetorical）
**意味**：說起來／最初（修辭性起始）  
**结构**：句首副詞  
**説明**：議論や説明の冒頭で「もともと」「元來」の意味で使う。論を立てる際の決まり文句。

1. **そもそも**、この問題の本質は何か。
   → 說到底，這個問題的本質是什麼？
2. **そもそも**、人間とは矛盾に満ちた存在である。
   → 說起來，人類就是充滿矛盾的存在。
3. **そもそも**、なぜこれを始めたのかを忘れてはならない。
   → 不能忘記當初為什麼要開始這件事。

## C2-019 ～ただし（however - proviso）
**意味**：但是／不過（附帶條件）  
**结构**：句首（條件・但書）  
**説明**：前文に條件や例外を追加する接続詞。法律文書や規約で頻出。

1. 參加費は無料です。**ただし**、事前登録が必要です。
   → 參加費免費。但需提前註冊。
2. 全て返金いたします。**ただし**、商品到着後7日以內に限ります。
   → 全額退款。但僅限商品送達後7天內。
3. 自由に使用できます。**ただし**、営利目的での利用は禁止します。
   → 可自由使用。但禁止用於營利目的。

## C2-020 ～ただし書き（proviso writing）
**意味**：附帶條件的文章／但書  
**结构**：名詞（法律・契約文脈）  
**説明**：契約書や法律で「ただし」以降に書かれる例外・條件を指す。名詞として獨立。

1. 契約書の**ただし書き**をよく確認すべきだ。
   → 應仔細確認合約的但書。
2. その**ただし書き**が、全ての責任を免除している。
   → 那個但書免除了所有責任。
3. 法律の**ただし書き**に該當するケースだ。
   → 屬於法律但書適用的情況。

# 第3章 修辭表現（Rhetorical Devices）

## C2-021 ～とは（surprise / indignation）
**意味**：竟然～（意外・驚き・憤怒）  
**结构**：文の普通形 + とは  
**説明**：ある事態への強い驚き、意外、憤怒を表す。

1. あの人がまさか來る**とは**。
   → 沒想到那個人竟然會來。
2. こんなに早く終わる**とは**思わなかった。
   → 沒想到這麼快就結束了。
3. 彼がそんなことを言う**とは**、信じられない。
   → 他竟然說出那種話，難以置信。

## C2-022 ～ことよ（emphatic assertion）
**意味**：～啊／～呀（強い斷定・詠嘆）  
**结构**：名詞 + ことよ  
**説明**：文語的な強い斷定や詠嘆。特に女性の会話や古風な表現で使われる。

1. まあ、美しい**ことよ**。
   → 哎呀，好美啊。
2. なんと無慘な**ことよ**。
   → 多麼殘酷啊。
3. 人生とは、かくも儚き**ことよ**。
   → 人生竟是如此虛幻啊。

## C2-023 ～ものを（but - regret / reproach）
**意味**：明明～卻～（遺憾・非難）  
**结构**：文の普通形 + ものを  
**説明**：事實と反する狀況への不満、遺憾、非難を表す。

1. 教えてくれればよかった**ものを**。
   → 明明告訴我就好了（卻沒告訴我）。
2. 早く言えば済んだ**ものを**。
   → 早點說就沒事了（卻不說）。
3. 誰も気づかなかった**ものを**、わざわざ指摘する必要はない。
   → 明明誰都沒注意到，沒必要特意指出來。

## C2-024 ～ものを（if only - counterfactual）
**意味**：如果～就好了（反實假想）  
**结构**：文の普通形 + ものを  
**説明**：實現不可能な願望を表す。上代・中古から続く願望の形。

1. もう一度會える**ものを**。
   → 如果能再見一次就好了。
2. あの時に戻れる**ものを**。
   → 如果能回到那時候就好了。
3. 全てを知った上で、選べた**ものを**。
   → 如果能在知道一切之後再選擇就好了。

## C2-025 ～に（rhetorical - 知るに）
**意味**：在～方面／對於～（限定範圍の文語的用法）  
**结构**：動詞終止形 + に  
**説明**：文語で用いられる限定や範囲を表す形式。後の否定や困難を伴うことが多い。

1. **知るに**知らぬという無念さ。
   → 雖想知道卻不得而知的那種不甘。
2. **言うに**言われぬ美しさだ。
   → 有難以言喻的美。
3. **思うに**餘る出來事だった。
   → 是超乎想像的事件。

## C2-026 ～に（excessive - 言うに言えない）
**意味**：即使～也～不了（困難・不可能）  
**结构**：動詞終止形 + に + 同一動詞否定形  
**説明**：「～しようとしてもできない」という意味の文語的固定表現。

1. **言うに言えない**苦しみを抱えている。
   → 懷抱著難以言說的痛苦。
2. **泣くに泣けない**狀況だ。
   → 是欲哭無淚的狀況。
3. **思うに思えない**複雑な心境だ。
   → 是無法如願思考的複雜心境。

## C2-027 ～に（parallel - あるにない）
**意味**：既～又不～（並列的對照）  
**结构**：動詞終止形 + に + 同一動詞否定形  
**説明**：「～ではあるが、そうではない」という矛盾や微妙な狀態を表す文語的表現。

1. そこには**あるにない**不思議な存在感があった。
   → 那裡有一種似有若無的神奇存在感。
2. 見え**るに見えない**彼方の燈火。
   → 遠處若隱若現的燈火。
3. 分か**るに分からない**關係性に戸惑った。
   → 對那種似懂非懂的關係感到困惑。

## C2-028 倒置法（inversion for emphasis）
**意味**：倒裝法（語順の逆轉による強調）  
**结构**：文中の要素を文末に移動（述語・目的語等）  
**説明**：語順を敢えて逆転させて、特定の要素を強調する修辭法。

1. 來たぞ、**待ちに待った春が**。
   → 來了，盼望已久的春天！
2. 何と美しいのだろう、**この世界は**。
   → 多麼美麗啊，這個世界！
3. 忘れはしない、**君との約束を**。
   → 我不會忘記，和你的約定。

## C2-029 反語（rhetorical question）
**意味**：反問（肯定形で否定を、否定形で肯定を表す）  
**结构**：疑問形で用いるが、答えを期待しない  
**説明**：形は疑問だが、實際には強い主張を表す修辭法。

1. そんなこと、**誰が信じるか**。
   → 那種事誰會信？（＝沒有人信）
2. 努力なくして、**成功があろうか**。
   → 不努力，豈有成功？（＝不會成功）
3. 愛とは、**それだけで十分ではないか**。
   → 所謂愛，難道僅此還不夠嗎？（＝足夠了）

## C2-030 體言止め（noun-ending for impact）
**意味**：名詞結尾（體言止め・名詞で文を終える技法）  
**结构**：文末を名詞（體言）で終える  
**説明**：餘韻や印象を殘すための修辭技法。詩歌やキャッチコピーで多用。

1. 秋深し、隣は何を**する人ぞ**。
   → 深秋，鄰居是何許人也。
2. 靜けさの**極み**。
   → 寂靜的極致。
3. これぞ、日本の**美**。
   → 此乃日本之美。

# 第4章 區別・ニュアンス（Nuanced Distinctions）

## C2-031 ～ものだ vs ～ことだ（distinction）
**意味**：～ものだ（本質・感慨） vs ～ことだ（具體事項・忠告）  
**结构**：ものだ＝普遍真理・本質・感慨；ことだ＝具體事項・評價・忠告  
**説明**：「ものだ」は本質的な性質や感慨、「ことだ」は具體的な事柄に対する評價や助言を表す。

1. 人間はいつか死ぬ**ものだ**。
   → 人終有一死。（普遍真理）
2. 時の経つのは早い**ものだ**。
   → 時間過得真快啊。（感慨）
3. 合格するためには、每日コツコツ勉強する**ことだ**。
   → 為了合格，要每天孜孜不倦地學習。（具體忠告）

## C2-032 ～わけだ vs ～はずだ（distinction）
**意味**：～わけだ（道理上當然） vs ～はずだ（預期・推測）  
**结构**：わけだ＝從事實推導出的結論；はずだ＝基於資訊的預期  
**説明**：「わけだ」は客觀的な理由や結果として當然であること、「はずだ」は主觀的な期待や確信を表す。

1. 彼は十年日本にいるから、日本語が上手な**わけだ**。
   → 他在日本住了十年，日語好是理所當然的。
2. 彼はもうすぐ來る**はずだ**。さっき電話したから。
   → 他應該馬上就來了。因為剛才打電話了。
3. 納得した**わけではない**が、受け入れるしかない。
   → 並不是理解了，只是不得不接受。

## C2-033 ～らしい vs ～っぽい vs ～気味（nuance）
**意味**：らしい（典型特徵） vs っぽい（傾向・性質） vs 気味（傾向・微兆）  
**结构**：らしい＝客觀典型；っぽい＝主觀印象；気味＝負面傾向  
**説明**：「らしい」は典型的特徵、「っぽい」は性質や傾向、「気味」は良くない傾向。

1. 彼は日本人**らしい**。
   → 他很有日本人的樣子。（典型特徵）
2. 彼女は飽き**っぽい**性格だ。
   → 她是容易厭倦的性格。（傾向）
3. 最近、疲れ**気味**だ。
   → 最近有點疲勞。（負面傾向）

## C2-034 ～ざるを得ない vs ～ないわけにはいかない（subtle difference）
**意味**：不得不～（客觀強制 vs 社會義務）  
**结构**：ざるを得ない＝客觀狀況迫使；ないわけにはいかない＝社會・道義上的義務  
**説明**：「ざるを得ない」は外的な狀況による強い強制。「ないわけにはいかない」は社會的期待や責任。

1. この狀況では、計畫を斷念せ**ざるを得ない**。
   → 在這種情況下，不得不放棄計畫。
2. 恩師の頼みなら、斷る**わけにはいかない**。
   → 如果是恩師的請求，不能拒絕。
3. 證據が揃っている以上、認め**ざるを得ない**。
   → 既然證據確鑿，不得不承認。

## C2-035 ～ものの vs ～ながらも vs ～にもかかわらず（register difference）
**意味**：雖然～但是（逆接の文体差）  
**结构**：ものの＝口語～文章；ながらも＝文語的；にもかかわらず＝正式書面  
**説明**：「ものの」は輕い逆接、「ながらも」はやや古風、「にもかかわらず」は公式な文書向け。

1. 行きたいと**思うものの**、時間がなくて行けない。
   → 雖然想去，但沒時間去。
2. 不可能と知り**ながらも**、挑戦し続けた。
   → 雖然知道不可能，仍繼續挑戰。
3. 再三の注意**にもかかわらず**、事故は發生した。
   → 儘管再三警告，事故還是發生了。

## C2-036 ～ことから vs ～ところから vs ～ゆえ（formality）
**意味**：因為～／由於～（理由表現の文体差）  
**结构**：ことから＝客觀根據；ところから＝推測的由來；ゆえ＝文語的理由  
**説明**：「ことから」は理由の説明、「ところから」は名稱の由來、「ゆえ」は古典的。

1. 貢獻が認められた**ことから**、昇進した。
   → 因為貢獻得到認可，晉升了。
2. この地域は雪が多い**ところから**「雪國」と呼ばれている。
   → 這個地區因為雪多而被稱為「雪國」。
3. 故**あって**、退職することにした。
   → 因故決定辭職。

## C2-037 ～ならでは vs ～にしては vs ～としても（comparison）
**意味**：只有～才有的 vs 作爲～來說 vs 即使～也  
**结构**：ならでは＝獨有的特徴；にしては＝超出預期的偏差；としても＝假定讓步  
**説明**：「ならでは」是唯一性、「にしては」是比較基準、「としても」是假定。

1. これはプロ**ならではの**技術だ。
   → 這是只有專業人士才有的技術。
2. 初心者**にしては**上出來だ。
   → 作為初學者來說表現不錯。
3. 假に合格**としても**、慢心は禁物だ。
   → 即使合格了，也不能驕傲。

## C2-038 ～かたがた vs ～がてら vs ～ついでに（nuance distinction）
**意味**：順便（兼ねて vs ながら vs ついで）  
**结构**：かたがた＝兼備兩個目的（文語）；がてら＝順便（口語～文章）；ついでに＝順便（口語）  
**説明**：三つとも「ある動作のついでに他の動作をする」表現だが、フォーマル度が異なる。

1. お禮**かたがた**、ご報告に伺いました。
   → 順便道謝，前來報告。（文語・正式）
2. 散歩**がてら**、買い物をしてきた。
   → 散步順便買了東西。
3. **ついでに**、この書類も出してきてください。
   → 順便也把這份文件交出去。

# 第5章 正式連語（Formal Fixed Expressions）

## C2-039 ～を以て（by means of - classical）
**意味**：以～／憑藉～（文語的手段表現）  
**结构**：名詞 + を以て  
**説明**：「～をもって」の漢字表記。手段・方法・理由の文語的表現。現在も公式文書で多用。

1. 書面**をもって**通知いたします。
   → 以書面通知。
2. 本日**をもって**営業を終了します。
   → 從本日起結束營業。
3. 實績**をもって**その能力を証明した。
   → 以實績證明了其能力。

## C2-040 ～を以てしても（even with）
**意味**：即使以～也（不充分）  
**结构**：名詞 + を以てしても  
**説明**：「～をもってしても」の形で、「それでもなお足りない」意味を持つ。

1. 萬全の準備**をもってしても**、失敗は起こりうる。
   → 即使做了萬全準備，失敗仍可能發生。
2. 最新の醫療**をもってしても**、治療は困難だ。
   → 即使以最先進的醫療，治療也很困難。
3. 全力**をもってしても**、彼には敵わなかった。
   → 即使全力以赴，也敵不過他。

## C2-041 ～をして（make / let - classical）
**意味**：使～（表示使役的文語格助詞）  
**结构**：名詞 + をして  
**説明**：文語の使役表現で、使役對象を「をして」で示す。後に「～しめる」を伴うことが多い。

1. 彼**をして**言わしめれば、それは誤解だという。
   → 若讓他說，那是誤解。
2. 時**をして**すべてを解決せしめよ。
   → 讓時間解決一切吧。
3. 自然**をして**吾らに真理を示さしめる。
   → 自然讓我們看到了真理。

## C2-042 ～をして～しめる（cause to - literary）
**意味**：使～做～（文語使役）  
**结构**：名詞 + をして + 動詞未然形 + しめる  
**説明**：「～に～させる」の文語形。高度に格式張った表現。

1. 彼**をして**その任に當たら**しめる**。
   → 讓他擔任那個職務。
2. 社會**をして**より良い方向に進ま**しめん**。
   → 讓社會朝着更好的方向前進。
3. 國**をして**繁榮に導か**しめよ**。
   → 引領國家走向繁榮。

## C2-043 ～しめる（causative - literary）
**意味**：使～（使役助動詞・文語）  
**结构**：動詞未然形 + しめる  
**説明**：文語の使役助動詞「しめる」は、現代の「せる・させる」に相當。公式文書で使用。

1. 以上の理由により、本件を卻下せ**しめる**。
   → 基於以上理由，駁回本案。
2. 社會に貢獻せ**しめる**人材を育成する。
   → 培育能貢獻社會的人才。
3. その決定は、多くの者を困惑せ**しめた**。
   → 那個決定使許多人感到困惑。

## C2-044 ～たてまつる（humbly do - classical）
**意味**：恭～／敬～（謙讓・文語）  
**结构**：動詞連用形 + たてまつる  
**説明**：「～申し上げる」の文語形。身分の高い人に対する動作を謹んで言う。

1. この書を陛下に奉**りたてまつる**。
   → 謹將此書呈獻給陛下。
2. 心より感謝申し**たてまつる**。
   → 謹致以衷心的感謝。
3. 貴君の無事を祈り**たてまつる**。
   → 謹祝閣下平安。

## C2-045 ～奉る（humble - classical）
**意味**：奉～（謹んでする／差し上げる）  
**结构**：動詞連用形 + まつる  
**説明**：「たてまつる」の省略形。神佛や高貴な人への動作を表す。

1. 神前に玉串を**奉った**。
   → 在神前供奉了楊桐枝。
2. この詩を師に**奉る**。
   → 將此詩獻給老師。
3. 祈りを**奉りて**、平和を願う。
   → 獻上祈禱，祈求和平。

## C2-046 ～仕る（humble - classical つかまつる）
**意味**：做～（謙讓・文語の最上級）  
**结构**：動詞連用形 + 仕る（つかまつる）  
**説明**：「する」の最上級の謙讓語。話者の動作を極めて低く言う。

1. この任、謹んでお受け**つかまつる**。
   → 謹接受此任務。
2. ご案内**つかまつります**。
   → 謹為您帶路。
3. ご注文の品を屆け**つかまつります**。
   → 謹送達您訂購的商品。

# 第6章 敬語高度（Advanced Keigo / Honorific）

## C2-047 二重敬語（double honorific）
**意味**：雙重敬語（二重尊敬）  
**结构**：お + 動詞ます形 + になられる／お召し上がりになる等  
**説明**：本來は過剩敬語とされるが、現代の最上級の敬意表現として社會で広く使われている。

1. 社長は**お召し上がりに**なられました。
   → 社長已經用過餐了。
2. 明日は**お越しに**なられますか。
   → 您明天會來嗎？
3. 書類は**お読みに**なられましたか。
   → 您看過文件了嗎？

## C2-048 尊敬語＋謙譲語混合（mixed honorific）
**意味**：尊敬語與謙讓語混合使用（敬語混用）  
**结构**：對方動作に尊敬語 + 自分の動作に謙譲語  
**説明**：同文中で尊敬語と謙讓語を使い分ける高度な敬語運用。

1. 先生が**おっしゃった**內容を、私が**申し上げます**。
   → 老師所說的內容，由我來陳述。
2. 社長が**ご覽**になった資料を、私が**お持ちいたしました**。
   → 社長看過的資料，我已經帶來了。
3. お客様が**お求め**の商品を、こちらに**ご用意**いたしました。
   → 客人您要的商品，我們已準備在此。

## C2-049 ～せよ（command - classical / written）
**意味**：～吧／要～（文語命令・書き言葉）  
**结构**：動詞未然形（サ變）＋せよ／動詞命令形  
**説明**：「する」の文語命令形「せよ」。法規、マニュアル、學術文章で使用。

1. 直ちに報告**せよ**。
   → 立即報告。
2. 準備を整え**よ**。
   → 做好準備。
3. よく聞き、よく判斷**せよ**。
   → 仔細聆聽，仔細判斷。

## C2-050 ～たまえ（polite command - formal）
**意味**：請～（稍正式的拜託/指示）  
**结构**：動詞連用形 + たまえ  
**説明**：「ください」より古風でやや強い指示。目上の人が目下に使う。男性語。

1. 席に**かけたまえ**。
   → 請坐。
2. 遠慮なく**食べたまえ**。
   → 請別客氣，吃吧。
3. 思うところを率直に**述べたまえ**。
   → 請坦率說出你的想法。

## C2-051 ～たまえ（gentle command）
**意味**：～吧（輕柔的指示或勸誘）  
**结构**：動詞連用形 + たまえ  
**説明**：C2-050より優しいニュアンス。親しい間柄で相手の負擔を軽くする指示。

1. まあ、座り**たまえ**。
   → 來，坐吧。
2. あまり焦らなくていい。ゆっくり**やりたまえ**。
   → 不用太着急，慢慢做吧。
3. その話は後に**しようではないか**。まずは食事**にしたまえ**。
   → 那件事晚點再說吧，先吃飯。

## C2-052 ～給う（archaic honorific - tamau）
**意味**：～給う（文語尊敬・やや古風）  
**结构**：動詞連用形 + 給う（たもう／たまう）  
**説明**：「～なさる」の文語形。現代では歴史小説やわざと古風に言う時に使う。

1. 王は兵を率いて出陣**たまうた**。
   → 國王率軍出征了。
2. 神よ、我らを救い**たまえ**。
   → 神啊，請拯救我們。
3. 君はよくぞ來**たもうた**。
   → 你來得正好。

# 第7章 詩的表現（Poetic / Aesthetic Expressions）

## C2-053 ～ぞ（emphasis - classical particle）
**意味**：～啊／～喲（強い主張・詠嘆）  
**结构**：文末の係助詞「ぞ」／文中の強調  
**説明**：文語の強意の係助詞。文末で強い主張や決意を表す。係り結びで連體形を導くこともある。

1. 今こそ**行くぞ**。
   → 現在就去！
2. 天の道は**恐ろしきものぞ**。
   → 天道是可怕的東西啊。
3. 我こそは**正義なりぞ**。
   → 我才是正義！

## C2-054 ～かも（perhaps - あるかも）
**意味**：也許～（文語的推量・舊形）  
**结构**：動詞・形容詞の已然形 + かも  
**説明**：「～かも知れない」の元形。古くは「ありかも」の形で使われた。

1. 願わく**ばありかも**、その機會。
   → 但願也許有那個機會。
2. 彼の言うことも、一理**あるかも**。
   → 他所說的也許也有道理。
3. この世に不思議**なることかも**。
   → 這世上也許有不可思議之事。

## C2-055 ～やら（uncertain - いつのやら）
**意味**：不知不覺～／～呢（不確定・疑問）  
**结构**：疑問詞 + やら  
**説明**：「～か」の不確定な文語形。疑問や不確かさを表す。

1. **いつやら**約束したっけ。
   → 是不是什麼時候約好了？
2. **どこへやら**消えてしまった。
   → 不知消失到哪裡去了。
3. **なにやら**不思議な音が聞こえる。
   → 聽到了某種不可思議的聲音。

## C2-056 ～だに（even - not even - classical）
**意味**：連～都（文語・强調）  
**结构**：名詞 + だに  
**説明**：「～すら」「～さえ」の文語形。否定と共に使うと「連～都不」。

1. **想像だに**しなかった結果だ。
   → 連做夢都沒想到的結果。
2. **聞くだに**恐ろしい話だ。
   → 連聽都覺得可怕的故事。
3. **一蹴だに**せず、彼は去っていった。
   → 他連一腳都沒踢就離開了。

## C2-057 ～すら（even - formal）
**意味**：連～都（形式・強調）  
**结构**：名詞 + すら  
**説明**：「さえ」の文語的・形式的表現。ある極端な例を挙げて強調する。

1. 自分の名前**すら**書けなかった。
   → 連自己的名字都不會寫。
2. 日常會話**すら**ままならない。
   → 連日常對話都難以應付。
3. そんなことは子供**すら**知っている。
   → 那種事連小孩都知道。

## C2-058 ～だにしない（doesn't even）
**意味**：連～都不做（強い否定）  
**结构**：動詞終止形 + だにしない  
**説明**：「～することさえしない」の文語形。強い否定を表す。

1. 彼は抗議**だにしなかった**。
   → 他連抗議都沒有。
2. それに気づき**だにしない**。
   → 連注意到它都沒有。
3. 一言の謝罪**だにしなかった**。
   → 連一句道歉都沒有。

## C2-059 ～さえ（even - emphasis）
**意味**：連～都（強調・舉例）  
**结构**：名詞 + さえ  
**説明**：極端な事例を取り上げて「當然他の～はなおさら」という含意を持つ。文語的でもあり口語的でもある。

1. 本人**さえ**知らない事實だ。
   → 是連本人都不知的事實。
2. 水**さえ**飲めない狀況だった。
   → 是連水都不能喝的狀況。
3. 子供**でさえ**分かる道理だ。
   → 是連小孩都懂的道理。

## C2-060 ～すらも（even - stronger）
**意味**：連～都（更強烈的強調）  
**结构**：名詞 + すらも  
**説明**：「すら」に「も」を加えて強調を強めた形式。文語的でやや古風。

1. 一つ**すらも**殘っていない。
   → 連一個都沒有剩下。
2. その名前**すらも**忘れられていた。
   → 連那個名字都被遺忘了。
3. 過ちを認める勇氣**すらも**なかった。
   → 連承認錯誤的勇氣都沒有。

# 第8章 稀な重要表現（Rare but Important Patterns）

## C2-061 ～を顧みず（regardless of）
**意味**：不顧～／不考慮～  
**结构**：名詞 + を顧みず  
**説明**：「～を気にしないで」「～を考慮に入れないで」の文語表現。自己犠牲や危険を伴う文脈で使う。

1. 彼は身の安全**を顧みず**、川に飛び込んだ。
   → 他不顧自身安全，跳進了河裡。
2. 危険**を顧みず**、救助活動を続けた。
   → 不顧危險，持續進行救助活動。
3. 批判**を顧みず**、改革を斷行した。
   → 不顧批評，斷然實施了改革。

## C2-062 ～を厭わず（without shying from）
**意味**：不嫌棄～／不迴避～  
**结构**：名詞 + を厭わず  
**説明**：「～を嫌がらずに」「～を避けずに」の文語表現。労苦や汚れを厭わないことを表す。

1. 彼女は汚れ**を厭わず**、ボランティアに參加した。
   → 她不嫌髒地參加了志願者活動。
2. 手間**を厭わず**、丁寧に作った料理だ。
   → 是不嫌麻煩精心製作的料理。
3. 批判**を厭わず**、己の信念を貫いた。
   → 不避批評，貫徹了自己的信念。

## C2-063 ～意図せず（unintentionally）
**意味**：無意中／非本意地  
**结构**：意図せず + 動詞／意図せざる + 名詞  
**説明**：「意図しないで」「無意識に」の文語表現。

1. **意図せず**して、彼の秘密を知ってしまった。
   → 無意中知道了他的秘密。
2. その発言は**意図せざる**誤解を生んだ。
   → 那發言產生了非本意的誤解。
3. **意図せず**も、同じ結果に至った。
   → 無意之中也達到了同樣的結果。

## C2-064 ～あろうことか（how could it be that）
**意味**：竟然會有這種事（出乎意料的非難）  
**结构**：あろうことか + 文  
**説明**：「信じられないことに」「意外にも」の意味で、否定的な出來事に使う。

1. **あろうことか**、彼は約束を破った。
   → 豈有此理，他竟然毀約了。
2. **あろうことか**、內部の者が情報を漏らしていた。
   → 怎麼會這樣，內部人員洩露了情報。
3. **あろうことか**、彼女は恩人を裏切ったのだ。
   → 怎麼會這樣，她背叛了恩人。

## C2-065 ～あに図らんや（who would have thought）
**意味**：豈能料到／誰能想到（文語反語）  
**结构**：あに + 動詞未然形 + んや  
**説明**：「まさか～とは思わなかった」という驚きを反語で表す古典表現。

1. **あに図らんや**、彼が優勝するとは。
   → 誰能料到，他竟然會奪冠。
2. 成功する**あに図らんや**。
   → 豈能預料到會成功？
3. **あに意想外ならんや**。
   → 豈不是出乎意料？

## C2-066 ～や（rhetorical question - classical）
**意味**：～嗎／～呢（文語反語）  
**结构**：文末 + や  
**説明**：文語の反語の係助詞。否定を強調する。係り結びで連體形を取ることもある。

1. かくまで苦しむこと**あらんや**。
   → 豈有如此痛苦之事？
2. 誰かこれを信じ**んや**。
   → 誰會相信這個？
3. 永遠の別れと**ならんや**。
   → 豈不是永別了嗎？

## C2-067 ～か（rhetorical - classical - あろうか）
**意味**：難道～嗎（反問）  
**结构**：文の普通形 + か（上昇調）  
**説明**：反語表現で、形は疑問だが實際は強い否定を表す。特に「あろうか」の形式で多用。

1. こんなこと、**許されようか**。
   → 這種事，豈能容許？
2. それで十分**であろうか**。
   → 那樣就足夠了嗎？（不，不夠）
3. 誰が彼を責め**られようか**。
   → 誰能責備他呢？（誰也不能）

## C2-068 ～んや（rhetorical - classical）
**意味**：難道～嗎（文語反語・文末）  
**结构**：動詞未然形 + んや  
**説明**：「～だろうか（いや～ではない）」の文語反語。强い否定を含意。

1. それ以上に幸福なこと**あらんや**。
   → 還有比這更幸福的事嗎？（不，沒有）
2. かくの如き悲劇**あらんや**。
   → 豈有此等悲劇！
3. 再び會うこと**あらんや**。
   → 豈能再相見？

# 第9章 諺・慣用句（Proverb and Idiom Grammar）

## C2-069 ～べからざる（proverb - 言うべからざる）
**意味**：不可～的（禁止・不可能の連體形）  
**结构**：動詞終止形 + べからざる + 名詞  
**説明**：成句やことわざで使われる文語的禁止の連體形。

1. **言うべからざる**こととは知りつつ、口に出た。
   → 明知不可說，還是說出口了。
2. これは**欠くべからざる**前提條件だ。
   → 這是不可或缺的前提條件。
3. 社會正義は**譲るべからざる**ものだ。
   → 社會正義是不可退讓的。

## C2-070 ～べくんば（if possible - literary）
**意味**：如果可以～的話（文語條件）  
**结构**：動詞終止形 + べくんば  
**説明**：「～できるならば」の文語表現。可能の條件を示す。

1. **知るべくんば**、事前に連絡いたします。
   → 如果能夠知道的話，我會事先聯繫。
2. **改めるべくんば**、今すぐにでも改めるべきだ。
   → 如果可以改的話，應該立即就改。
3. **助くるべくんば**、力を貸したまえ。
   → 如果能幫助，請伸出援手。

## C2-071 ～べくして（inevitable - 起こるべくして）
**意味**：必然～／註定～  
**结构**：動詞終止形 + べくして + 同一動詞  
**説明**：「～することが當然の成り行きとして起こる」という必然性を表す。

1. この事故は**起こるべくして**起こった。
   → 這場事故是必然發生的。
2. 彼の成功は**得るべくして**得たものだ。
   → 他的成功是理所應得的。
3. 二人の別れは**避けるべくして**避けられなかった。
   → 兩人的分離是命中註定無法避免的。

## C2-072 ～ん（negative volitional - 行かん）
**意味**：不～（推量・否定意志の文語形）  
**结构**：動詞未然形 + ん  
**説明**：「～ないだろう」「～まい」の文語形。否定推量や否定意志を表す。

1. いざ**行かん**、戰場へ。
   → 出發吧，向戰場！（此處是意志）
2. 二度と**來じ**（來んじ）と思えども。
   → 雖想不會再來第二次。
3. 成る**べからん**や。
   → 豈能成功？（反語）

## C2-073 ～んがため（in order to - literary）
**意味**：為了～（文語目的）  
**结构**：動詞未然形 + んがため  
**説明**：「～しようとして」「～するために」の文語表現。

1. 真実を明らかに**せんがため**、調査を続けた。
   → 為了查明真相，持續調查。
2. 後世に殘**さんがため**、記録を殘す。
   → 為了流傳後世，留下記錄。
3. 勝た**んがため**なら、どんな努力も惜しまない。
   → 如果是為了取勝，不惜任何努力。

## C2-074 ～んとす（try to / be about to）
**意味**：正想要～／即將～（文語意志）  
**结构**：動詞未然形 + んとす  
**説明**：「～うとする」の文語形。動作の直前や意志を表す。

1. 船は今まさに港を出**んとしている**。
   → 船即將離港。
2. 彼は何かを言わ**んとして**、口を開いた。
   → 他正想說什麼，張開了嘴。
3. 時代は大きく変わら**んとしている**。
   → 時代正將發生巨變。

# 第10章 超微妙區別（Extra-Fine Nuance Distinctions）

## C2-075 得る vs うる vs える（register）
**意味**：得る（える）＝口語；うる＝文語可能；えうる＝古風重複  
**结构**：動詞連用形 + 得る／得る（うる）  
**説明**：「得る」は「える」「うる」二種の読みがあり、「える」は口語的、「うる」は文語的・書面的。否定形は「えない」が共通。

1. それは**あり得る**（うる）話だ。
   → 那是可能的事。（文語・書面）
2. そう考えることも**あり得る**（える）。
   → 也可以那樣考慮。（口語）
3. 予測し**得ない**事態が発生した。
   → 發生了無法預測的事態。

## C2-076 ～ざる負えない vs ～ざるを得ない（colloquial vs formal）
**意味**：不得不～（口語誤用形 vs 正式形）  
**结构**：ざるを得ない（正式）；ざる負えない（口語的誤用）  
**説明**：「ざるを得ない」が正統な文語表現。「ざる負えない」は誤用だが、口語で広く浸透している。

1. この狀況では、承認せ**ざるを得ない**。
   → 在這種情況下，不得不承認。（正式）
2. 殘業せ**ざる負えなかった**よ。
   → 不得不加班啊。（口語・誤用）
3. 彼の意見に賛成せ**ざるを得なかった**。
   → 對他的意見不得不贊成。

## C2-077 うちに vs 間に vs 最中に（time nuance）
**意味**：うちに（在某狀態持續期間内） vs 間に（某時間區間内） vs 最中に（正在進行中）  
**结构**：うちに＝狀態持續期間；間に＝時間範圍；最中に＝動作的頂點  
**説明**：「うちに」は状態の継続中に別の動作が起こる。「間に」は特定的な時間幅。「最中に」は動作の真っ只中。

1. 明るい**うちに**帰ろう。
   → 趁天亮回去吧。（狀態持續期間）
2. 夏休みの**間に**、この本を読むつもりだ。
   → 打算在暑假内讀完這本書。（時間範圍）
3. 會議の**最中に**電話が鳴った。
   → 會議進行中電話響了。（動作頂點）

## C2-078 ～に足る vs ～に値する vs ～する価値がある（synonym nuance）
**意味**：值得～／有～的價值（信頼性 vs 價值 vs 一般）  
**结构**：に足る＝信頼に値する；に値する＝評價に値する；価値がある＝一般的  
**説明**：「に足る」は信頼や滿足の基準を満たす。「に値する」は評價や賞罰にふさわしい。「価値がある」は一般的。

1. 彼は信頼**するに足る**人物だ。
   → 他是值得信賴的人物。
2. その研究成果は賞**に値する**。
   → 那項研究成果值得獲獎。
3. 一生**思い出する価値がある**旅だった。
   → 是一生值得回憶的旅行。

## C2-079 ～こと vs ～の（nominalizer register）
**意味**：こと（抽象・形式） vs の（具體・感覚）  
**结构**：こと＝抽象内容、概念；の＝具體的場面、知覺  
**説明**：「こと」は客観的・抽象的な名詞化。「の」は主観的・感覚的な名詞化。特定動詞（忘れる、知らせる等）は「こと」を取る。

1. 彼が合格した**こと**を聞いた。
   → 聽説他合格了。（客觀事實）
2. 彼が笑っている**の**が見える。
   → 看得見他在笑。（知覺・具體）
3. 大切な**こと**は、諦めない**こと**だ。
   → 重要的是不放棄。（抽象概念）

## C2-080 である vs であります vs でございます（formality gradient）
**意味**：である（文語・論文） vs であります（丁寧） vs でございます（最敬語）  
**结构**：である＝斷定・文語；であります＝禮貌形；でございます＝最敬語  
**説明**：「である」は論文や法規、「であります」は演説や改まった口語、「でございます」は接客・最敬語。

1. これは事實**である**。
   → 這是事實。（論文・客觀）
2. 平和こそ我々の願い**であります**。
   → 和平才是我們的願望。（演説）
3. こちらがお求めの品**でございます**。
   → 這就是您要的商品。（接客）
