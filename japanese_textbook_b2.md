# 第1章 結果・終點

## B2-001 ～あげく
**意味**：……的結果（最終導致不好的結果）  
**结构**：Vた形 + あげく（に）/ N + の + あげく  
**説明**：表示經過長時間或複雜過程後，最終得到了令人失望或不好的結果。

1. いろいろ考えた**あげく**、やっぱり彼と別れることにした。
   → 考慮了很久的結果，還是決定和他分手。
2. 長い話し合いの**あげく**、何も決まらなかった。
   → 長時間討論的結果，什麼也沒決定。
3. 無理なダイエットの**あげく**、病気になってしまった。
   → 勉強減肥的結果，生病了。

## B2-002 ～末に
**意味**：經過……之後（最終結果）  
**结构**：Vた形 + 末（すえ）に / N + の + 末に  
**説明**：表示經過努力或長時間的過程後，最終達成了某個結果。比「あげく」更中性或正面。

1. 何度も失敗した**末に**、ついに研究が成功した。
   → 經過多次失敗之後，研究終於成功了。
2. 苦労した**末に**、自分の会社を設立した。
   → 辛苦奮鬥之後，創立了自己的公司。
3. 相談の**末に**、彼女と結婚することにした。
   → 商量之後，決定和她結婚。

## B2-003 ～た結果
**意味**：……的結果（客觀事實）  
**结构**：Vた形 + 結果（けっか）/ N + の + 結果  
**説明**：客觀陳述某個行動或現象所導致的結果。常用於論文、報告等正式場合。

1. 新しい薬を開発**た結果**、多くの患者が助かった。
   → 開發新藥的結果，很多患者得救了。
2. 調査し**た結果**、環境汚染が深刻であることがわかった。
   → 調查的結果，發現環境汙染很嚴重。
3. 努力の**結果**、第一志望の大学に合格した。
   → 努力的結果，考上了第一志願的大學。

## B2-004 ～きり
**意味**：自從……之後（就再也沒……）  
**结构**：Vた形 + きり  
**説明**：表示某動作完成後，一直處於沒有後續變化的狀態。通常與否定形式搭配。

1. 彼は「行ってきます」と言った**きり**、戻ってこなかった。
   → 他說了一聲「我出門了」之後，就再也沒回來。
2. あの漫画を読んだ**きり**、ずっと本棚に置いてある。
   → 那本漫畫讀過一次之後，就一直放在書架上。
3. 去年結婚式で会った**きり**、彼女に会っていない。
   → 去年在婚禮上見過之後，就再也沒見過她。

## B2-005 ～っきり
**意味**：自從……之後就（強調狀態不變）  
**结构**：Vた形 + っきり  
**説明**：「きり」的口語強調形，表示動作完成後再也沒有後續。

1. 彼女は電話を切った**っきり**、連絡してこない。
   → 她掛了電話之後，就再也沒聯絡。
2. 母は朝出かけた**っきり**、まだ帰ってこない。
   → 媽媽早上出門之後，到現在還沒回來。
3. あの事件以来、彼は口をきいた**っきり**だ。
   → 自從那事件以來，他只說了那一次話。

## B2-006 ～限り
**意味**：只要……就（條件限定）  
**结构**：V辞書形 / V可能形 / である + 限（かぎ）り  
**説明**：表示只要前項條件持續，後項就成立。

1. 私がここにいる**限り**、あなたを守ります。
   → 只要我在這裡，我就會保護你。
2. 雨が降らない**限り**、試合は行われる。
   → 只要不下雨，比賽就會舉行。
3. 彼が謝らない**限り**、許さないつもりだ。
   → 只要他不道歉，我就不打算原諒他。

## B2-007 ～ない限り
**意味**：除非……（否則就不）  
**结构**：Vない形 + ない限（かぎ）り  
**説明**：表示除非前項條件成立，否則後項不會發生。

1. あなたが謝ら**ない限り**、彼は許してくれないだろう。
   → 除非你道歉，否則他大概不會原諒你。
2. 新しい証拠が出**ない限り**、この事件は迷宮入りだ。
   → 除非出現新證據，否則這個案件將陷入僵局。
3. 居眠り運転をし**ない限り**、事故は起こらないはずだ。
   → 除非打瞌睡駕駛，否則應該不會發生事故。

## B2-008 ～に限る
**意味**：最好是……（最理想）  
**结构**：V辞書形 / N + に限（かぎ）る  
**説明**：表示在某種情況下，某選項是最好的選擇。

1. 疲れたときは、温泉に入る**に限る**。
   → 累的時候，最好就是去泡溫泉。
2. 夏はやっぱりビール**に限る**。
   → 夏天還是啤酒最好。
3. 風邪を引いたときは、ゆっくり寝る**に限る**。
   → 感冒的時候，最好就是好好睡覺。

# 第2章 程度・限定

## B2-009 ～あまり
**意味**：過度……以至於（程度過甚）  
**结构**：V辞書形 / N + の + あまり  
**説明**：表示因某種情緒或狀態過度，導致某種結果。

1. 驚きの**あまり**、声も出なかった。
   → 驚訝得說不出話來。
2. 彼は緊張の**あまり**、自分の名前さえ忘れてしまった。
   → 他過度緊張，連自己的名字都忘了。
3. 嬉しさの**あまり**、涙が出た。
   → 高興得流出了眼淚。

## B2-010 ～ばかりに
**意味**：正因為……（導致不好的結果）  
**结构**：Vた形 / イ形容詞 / ナ形容詞な / N + である + ばかりに  
**説明**：表示就是因為某個原因，導致了令人遺憾的結果。

1. 軽い気持ちでうそをついた**ばかりに**、信頼を失った。
   → 就因為輕率地說了謊，失去了信任。
2. 彼女は美人である**ばかりに**、いろいろと大変なんだ。
   → 她正因為是美女，反而有各種麻煩。
3. 時間がない**ばかりに**、十分なチェックができなかった。
   → 就因為沒有時間，沒能充分檢查。

## B2-011 ～だけに
**意味**：正因為……（所以更加）  
**结构**：普通形 + だけに（ナ形容詞な / Nである）  
**説明**：表示因為某種原因或特點，後項的程度比平時更加強烈。

1. 長年一緒にいた**だけに**、別れがつらかった。
   → 正因為在一起多年，分手才更痛苦。
2. 彼は有名な**だけに**、プライバシーが守られない。
   → 正因為他有名，隱私才得不到保護。
3. 簡単な問題だった**だけに**、間違えて悔しかった。
   → 正因為是簡單的問題，錯了才更懊悔。

## B2-012 ～だけあって
**意味**：不愧是……（名副其實）  
**结构**：普通形 + だけあって（ナ形容詞な / Nである）  
**説明**：表示因為有某種資格、名聲或特點，所以確實與之相稱。

1. さすがにプロの選手**だけあって**、プレーが違う。
   → 不愧是非職業選手，比賽表現就是不一樣。
2. 留学していた**だけあって**、彼の英語は完璧だ。
   → 不愧是留學過的人，他的英語很完美。
3. 高いレストラン**だけあって**、料理の味が素晴らしい。
   → 不愧是高級餐廳，料理的味道很棒。

## B2-013 ～だけのことはある
**意味**：沒有白費……（值得）  
**结构**：V辞書形 / N + の + だけのことはある  
**説明**：表示做某事的努力沒有白費，結果與付出相稱。

1. 練習し**ただけのことはあって**、試合に勝てた。
   → 沒有白費練習，比賽贏了。
2. この本は読む**だけのことはある**よ。とても感動した。
   → 這本書值得一讀，我很感動。
3. 高いお金を払った**だけのことはあって**、サービスが素晴らしかった。
   → 沒有白花那麼多錢，服務非常好。

## B2-014 ～くらいなら
**意味**：與其……還不如（比較選擇）  
**结构**：V辞書形 + くらいなら  
**説明**：表示與其做前項，不如做後項。通常帶有強烈的否定語氣。

1. あの人に頼む**くらいなら**、自分でやったほうがましだ。
   → 與其拜託那個人，還不如自己做。
2. そんなまずい店で食べる**くらいなら**、家で作る。
   → 與其在那種難吃的店吃，還不如在家做。
3. 借金して留学する**くらいなら**、行かないほうがいい。
   → 與其借錢留學，還不如不去。

## B2-015 ～どころか
**意味**：別說……就連……（否定程度遞進）  
**结构**：普通形 + どころか（N / ナ形容詞である）  
**説明**：強烈否定前項，指出實際情況遠非如此，甚至相反。

1. 彼は英語**どころか**、日本語もペラペラだ。
   → 別說英語了，他連日語都很流利。
2. もう大丈夫**どころか**、ますます悪くなっている。
   → 哪裡好轉了，反而越來越糟。
3. 休み**どころか**、毎日残業続きだ。
   → 別說休息了，每天都在加班。

## B2-016 ～どころではない
**意味**：不是做……的時候（無法顧及）  
**结构**：V辞書形 / N + どころではない  
**説明**：表示因為某種情況，完全沒有做某事的餘力或心情。

1. 試験前で、遊んでいる**どころではない**。
   → 考試前，根本不是玩的時候。
2. 忙しくて、休暇**どころではない**。
   → 忙得很，根本不是度假的時候。
3. 怪我をしてしまって、旅行**どころではない**。
   → 受傷了，根本不是去旅行的時候。

## B2-017 ～に過ぎない
**意味**：只不過是……（貶低程度）  
**结构**：普通形 + に過（す）ぎない（Nである）  
**説明**：表示程度很低，不是什麼了不起的事。

1. これは単なる噂**に過ぎない**。
   → 這只不過是謠言。
2. 私の意見は参考**に過ぎません**。
   → 我的意見僅供參考。
3. 彼が言ったことは真実を曲げた**に過ぎない**。
   → 他所說的不過是歪曲了事實。

## B2-018 ～にすぎない
**意味**：只不過是（文語変体）  
**结构**：普通形 + にすぎない  
**説明**：與「に過ぎない」相同，為書面語形式。

1. この数字は全体の一部**にすぎない**。
   → 這個數字只不過是整體的一部分。
2. 彼は一社員**にすぎません**が、大きな夢を持っている。
   → 他只不過是一名員工，但有遠大的夢想。
3. それは想像**にすぎない**。確かな証拠はない。
   → 那不過是想像，沒有確鑿證據。

# 第3章 對比・讓歩

## B2-019 ～に対して
**意味**：與……相對／針對……  
**结构**：N / V辞書形 + の + に対して  
**説明**：表示對比兩個事物，或表示對某對象的態度、行為。

1. 日本の電車は時間通りなの**に対して**、外国ではよく遅れる。
   → 日本的電車準時，與之相對，外國經常誤點。
2. 彼はクラシックが好きなの**に対して**、彼女はロックが好きだ。
   → 他喜歡古典音樂，與之相對，她喜歡搖滾。
3. その提案**に対して**、反対意見が出された。
   → 針對那個提案，提出了反對意見。

## B2-020 ～に反して
**意味**：與……相反（預期不符）  
**结构**：N + に反（はん）して  
**説明**：表示結果與預測、期待、規則等相反。

1. 予想**に反して**、試合は日本チームが負けてしまった。
   → 與預測相反，日本隊輸掉了比賽。
2. 彼の期待**に反して**、娘は医者にならなかった。
   → 與他的期望相反，女兒沒有當醫生。
3. 法律**に反して**、駐車した車がレッカー移動された。
   → 違反法律停車的車被拖吊了。

## B2-021 ～一方（で）
**意味**：一方面……另一方面（對比）  
**结构**：V辞書形 / ナ形容詞である / Nである + 一方（いっぽう）（で）  
**説明**：同時列舉兩個對比性的事物。

1. 都市の人口が増えている**一方**、地方の人口は減り続けている。
   → 一方面城市人口在增加，另一方面地方人口持續減少。
2. 彼女は仕事ができる**一方**で、家庭を大切にする。
   → 她一方面工作能力強，另一方面也很重視家庭。
3. この車は燃費がいい**一方**で、パワーが不足している。
   → 這輛車一方面油耗低，另一方面馬力不足。

## B2-022 ～反面
**意味**：另一方面（書面）  
**结构**：V辞書形 / ナ形容詞である + 反面（はんめん）  
**説明**：表示同一事物的兩個相反側面，比「一方で」更正式。

1. この薬はよく効く**反面**、副作用が強い。
   → 這個藥很有效，但另一方面副作用也強。
2. 彼は厳しい**反面**、とても優しいところもある。
   → 他很嚴格，但另一方面也有很溫柔的地方。
3. IT化は業務を効率化する**反面**、セキュリティのリスクが増える。
   → 資訊化提高了業務效率，另一方面也增加了安全風險。

## B2-023 ～かわりに
**意味**：代替……／作為交換……  
**结构**：V辞書形 / N + の + かわりに  
**説明**：表示代替某人做某事，或用前項交換後項。

1. 母の**かわりに**、私が弟を迎えに行った。
   → 代替媽媽，我去接了弟弟。
2. 日本語を教えてもらう**かわりに**、英語を教えている。
   → 作為有人教我日語的交換，我教英語。
3. 電車で行く**かわりに**、自転車で行くことにした。
   → 代替搭電車，決定騎自行車去。

## B2-024 ～にもかかわらず
**意味**：儘管……卻（讓歩）  
**结构**：普通形 + にもかかわらず（Nである / ナ形容詞である）  
**説明**：正式讓歩表達，前項條件與後項結果通常不相符。

1. 雨が激しかった**にもかかわらず**、試合は行われた。
   → 儘管雨很大，比賽還是舉行了。
2. 彼は何度も注意した**にもかかわらず**、同じミスを繰り返した。
   → 儘管提醒了他多次，他還是重複同樣的錯誤。
3. 周囲の反対**にもかかわらず**、彼女は自分の夢を追い続けた。
   → 儘管周圍的人反對，她仍繼續追逐自己的夢想。

## B2-025 ～ものの
**意味**：雖然……但是（逆接）  
**结构**：普通形 + ものの（ナ形容詞な / Nである）  
**説明**：承認前項的事實，但後項的結果卻不符預期。

1. 大学は卒業した**ものの**、仕事はまだ見つかっていない。
   → 雖然大學畢業了，但還沒找到工作。
2. 新しいスマホを買った**ものの**、使い方がよくわからない。
   → 雖然買了新手機，但不太會用。
3. 留学したい**ものの**、お金がなくて諦めた。
   → 雖然想留學，但沒錢所以放棄了。

## B2-026 ～とはいうものの
**意味**：雖說……但是（讓歩）  
**结构**：普通形 + とはいうものの  
**説明**：先承認一般說法，再提出現實的差異。

1. 健康が大事だ**とはいうものの**、なかなか運動する時間がない。
   → 雖說健康很重要，但很難有時間運動。
2. 日本語が上手になった**とはいうものの**、まだ敬語に苦労している。
   → 雖說日語變好了，但還在敬語上費勁。
3. 便利な時代だ**とはいうものの**、人間関係が希薄になった。
   → 雖說是方便的時代，但人際關係變淡薄了。

## B2-027 ～といえども
**意味**：即使……也／雖說……（文語讓歩）  
**结构**：N / 普通形 + といえども  
**説明**：文語表達，表示即使是有某種資格或情況，也不能例外。

1. 社長**といえども**、社内のルールには従わなければならない。
   → 即使是社長，也必須遵守公司規則。
2. プロの選手**といえども**、練習を怠れば勝てない。
   → 即使是職業選手，怠慢練習也贏不了。
3. 天才**といえども**、努力なしでは成功できない。
   → 即使是天才，不努力也無法成功。

# 第4章 條件・假定

## B2-028 ～さえ～ば
**意味**：只要……就（最小條件）  
**结构**：Vます形 + さえ + すれば / イ形容詞く + さえ + あれば / N + さえ + ～ば  
**説明**：表示只要滿足最低限度的條件，後項就能成立。

1. お金**さえ**あれ**ば**、何でも買えると思っている人がいる。
   → 有些人認為只要有錢，什麼都能買。
2. 彼が来**さえ**すれ**ば**、会議は始められる。
   → 只要他來，會議就能開始。
3. 時間**さえ**あれ**ば**、旅行に行きたいのだが。
   → 只要有時間，我想去旅行。

## B2-029 ～さえ
**意味**：甚至……（極端舉例）  
**结构**：N + さえ  
**説明**：舉出極端的例子，暗示其他就更不用說了。

1. そんなことは子供**さえ**知っている。
   → 那種事情連小孩都知道。
2. 彼は自分の名前**さえ**書けなかった。

# 第5章 原因・理由

## B2-038 ～からには
**意味**：既然……就（決意）  
**结构**：V辞書形 / Vた形 + からには  
**説明**：表示既然前項是事實，就應該做後項。帶有責任感或決心。

1. 約束した**からには**、必ず守らなければならない。
   → 既然答應了，就必須遵守。
2. 日本に来た**からには**、日本語をしっかり勉強したい。
   → 既然來了日本，就想把日語學好。
3. 選んだ**からには**、最後まで頑張ろう。
   → 既然選擇了，就努力到最後吧。

## B2-039 ～以上（は）
**意味**：既然……就（正式）  
**结构**：V辞書形 / Vた形 / ナ形容詞である + 以上（いじょう）（は）  
**説明**：與「からには」類似，但更正式。表示前項是既成事實，後項為相應的義務或決心。

1. 学生である**以上**、勉強を最優先すべきだ。
   → 既然是學生，就應該以學業為最優先。
2. 日本で働く**以上**、日本語をマスターしなければならない。
   → 既然要在日本工作，就必須掌握日語。
3. 約束した**以上は**、守る責任がある。
   → 既然約定了，就有遵守的責任。

## B2-040 ～上は
**意味**：既然……就（強烈）  
**结构**：Vた形 / V辞書形 + 上（うえ）は  
**説明**：比「以上」語氣更強，表示既然前項發生了，後項必須做。

1. こうなった**上は**、最後まで戦うしかない。
   → 既然變成這樣了，只能戰鬥到最後。
2. 引き受けた**上は**、責任を持ってやり遂げます。
   → 既然接受了，就會負責任地完成。
3. 覚悟を決めた**上は**、迷っている場合じゃない。
   → 既然下定決心了，就不是猶豫的時候。

## B2-041 ～ことから
**意味**：從……來看／因為（根據）  
**结构**：普通形 + ことから（Nである / ナ形容詞である）  
**説明**：表示以前項為判斷依據，推測或命名。

1. 誰も来なかった**ことから**、彼は会議が中止になったと思った。
   → 因為誰都沒來，他以為會議取消了。
2. この地域は桜が美しい**ことから**、「桜の里」と呼ばれている。
   → 這個地區因為櫻花很美，被稱為「櫻花之鄉」。
3. 二人の証言が一致しない**ことから**、事件は複雑だ。
   → 兩人的證詞不一致，所以案件很複雜。

## B2-042 ～ゆえに
**意味**：因為……所以（文語原因）  
**结构**：N / ナ形容詞である / V辞書形 + ゆえに  
**説明**：文語表達，表示原因。常用於論文、法律等正式文體。

1. 準備不足**ゆえに**、計画は失敗に終わった。
   → 因為準備不足，計畫以失敗告終。
2. 彼は無実である**ゆえに**、釈放された。
   → 因為他是無罪的，所以被釋放了。
3. 金欠**ゆえに**、旅行を諦めた。
   → 因為沒錢，放棄了旅行。

## B2-043 ～がゆえに
**意味**：正因為……所以（古典原因）  
**结构**：普通形 + がゆえに（ナ形容詞である / Nである）  
**説明**：古典表達，與「ゆえに」類似但更添文學色彩。

1. 彼女は美しい**がゆえに**、周囲から妬まれた。
   → 正因為她美麗，所以被周圍的人嫉妒。
2. 真実を語った**がゆえに**、彼は罰せられた。
   → 正因為說出了真相，他受到了懲罰。
3. 愛する**がゆえに**、別れを選ぶこともある。
   → 正因為愛，有時候也會選擇分離。

## B2-044 ～に伴って
**意味**：伴隨著……（同時變化）  
**结构**：N / V辞書形 + に伴（ともな）って  
**説明**：表示隨著前項的變化，後項也發生變化。

1. 経済の発展**に伴って**、環境問題が深刻化した。
   → 隨著經濟發展，環境問題變得嚴重。
2. 年を取る**に伴って**、体力が衰えてきた。
   → 隨著年齡增長，體力衰退了。
3. 人口の増加**に伴って**、住宅不足が問題になっている。
   → 伴隨人口增加，住房不足成了問題。

## B2-045 ～とあって
**意味**：因為是……所以（特殊情況）  
**结构**：普通形 + とあって（Nである）  
**説明**：表示因為前項的特殊情況或原因，後項的結果自然發生。

1. 久しぶりの再会**とあって**、皆とても喜んでいた。
   → 因為是久別重逢，大家都很高興。
2. 初めての海外旅行**とあって**、彼は前日から興奮していた。
   → 因為是第一次出國旅行，他從前一天就很興奮。
3. 有名な歌手のコンサート**とあって**、会場は超満員だった。
   → 因為是有名歌手的演唱會，會場爆滿。

## B2-046 ～につき
**意味**：因……（書面通知）  
**结构**：N + につき  
**説明**：正式書面用語，表示原因。常見於公告、通知、牌示。

## B2-092 ～にして
**意味**：到了……才／連……都（階段・讓歩）  
**结构**：N / Vた形 + にして  
**説明**：表示到了某個階段才發生某事，或連某種身分的人都……

## B2-093 ～だからこそ
**意味**：正因為……才（強調原因）  
**结构**：普通形 + だからこそ（N + だからこそ）  
**説明**：強調正是因為前面的原因，才有後面的結果。

1. 好きな仕事**だからこそ**、辛くても頑張れる。
   → 正因為是喜歡的工作，即使辛苦也能努力。
2. 彼がいた**だからこそ**、今の私がある。
   → 正因為有他，才有現在的我。
3. 簡単な問題**だからこそ**、油断して間違えた。
   → 正因為是簡單的問題，才大意出錯了。

## B2-094 ～こそ
**意味**：才是／正是（強調）  
**结构**：N + こそ  
**説明**：強烈提示某事物，強調其正是某種意義上的代表。

1. 今度**こそ**、試験に合格してみせる。
   → 這次一定要通過考試給你看。
2. これ**こそ**、私が探していた本です。
   → 這才是我在找的書。
3. 努力**こそ**が成功の鍵だ。
   → 努力才是成功的關鍵。

## B2-095 ～からこそ
**意味**：正因為……才（主觀原因）  
**结构**：普通形 + からこそ  
**説明**：強調主觀的原因，帶有強烈的情感色彩。

1. 好き**だからこそ**、厳しくしているんだ。
   → 正因為喜歡，才嚴格要求的。
2. 君のことを思う**からこそ**、言っているんだ。
   → 正因為為你著想，我才說的。
3. 助け合う**からこそ**、困難を乗り越えられる。
   → 正因為互相幫助，才能克服困難。

## B2-096 ～ばこそ
**意味**：正因為……才（文語強調）  
**结构**：V条件形 + ばこそ / ナ形容詞であればこそ  
**説明**：文語表達，與「からこそ」相同，語氣更古雅。

1. 努力すれ**ばこそ**、成功があるのだ。
   → 正因為努力，才有成功。
2. 健康であれ**ばこそ**、仕事に打ち込める。
   → 正因為健康，才能投入工作。
3. お互いを信じれ**ばこそ**、長く付き合える。
   → 正因為互相信任，才能長久交往。

## B2-097 ～すら／ですら
**意味**：甚至……都（舉例強調）  
**结构**：N + すら / ですら  
**説明**：舉出極端例子，暗示其他就更不用說。比「さえ」更書面。

1. そんなことは子供**すら**知っている。
   → 那種事連小孩都知道。
2. 彼は自分の名前**すら**思い出せなかった。
   → 他連自己的名字都想不起來。
3. 父親**ですら**、彼の行動を理解できなかった。
   → 連父親都無法理解他的行為。

## B2-098 ～など／なんか
**意味**：……之類（輕視・謙虛）  
**结构**：N + など / なんか  
**説明**：表示輕視、貶低或謙虛的語氣。口語常用「なんか」。

1. 私**なんか**、まだまだ勉強が足りません。

2. 忙しいのに、ゲーム**など**している場合じゃない。

   → 他的意見，連聽的價值都沒有。

**结构**：N + たる（N + たる + N）  
**説明**：文語表達，表示具有某種資格或身分的人。

1. 教師**たる**者、生徒の手本とならなければならない。
   → 作為教師，必須成為學生的榜樣。
2. リーダー**たる**者、決断力が求められる。
   → 作為領導者，需要決斷力。
3. 親**たる**者、子供を守る責任がある。
   → 作為父母，有保護小孩的責任。

## B2-192 ～たるもの
**意味**：作為……的人（文語角色）  
**结构**：N + たるもの  
**説明**：與B2-191相同，強調該身分應盡的義務。

1. 医者**たるもの**、患者の命を最優先に考えるべきだ。
   → 作為醫生，應該最優先考慮患者的生命。
2. 公務員**たるもの**、私利私欲のために動いてはいけない。
   → 作為公務員，不應為私利而行動。
3. 大人**たるもの**、自分の発言に責任を持たなければならない。
   → 作為大人，必須對自己的發言負責。

## B2-193 ～たりとも
**意味**：即使是……也不（文語微量否定）  
**结构**：N + たりとも  
**説明**：文語表達，表示即使一點點也不允許。

1. 試験は明日だ。一秒**たりとも**無駄にできない。
   → 明天考試，一秒鐘都不能浪費。
2. 一円**たりとも**無駄にしてはいけない。
   → 一分錢都不能浪費。
3. 一人**たりとも**遅刻してはいけないと先生が言った。
   → 老師說了，連一個人都不准遲到。

## B2-194 ～であれ（文語讓歩）
**意味**：無論……都（讓歩）  
**结构**：N + であれ  
**説明**：與B2-035相同，文語讓歩表達。

1. どんな困難**であれ**、決して諦めてはいけない。
   → 無論什麼困難，都絕不能放棄。
2. たとえ夢**であれ**、追い続ける価値はある。
   → 即使是夢想，也有追逐的價值。
3. 結果がどう**であれ**、全力を尽くしたことに悔いはない。
   → 無論結果如何，對全力以赴沒有遺憾。

## B2-195 ～であろうと
**意味**：無論……都（文語）  
**结构**：N + であろうと  
**説明**：文語表達，表示無論前項情況如何。

1. 金持ち**であろうと**、貧乏**であろうと**、人間の価値は変わらない。
   → 無論是富是貧，人的價值不變。
2. 子供**であろうと**、大人**であろうと**、ルールは守るべきだ。
   → 無論小孩還是大人，都應遵守規則。
3. 賛成**であろうと**、反対**であろうと**、意見を聞かせてほしい。
   → 無論贊成還是反對，請讓我聽聽意見。

## B2-196 ～か～ないかのうちに
**意味**：剛……就（即時）  
**结构**：V辞書形 + か + Vない形 + ないかのうちに  
**説明**：表示某動作剛開始或剛結束，下一個動作就發生了。

1. ベルが鳴った**か鳴らないかのうちに**、生徒たちは教室を飛び出した。
   → 鈴聲剛響（還沒響完），學生們就衝出了教室。
2. 彼は家に着く**か着かないかのうちに**、また電話がかかってきた。
   → 他剛到家（還沒完全到家），電話又響了。
3. 彼女は話し始める**か始めないかのうちに**、泣き出してしまった。
   → 她剛開始說話（還沒開始說），就哭了起來。

# 第25章 副詞的表現・文語

## B2-197 ～ことに
**意味**：令人……的是（情感修飾）  
**结构**：イ形容詞い / ナ形容詞な + ことに（Vた + ことに）  
**説明**：用於句首，修飾說話人對後項內容的情感判斷。

1. **残念なことに**、彼はパーティーに来なかった。
   → 遺憾的是，他沒有來派對。
2. **驚いたことに**、彼女はたったの10歳でピアニストデビューした。
   → 令人驚訝的是，她僅僅10歳就以鋼琴家身分出道了。
3. **幸運なことに**、事故で誰も怪我をしなかった。
   → 幸運的是，事故中沒有人受傷。

## B2-198 ～こと（感嘆・命令）
**意味**：多麼……啊（感嘆）  
**结构**：イ形容詞い / ナ形容詞な / Vた + こと  
**説明**：用於句末，表達強烈的感嘆或情感。

1. なんて美しい花**だろう**→ なんて美しい花**だこと**。
   → 多麼美的花啊。
2. まあ、かわいい**こと**。
   → 哎呀，真可愛啊。
3. 驚いた**こと**に、誰も知らなかった。
   → 令人驚訝的是，誰都不知道。

## B2-199 ～とは
**意味**：竟然……（驚訝・意外）  
**结构**：普通形 + とは（N / ナ形容詞とは）  
**説明**：表示對某事感到非常驚訝或意外。

1. まさか彼が海外で活躍する**とは**。
   → 沒想到他居然在國外活躍。
2. 彼女が社長になる**とは**、夢にも思わなかった。
   → 做夢也沒想到她會成為社長。
3. こんなに簡単に問題が解決する**とは**思わなかった。
   → 沒想到問題這麼簡單就解決了。

## B2-200 ～なんて
**意味**：竟然……／……之類（輕視・驚訝）  
**结构**：普通形 + なんて（N + なんて）  
**説明**：表示驚訝、意外或輕視。口語常用。

1. 彼がうそをつく**なんて**、信じられない。
   → 他竟然說謊，難以置信。
2. 毎日勉強する**なんて**、偉いね。
   → 每天學習，真了不起啊。
3. 忙しくて、休暇**なんて**取れないよ。
   → 忙得很，哪能休假啊。

## B2-201 ～というのは
**意味**：所謂的……是（定義説明）  
**结构**：普通形 / N + というのは  
**説明**：用於定義或説明某個詞語或概念。

1. 「日本人」**というのは**、日本国籍を持つ人のことです。
   → 所謂日本人，是指擁有日本國籍的人。
2. 本当の友情**というのは**、お互いを信じ合うことだ。
   → 所謂真正的友情，是彼此信任。
3. 幸せ**というのは**、お金だけでは測れないものだ。
   → 所謂幸福，不是僅用金錢就能衡量的。

## B2-202 ～とはいうものの
**意味**：雖説……但是  
**结构**：普通形 + とはいうものの  
**説明**：與B2-026相同，表示讓歩。

1. 便利になった**とはいうものの**、昔の良さも忘れたくない。
   → 雖説變方便了，但也不想忘記過去的好處。
2. 資格を取った**とはいうものの**、実務経験が足りない。
   → 雖説取得了資格，但實務經驗不足。
3. 春**とはいうものの**、まだまだ寒い日が続いている。
   → 雖説是春天，但寒冷的日子還在持續。
